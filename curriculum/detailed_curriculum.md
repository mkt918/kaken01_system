# 週単位：詳細カリキュラム（Step 1 & 2）

## 🐍 Step 1: Python基礎（8週間）
| 週 | テーマ | 学習内容 | テストのポイント |
| :--- | :--- | :--- | :--- |
| **01** | 変数と型 | `print()`, `type()`, 文字と数字の違い | `str` と `int` の足し算エラー |
| **02** | 数字と計算 | 四則演算、`int()`への変換、余り（%） | `input()`は常に文字であること |
| **03** | 条件分岐(1) | `if`, `==`, `!=`, インデントのルール | 正しい比較演算子の選択 |
| **04** | 条件分岐(2) | `else`, `elif`, 複数条件の組み合わせ | 境界値（18歳以上、等）の判定 |
| **05** | 繰り返し(1) | `while`, 無限ループの回避, `break` | ループが止まる条件の理解 |
| **06** | 繰り返し(2) | `for`, `range()`, リストの中身を表示 | 繰り返し回数の数え方 |
| **07** | リスト操作 | `append()`, `len()`, インデックス番号 | 0番目から始まるルールの定着 |
| **08** | 関数と復習 | `def`の基本、引数、Step 1総まとめ | Step 1の内容を網羅した大テスト |

## 📦 Step 2: SQL基礎（6週間）
| 週 | テーマ | 学習内容 | テストのポイント |
| :--- | :--- | :--- | :--- |
| **09** | DBとは？ | 表・列・レコード、DBとExcelの違い | データの型（INTEGER, TEXT） |
| **10** | SELECTの基本 | `SELECT *`, `WHERE`, `ORDER BY` | 条件に合うデータの絞り込み |
| **11** | SELECT応用 | `LIKE`（曖昧検索）, `LIMIT`, `DISTINCT` | 特定のキーワードを含む検索 |
| **12** | データ更新 | `INSERT`, `UPDATE`, `DELETE` | `WHERE`を忘れた時の危険性 |
| **13** | 集計と加工 | `COUNT`, `SUM`, `AVG`, `GROUP BY` | グループごとの合計の出し方 |
| **14** | 総復習 | 複数テーブルの概念（少しだけJOIN） | SQL単体での操作大テスト |

## 🚀 Step 3: システム連携開発（12週間）
| 週 | テーマ | 学習内容 | 活動内容 |
| :--- | :--- | :--- | :--- |
| **15** | Python+SQL (1) | `sqlite3`の接続、`execute()` | Pythonから表を作る |
| **16** | Python+SQL (2) | `?`（プレースホルダ）で安全に保存 | 入力内容をDBへ保存する |
| **17** | GUI連携 (1) | Tkinterの復習、リスト表示 | DBの内容を画面に一覧表示 |
| **18** | GUI連携 (2) | 削除ボタン、更新機能の追加 | 画面からDBを操作する |
| **19** | 企画・設計 | 「何を作る？」のアイデア出し | テーブル設計図の作成 |
| **20** | 開発開始 | 基本的な登録・表示機能の実装 | プログラムの骨組み作成 |
| **21-23** | 集中開発 | 個別のこだわり機能を追加 | 講師とマンツーマンで相談 |
| **24** | デバッグ | エラーの修正、使いやすさの改善 | 友達に使ってもらいフィードバック |
| **25** | 仕上げ | デザイン調整、発表資料の準備 | スクリーンショット撮影など |
| **26** | **成果発表会** | 自分の作ったシステムを実演！ | 相互評価と修了証授与 |

### Step 3 のプロジェクト例
- **学校図書管理システム**: 借りた本と返却予定日の管理。
- **お小遣い帳アプリ**: カテゴリ別の支出集計。
- **オリジナル単語帳**: 苦手な単語をDBに溜めてランダムに出題。
