## 指導ポイント

### B/Sの「左右一致」を視覚的に見せる

黒板に実際のB/S（簡略版）を書いて、
「左右の合計が必ず同じになる」ことを確認させる。

**発問例**：「なぜ左右が必ず一致するの？」
**答え**：「資産はすべて、誰かのお金で買われているから（借りたお金＝負債、自分のお金＝純資産）」

### P/Lの「フロー」概念

B/Sは「写真」（ある瞬間の状態）
P/Lは「ビデオ」（一定期間の動き）
このアナロジーで説明すると理解しやすい。

### 注意点

- 「資産・負債・純資産」の詳細はW3で扱うため、今回は「B/Sの左右に何があるか」程度に留める
- 「収益・費用」の詳細もW4で扱う

## ドリル2-2 解答

```
当期純利益 ＝ 350,000 − 300,000 ＝ 50,000円
```

## 追加問題（時間が余った場合）

次のB/Sの空欄を埋めなさい。

```
資産合計 800,000円、負債合計 300,000円
純資産 ＝ （　　　）円
```

答え：500,000円（800,000 − 300,000）
