## 指導ポイント

### 「他人振出小切手＝現金」は頻出

3級試験で必ず出題される。次の対比を板書で強調：

```
✅ 他社（得意先）から受け取った小切手 → 現金
❌ 自社が振り出した小切手           → 当座預金（の減少）
```

「振り出した」とあれば当座預金、「受け取った」とあれば現金、と
動詞で判断する習慣をつけさせる。

### 現金過不足の4パターン整理

板書で以下のフローを示すと混乱が減る：

```
①不一致発見
  ├─ 実際 < 帳簿 → 借方：現金過不足 ／ 貸方：現金
  └─ 実際 > 帳簿 → 借方：現金 ／ 貸方：現金過不足

②原因判明
  → 現金過不足 を正しい勘定へ振り替え

③決算時・原因不明
  ├─ 借方残（不足） → 雑損（費用）
  └─ 貸方残（過剰） → 雑益（収益）
```

### 現金出納帳の注意

「総勘定元帳」との違いを明確に：
- 総勘定元帳：すべての勘定を記録
- 現金出納帳：現金だけを詳しく記録する補助簿

### よくある誤答

| 誤り | 正解 | 補足 |
|------|------|------|
| 小切手受取→受取手形 | → 現金 | 小切手は手形ではない |
| 過不足（不足）→貸方に現金過不足 | 借方：現金過不足 | 現金を減らすのは貸方 |
