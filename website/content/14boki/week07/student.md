# W7: 小口現金

会社では毎日、交通費・切手代・コーヒー代など **小さな現金支払い** が発生します。
これをすべて経理部門（会計係）が管理すると非効率なため、**小口現金（こぐちげんきん）** という仕組みを使います。

---

## レッスン 1: 小口現金とは

### 小口現金の仕組み

> 日常的な少額の現金支払いを、担当者（**用度係**）に一定額を預けて管理させる制度

```
┌────────────────────────────────────────────────┐
│                小口現金の流れ                    │
│                                                │
│  会計係（経理）                用度係（小口担当）  │
│  ─────────                    ─────────────    │
│  ① 一定額を前渡し  →────────→  受け取って保管   │
│                                 ↓              │
│                              ② 少額の支払い     │
│                              （交通費・消耗品等）  │
│                                 ↓              │
│  ④ 補給          ←────────  ③ 支払内容を報告  │
└────────────────────────────────────────────────┘
```

### 会計係と用度係の役割

| 役割 | 担当 | 仕訳を切るタイミング |
|------|------|-------------------|
| **会計係** | 経理部門 | 前渡し時・補給時・報告受取時 |
| **用度係** | 各部署の担当者 | 小口現金出納帳に記入するのみ（仕訳は会計係） |

---

## レッスン 2: インプレスト・システム（定額前渡し制）

### インプレスト・システムとは

> 毎週（または毎月）、**あらかじめ決めた一定額**を前渡しし、使った分だけ補給することで残高を常に一定に保つ仕組み

```
例：週初めに30,000円を前渡し
  → 今週の支払い合計：23,000円
  → 週末に23,000円を補給
  → 翌週もまた30,000円の状態に戻る
```

### 前渡し時の仕訳（会計係が行う）

```
取引例：週初めに当座預金から30,000円を小口現金として用度係に渡した。

（借方）小口現金　30,000　／　（貸方）当座預金　30,000
```

> 小口現金は **資産勘定**（現金に近い資産）

---

## レッスン 3: 小口現金出納帳

### 小口現金出納帳とは

> 用度係が記入する、小口現金の受け入れ・支払いを記録する帳簿

### 小口現金出納帳の形式

| 受入 | 日付 | 摘要 | 支払 | 交通費 | 通信費 | 消耗品費 | 雑費 |
|------|------|------|------|--------|--------|---------|------|
| 30,000 | 4/1 | 前渡し | | | | | |
| | 4/2 | 電車代 | 800 | 800 | | | |
| | 4/3 | 切手代 | 1,200 | | 1,200 | | |
| | 4/4 | コピー用紙 | 2,200 | | | 2,200 | |
| | 4/5 | お茶代 | 500 | | | | 500 |
| | 4/5 | 合計 | **4,700** | 800 | 1,200 | 2,200 | 500 |
| | | 次週繰越 | | | | | |
| | | 残高 | **25,300** | | | | |

**記入ルール**：
- 前渡し・補給を受けた → **受入欄**に記入
- 支払った → **支払欄**（+ 費用内訳欄）に記入
- 残高 ＝ 受入合計 − 支払合計

---

## レッスン 4: 支払報告と補給の仕訳

### ① 用度係から支払報告を受けたとき（費用の計上）

週末などに用度係が「今週○○円使いました」と報告してきたとき、会計係は費用を仕訳します。

```
取引例：用度係から以下の支払い報告を受けた。
  交通費 800円、通信費 1,200円、消耗品費 2,200円、雑費 500円

（借方）旅費交通費　  800　／　（貸方）小口現金　4,700
（借方）通信費　　  1,200
（借方）消耗品費　  2,200
（借方）雑費　　　　  500
```

→ 報告時は「**小口現金を減らす**」（貸方：小口現金）

### ② 補給時の仕訳

使った分を補給する（残高を元の定額に戻す）。

```
取引例：用度係に現金4,700円を補給し、当座預金から支払った。

（借方）小口現金　4,700　／　（貸方）当座預金　4,700
```

### ③ 報告と補給を同時に行う場合（最重要！）

実務・試験では「**報告を受け、ただちに補給した**」というパターンが頻出です。

```
取引例：用度係から支払い報告（交通費800円・通信費1,200円・
        消耗品費2,200円・雑費500円）を受け、
        ただちに同額を当座預金から補給した。

（借方）旅費交通費　  800　／　（貸方）当座預金　4,700
（借方）通信費　　  1,200
（借方）消耗品費　  2,200
（借方）雑費　　　　  500
```

→ 「報告＋即時補給」では **小口現金は借方にも貸方にも登場しない**

**なぜ？**：小口現金は補給で元の金額に戻るため、増減がゼロになるから。

---

## レッスン 5: 仕訳パターンの整理

| タイミング | 借方 | 貸方 |
|-----------|------|------|
| 前渡し | 小口現金 | 当座預金（または現金） |
| 報告のみ | 各費用勘定 | 小口現金 |
| 補給のみ | 小口現金 | 当座預金（または現金） |
| **報告＋即時補給** | **各費用勘定** | **当座預金（または現金）** |

## まとめ

| 論点 | ポイント |
|------|---------|
| 前渡し | 借方：小口現金 ／ 貸方：当座預金 |
| 報告のみ | 借方：各費用 ／ 貸方：小口現金 |
| 補給のみ | 借方：小口現金 ／ 貸方：当座預金 |
| 報告＋即時補給 | 借方：各費用 ／ 貸方：当座預金（小口現金は不要） |
| 小口現金出納帳 | 受入・支払・残高・費用内訳を記入する補助簿 |
