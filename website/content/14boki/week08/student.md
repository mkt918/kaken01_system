# W8: 現金・預金 総合演習

W5〜W7で学んだ「現金・現金過不足・普通預金・定期預金・当座預金・小口現金」の知識を総仕上げします。
試験形式の問題で実力を確認しましょう。

---

## 復習チェックシート

演習前に以下の項目を確認しましょう。

| チェック項目 | 確認 |
|-------------|------|
| 他社振出の小切手を受け取った → **現金** | □ |
| 自社が小切手を振り出した → **当座預金の減少** | □ |
| 他社小切手をただちに当座預金へ → **現金を経由しない** | □ |
| 現金が帳簿より少ない → **借方：現金過不足** | □ |
| 現金過不足・原因不明のまま決算 → **雑損（不足）/ 雑益（過剰）** | □ |
| 定期預金の満期利息 → **受取利息（収益）** | □ |
| 当座借越の決算振替 → **当座借越（負債）** | □ |
| 小口現金の前渡し → **借方：小口現金 ／ 貸方：当座預金** | □ |
| 小口現金の報告＋即時補給 → **費用勘定 ／ 当座預金（小口現金は不要）** | □ |

## 総合演習 問題5: 現金過不足の総合問題

次の一連の流れについて、各時点の仕訳を行いなさい。

**状況**：
- 9月10日：帳簿上の現金残高180,000円に対し、実際有高は184,000円だった。
- 9月18日：過剰4,000円のうち3,000円は、受取手数料の記帳漏れと判明した。
- 12月31日（決算日）：残り1,000円の原因は不明のままだった。

**① 9月10日 過剰発見時**

```
（借方）（　　　　）　（　　）　／　（貸方）（　　　　）　（　　）
```

**② 9月18日 原因判明時**

```
（借方）（　　　　）　（　　）　／　（貸方）（　　　　）　（　　）
```

**③ 12月31日 決算時**

```
（借方）（　　　　）　（　　）　／　（貸方）（　　　　）　（　　）
```

### 解答

```
① 現金 4,000 ／ 現金過不足 4,000
② 現金過不足 3,000 ／ 受取手数料 3,000
③ 現金過不足 1,000 ／ 雑益 1,000
```

**解説**：
- ①で貸方に現金過不足4,000円が計上される（現金が帳簿より多い＝過剰）
- ②で3,000円が受取手数料へ振り替えられ、現金過不足の残高は1,000円（貸方）
- ③決算で貸方残1,000円→雑益（収益）へ振り替え

## W5〜W8 総まとめ表

| 勘定 | 種類 | 増加（借方） | 減少（貸方） | 特記事項 |
|------|------|------------|------------|---------|
| 現金 | 資産 | 受取・通貨代用証券 | 支払い | 他社小切手は現金 |
| 現金過不足 | 一時勘定 | 不足発生 | 過剰発生 | 決算で雑損/雑益へ |
| 普通預金 | 資産 | 入金・振替 | 出金・振替 | 利息→受取利息 |
| 定期預金 | 資産 | 預け入れ | 満期解約 | 利息→受取利息 |
| 当座預金 | 資産 | 預入・受取 | 小切手振出 | 利息なし |
| 当座借越 | **負債** | 翌期再振替 | 決算振替 | 当座預金のマイナス |
| 小口現金 | 資産 | 前渡し・補給 | 支払報告 | 即時補給は不要 |

## 試験直前チェックリスト

**現金・現金過不足**
- [ ] 他社小切手・郵便為替証書・利札 → 現金（通貨代用証券）
- [ ] 現金不足 → 借方：現金過不足 ／ 貸方：現金
- [ ] 現金過剰 → 借方：現金 ／ 貸方：現金過不足
- [ ] 原因判明 → 正しい費用・収益勘定へ振替
- [ ] 決算・不足不明 → 雑損（費用）
- [ ] 決算・過剰不明 → 雑益（収益）

**普通預金・定期預金**
- [ ] 定期預金に預け入れ → 借方：定期預金
- [ ] 定期預金の満期解約 → 貸方：定期預金（元本）＋貸方：受取利息（利息）

**当座預金・当座借越**
- [ ] 自社小切手振出 → 貸方：当座預金
- [ ] ただちに当座預金へ → 現金経由なし
- [ ] 当座借越の決算振替 → 当座借越（負債）
- [ ] 翌期首の再振替 → 当座預金へ戻す

**小口現金**
- [ ] 前渡し → 借方：小口現金 ／ 貸方：当座預金
- [ ] 報告のみ → 各費用 ／ 小口現金
- [ ] 補給のみ → 小口現金 ／ 当座預金
- [ ] 報告＋即時補給 → 各費用 ／ 当座預金（小口現金は登場しない！）
