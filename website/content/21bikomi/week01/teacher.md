# W1 教員用ガイド: 組織とビジネスの仕組み

## 授業の目標

- 組織の3層構造（経営者層・管理者層・一般従業員）を説明できる
- 職能別組織と事業部制組織の違いを説明できる
- 組織運営の3原則を具体例で理解できる

## 指導ポイント

### 導入：アプリの活用

授業冒頭に職種クイズアプリ（https://mkt918.github.io/01bikomi_shokui/）を使い、ビジネスの世界にある職種の多様性を体感させる。
その後「これらの職種は会社の中でどう組み合わさっているのか？」という問いに繋げる。

### 3層構造の教え方

「ピラミッド型」をイメージさせる。

- 上に行くほど人数は少ないが、影響範囲は大きい
- 下に行くほど人数は多く、現場の実務を担う
- 生徒自身のアルバイト経験（店員・店長・オーナー）に例えると身近に感じやすい

### 職能別 vs 事業部制：混乱しやすいポイント

「縦に切るか横に切るか」で板書すると分かりやすい。

```
職能別：機能で縦割り
  → 「営業だけの部屋」「製造だけの部屋」

事業部制：製品で横割り（各部屋に何でも揃う）
  → 「家電事業部の部屋（営業も製造も経理も）」
```

### 3原則：身近な例での説明

- **命令系統一元化**：「部活で顧問の先生とコーチ、両方から別の指示が来たら？」
- **権限と責任一致**：「文化祭の責任者なのに、お金の使い方を一切決められないとしたら？」
- **統制の範囲**：「クラスで班長が40人全員を管理するのは無理。だから班がある」

### 稟議の説明

「民主主義的な意思決定」として捉えさせる。
稟議は遅いが「みんなで確認する」日本的な文化。欧米は素早いが独断になりやすい。
どちらが良い悪いではなく、「それぞれの文化・考え方の違い」として扱う。

## ドリル解答と解説

### ドリル 1-1

```
① 経営者層（会社全体の戦略・意思決定）
② 一般従業員（担当顧客への営業・事務作業）
③ 管理者層（メンバーをまとめ管理する）
```

### ドリル 1-2

```
① 権限と責任一致の原則の違反
   （責任だけ与えて権限がない → 機能しない）

② 統制の範囲の原則の違反
   （100人は1人では管理不能 → 適正範囲を超えている）

③ 命令系統一元化の原則の違反
   （2人の上司から指示 → 矛盾が生じる）
```

## 補足資料

### 日本企業と外資系企業の意思決定比較（参考）

| 項目 | 日本企業（稟議型） | 外資系企業（トップダウン型） |
|------|-------------------|--------------------------|
| 決定の速さ | 遅い | 速い |
| 合意形成 | 広い（多くの人が関わる） | 狭い（上層部が決める） |
| 責任の所在 | 曖昧になりやすい | 明確 |
| リスク | 低い（みんなで確認） | 高い（一人の判断に依存） |

### 次週（W2）の予告

W2ではビジネスコミュニケーションの基本スキル（報連相・ビジネスマナーの基礎）を扱う予定。
今週学んだ「組織の上下関係」が、報連相を行う理由の背景知識となる。
