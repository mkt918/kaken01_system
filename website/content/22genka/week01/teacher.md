## 指導ポイント

### 導入の発問例

> 「スマートフォン1台を作るには、どんなコストがかかると思う？」

生徒に自由に考えさせた後、以下を整理します。
- 材料（プラスチック、ガラス、金属、電子部品）
- 工場で働く人の給料
- 工場の電気代・家賃
→「こういったコストをまとめて計算することが原価計算」と繋げる。

### 商業簿記 vs 工業簿記：よくある誤解と対応

**誤解①**「商業簿記と工業簿記は別の科目」
→「基本的な記録（仕訳）の仕組みは同じ。製造工程が追加されるイメージ」と伝える。

**誤解②**「工業簿記は製造業だけのもの」
→ ホテル・病院・学校など身近な例を挙げ、あらゆる業種に応用可能であることを伝える。

### 原価の3区分（製造原価・販売費・一般管理費）のイメージ化

黒板に以下の図を描くと分かりやすい：

```
【材料】
  ↓  工場の中（製造原価）
【製品完成】
  ↓  工場の外（販売費）
【販売・配送】

  + 本社・管理部門（一般管理費）
  = 総原価
```

「工場の中でかかったコスト = 製造原価」と覚えさせる。

## ドリル解答と解説

### ドリル 1-1: 商業簿記 vs 工業簿記

```
① コンビニが飲料を仕入れて販売 → 商業簿記
② 自動車メーカーが部品を組み立てて車を製造・販売 → 工業簿記
③ 衣料品店が服を仕入れてそのまま販売 → 商業簿記
④ 食品メーカーが野菜を仕入れ、加工して冷凍食品を製造 → 工業簿記
```

**判断基準**：「加工・製造が発生するか」がポイント。そのまま売れば商業簿記。

### ドリル 1-2: 原価の分類

```
① 工場の金属材料費 → 製造原価（材料費）
② テレビCMの広告費 → 販売費
③ 工場の電気代 → 製造原価（経費）
④ 本社経理担当者の給料 → 一般管理費
⑤ 工場従業員の給料 → 製造原価（労務費）
⑥ 配送トラックの燃料代 → 販売費
```

**混乱しやすいポイント**：「給料」は誰の給料かで分類が変わる。
- 工場の作業員 → 製造原価（労務費）
- 本社の事務員・営業 → 一般管理費 or 販売費

## 補足資料

### 総原価の計算イメージ

製品の販売価格はこう決まる：

```
販売価格 = 総原価 + 利益

  総原価 = 製造原価 + 販売費 + 一般管理費
```

原価計算で製造原価を正確に把握することで、いくらで売れば利益が出るかが分かる。

### 次週（W2）の予告

W2では原価の3要素である「材料費・労務費・経費」を学ぶ。
- 今週の「製造原価」がさらに細かく分類される
- 「何費」という言葉の使い方と費目の覚え方を指導する
