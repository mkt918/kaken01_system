## レッスン 1: 発生形態別分類（材料費・労務費・経費）

### 発生形態別分類とは？

> 「**何を使ったために発生した原価なのか**」という基準で、原価を3つに分類する方法です。

| 費目 | 内容 | 具体例 |
|------|------|--------|
| **材料費** | 製品の製造に使用した**物品**の消費高 | 鉄板、木材、プラスチック、ネジ |
| **労務費** | 製品の製造に使用した**労働力**の対価 | 工場作業員の賃金・給料 |
| **経費** | 材料費・労務費**以外**のすべての原価 | 電気代・減価償却費・外注加工費 |

### 材料費の詳細

製品を製造するために消費される**物品（材料）**のコストです。

| 種類 | 内容 | 例 |
|------|------|-----|
| **主要材料費** | 製品の主体となる材料 | 鉄板、木材、プラスチック |
| **補助材料費** | 製造を補助する材料 | 機械油、接着剤、塗料 |
| **工場消耗品費** | 工場で消費される消耗品 | 軍手、ウエス（拭き取り布） |
| **買入部品費** | 外部から購入した部品 | ネジ、モーター、ICチップ |

> **ポイント**：材料費は「製品の中に入るもの・製品を作るために直接使うもの」と覚えましょう。

### 労務費の詳細

製品の製造に携わる人への支払いです。**工場で働く人**への支払いが対象です。

| 種類 | 内容 | 例 |
|------|------|-----|
| **賃金** | 工場の作業員への時間・出来高払いの対価 | 製造ラインの作業員 |
| **給料** | 工場の監督者・管理者への月給 | 工場長、品質管理担当者 |
| **法定福利費** | 社会保険料など会社負担分 | 健康保険・厚生年金の会社負担 |
| **退職給付費用** | 退職金の積立費用 | 工場従業員の退職金引当 |

> **注意**：本社の事務員・営業担当者の給料は「労務費」ではなく**一般管理費・販売費**になります。

### 経費の詳細

材料費でも労務費でもないコストをすべてまとめて「経費」といいます。

| 種類 | 例 |
|------|-----|
| **光熱費** | 工場の電気代・水道代・ガス代 |
| **減価償却費** | 機械・設備を耐用年数で按分したコスト |
| **賃借料** | 工場建物・機械の賃貸料 |
| **外注加工費** | 外部に加工を依頼した費用 |
| **修繕費** | 機械・設備の修理代 |

> **ポイント**：経費の中で最重要なのが**外注加工費**と**減価償却費**。試験にもよく出ます。

## レッスン 2: 製品との関連による分類（直接費・間接費）

### 直接費と間接費とは？

製品ごとに「このコストがいくらかかったか」を**直接計算できるかどうか**で分類します。

| 分類 | 意味 | 例 |
|------|------|----|
| **製造直接費** | 特定の製品への消費額を**直接計算できる**原価 | 製品Aだけに使った鉄板 |
| **製造間接費** | 複数の製品に共通して発生し、**直接計算できない**原価 | 工場全体の電気代 |

### 6つの原価分類

材料費・労務費・経費 × 直接・間接 の組み合わせで**6分類**になります。

| | 直接費 | 間接費 |
|--|--------|--------|
| **材料費** | 直接材料費 | 間接材料費 |
| **労務費** | 直接労務費 | 間接労務費 |
| **経費** | 直接経費 | 間接経費 |

| 分類 | 具体例 |
|------|--------|
| 直接材料費 | 製品Aだけに使った鉄板（何kg使ったか明確） |
| 間接材料費 | 工場全体で使う機械油・補助材料 |
| 直接労務費 | 製品Aの組立専任作業員の賃金（何時間か明確） |
| 間接労務費 | 複数製品ラインを管理する工場長の給料 |
| 直接経費 | 製品A専用の外注加工費 |
| 間接経費 | 工場全体の電気代・減価償却費 |

### 賦課と配賦

直接費と間接費では、製品原価への計上のしかたが異なります。

| 用語 | 対象 | 意味 |
|------|------|------|
| **賦課（ふか）** | 製造**直接費** | 発生した金額をそのまま特定の製品に割り当てる |
| **配賦（はいふ）** | 製造**間接費** | 一定の基準（配賦基準）で複数の製品に按分して割り当てる |

```
【賦課のイメージ】
製品A専用の材料費 10,000円 → そのまま製品Aに 10,000円 計上

【配賦のイメージ】
工場の電気代 60,000円 ÷ 製品A・B・Cで按分
  → 製品A: 20,000円、製品B: 25,000円、製品C: 15,000円
```

> **配賦基準の例**：直接作業時間、機械稼働時間、直接材料費の割合など

## レッスン 3: 操業度による分類（変動費・固定費）

原価は、生産量（**操業度**）の変化に応じてどう変わるかでも4つに分類できます。

### 変動費

> 操業度の増減に**比例して**増減する原価です。

```
例：直接材料費（100個作れば100個分の材料費がかかる）
   ゼロ個 → 材料費ゼロ　/　200個 → 材料費2倍
```

### 固定費

> 操業度の増減に**関係なく**、一定額発生する原価です。

```
例：工場の家賃、機械の減価償却費
   0個作っても・1万個作っても → 毎月同じ金額
```

### 準変動費

> 操業度がゼロでも**一定額**発生し、さらに操業度の増加に応じて**比例的に増加**する原価です。

```
例：電気代（基本料金＋使用量分）
   0個作っても → 基本料金5,000円かかる
   100個作る → 基本料金5,000円 ＋ 使用量分10,000円
```

### 準固定費

> ある一定の操業度の範囲内では**固定**だが、それを超えると**段階的に増加**する原価です。

```
例：パート従業員の雇用
   〜100個   → 5人で対応 → 人件費50万円
   101〜200個 → 8人に増員 → 人件費80万円
   201個〜   → 12人に増員 → 人件費120万円
```

### 4分類のグラフイメージ

```
  原価
   ↑
   │    変動費     準変動費    固定費       準固定費
   │   ／        ／           ────      ──┐
   │  ／        ／ ↗          一定       ┌─┘
   │ ／        ●             ────    ┌─┘
   │／         固定部分あり                └──→ 段階的増加
   └────────────────────────→ 操業度
```

> **試験対策**：**変動費と固定費**の違いを確実に押さえましょう。準変動費・準固定費はイメージで理解すればOKです。

## レッスン 4: なぜ原価計算を行うのか

原価計算には、次の **4つの目的** があります。

| 目的 | 内容 |
|------|------|
| **① 財務諸表の作成** | 製造原価を正確に計算し、損益計算書・貸借対照表を作成する |
| **② 価格の計算** | 製品1個あたりのコストを把握し、適切な販売価格を設定する |
| **③ 原価管理** | 目標原価と実際原価を比較し、コストの無駄・超過を発見・改善する |
| **④ 利益計画** | 将来の生産量・販売量を想定し、利益目標を達成するための計画を立てる |

```
例：ケーキ1個の原価が800円と分かれば…

① 財務諸表 → 売上原価として正確に計上できる
② 価格設定 → 1,200円で売れば400円の利益と分かる
③ 原価管理 → 先月より材料費が高い → 仕入先を見直そう
④ 利益計画 → 月100万円の利益を出すには何個売ればいい？
```

## レッスン 5: 原価計算の流れ

原価計算は、次の **3段階** を順番に行います。計算期間は**1か月単位**が標準です。

```
① 費目別計算 → ② 部門別計算 → ③ 製品別計算
```

| 段階 | 内容 |
|------|------|
| **① 費目別計算** | 原価を材料費・労務費・経費に分類し、消費高を計算する |
| **② 部門別計算** | 製造間接費を工場の各部門（切削・組立など）ごとに集計する |
| **③ 製品別計算** | 製品ごとに原価を集計し、1個あたりの製造原価を計算する |

```
【全体の流れ】

 材料費 ┐
 労務費 ┼─→ ①費目別計算 ─→ ②部門別計算 ─→ ③製品別計算
 経　費 ┘   (3分類へ整理)  (部門ごとに集計) (製品1個の原価)
```

> **なぜ1か月単位か？**  
> 毎月原価を計算することで「今月は材料費が増えた」などの問題を素早く発見・対処できるためです。

## レッスン 6: 生産形態による計算方法の違い

製品の作り方によって、原価の計算方法が変わります。

### 個別原価計算 vs 総合原価計算

| | 個別原価計算 | 総合原価計算 |
|--|-------------|-------------|
| **生産形態** | 受注生産（オーダーメイド） | 大量見込み生産 |
| **集計単位** | 注文（製造指図書）ごと | 1か月ごと |
| **原価の計算** | 各注文の集計額がそのまま製品原価 | 合計 ÷ 完成個数 |
| **向いている業種** | 建設・造船・特注品 | 飲料・食品・自動車 |

```
総合原価計算の計算式：
1個あたりの製造原価 ＝ 1か月の製造原価合計 ÷ 1か月の完成個数
```

### 実際原価計算 vs 標準原価計算

| | 実際原価計算 | 標準原価計算 |
|--|-------------|-------------|
| **内容** | 実際にかかった原価をそのまま集計 | 目標原価（原価標準）を事前に設定して計算 |
| **目的** | 実績の正確な把握 | 目標 vs 実績の差異を分析し原価管理に活用 |

> **原価標準**：製品1単位を製造するのに「かかるべきコスト」として事前に定めた目標原価。

### 全部原価計算 vs 部分原価計算

| | 全部原価計算 | 部分原価計算 |
|--|-------------|-------------|
| **内容** | 製造にかかった**すべての原価**で計算 | **変動費のみ**など一部で計算 |
| **主な用途** | 財務諸表の作成（外部報告） | 内部の意思決定・利益計画 |

## ドリル 2-2: 賦課か配賦か？

次のコストは「賦課」と「配賦」のどちらで製品原価に計上しますか？

```
（  ）① 製品A専用の特殊鋼材費（どの製品に使ったか明確）
（  ）② 工場全体の電気代（複数製品で共通）
（  ）③ 製品B専用の外注加工費（契約書で金額が明確）
（  ）④ 工場長の給料（複数製品の管理をしている）
```

<details>
<summary>答えを見る</summary>

- 賦課：① ③（特定製品に直接計算できる → 直接費）
- 配賦：② ④（複数製品に共通 → 間接費を基準で按分）

</details>

## まとめ

| キーワード | 意味 |
|-----------|------|
| 材料費 | 製造に使う物品の消費高 |
| 労務費 | 製造に使う労働力の対価（賃金など） |
| 経費 | 材料費・労務費以外のすべての原価 |
| 製造直接費 | 製品ごとに直接計算できる原価 |
| 製造間接費 | 製品ごとに直接計算できない原価 |
| 賦課 | 直接費を特定製品にそのまま割り当てること |
| 配賦 | 間接費を基準で複数製品に按分すること |
| 変動費 | 操業度に比例して増減する原価 |
| 固定費 | 操業度に関わらず一定額発生する原価 |
| 準変動費 | 固定部分＋変動部分の両方を持つ原価 |
| 準固定費 | 操業度の範囲内で固定、超えると段階増加する原価 |
| 費目別計算 | 原価を3費目に分類し消費高を計算（第1段階） |
| 部門別計算 | 製造間接費を部門ごとに集計（第2段階） |
| 製品別計算 | 製品1個あたりの原価を計算（第3段階） |
| 個別原価計算 | 受注生産向け・注文ごとに原価集計 |
| 総合原価計算 | 大量生産向け・月単位で集計し個数で割る |
| 実際原価計算 | 実際にかかった原価で計算 |
| 標準原価計算 | 目標原価を設定し差異を分析 |
| 全部原価計算 | すべての原価で製品原価を計算 |
| 部分原価計算 | 一部の原価（変動費など）のみで計算 |
