## 指導ポイント

### 導入の発問例

> 「工場でスマートフォンを作り始めてから、利益が出るまでに何段階あると思う？」

生徒に考えさせてからスライドで全体の流れを見せると、勘定連絡図の意義が伝わりやすい。

### 製造間接費の教え方

「バケツ」のイメージが有効。

```
間接材料費 ─┐
間接労務費 ─┼──→ [製造間接費のバケツ] ──→ 各製品へ配賦
間接経費  ─┘
```

「どの製品のための費用か分からないものを、まず一か所に集めてから分けて配る」と説明する。

### 仕掛品・製品・売上原価の区別

状態の違いを強調する。「モノ（製品）に変化はないが、状態によって勘定が変わる」のがポイント。

| 状態 | 勘定 | 資産？費用？ |
|------|------|-------------|
| 製造中 | 仕掛品 | 資産 |
| 完成・在庫 | 製品 | 資産 |
| 販売済み | 売上原価 | 費用 |

「売れるまで費用にならない」ことを特に強調する。

### 損益計算の教え方

「ケーキ屋さん」で具体的に数字を当てはめて説明すると分かりやすい。

```
例：ケーキ1個800円のコストで作り、1,200円で100個販売

売上高：1,200円 × 100個 = 120,000円
売上原価：800円 × 100個 = 80,000円
売上総利益：120,000 − 80,000 = 40,000円
販売費及び一般管理費：12,000円（広告費・店員給料など）
営業利益：40,000 − 12,000 = 28,000円
```

## ドリル解答と解説

### ドリル 3-1

```
正しい順番：オ → エ → ウ → イ → ア → カ
（費用発生 → 製造間接費 → 仕掛品 → 製品 → 売上原価 → 利益）
```

「エ（製造間接費）の前にオ（発生）がある」「ウ（仕掛品）は製造の途中」という流れを意識させる。

### ドリル 3-2

```
直接「仕掛品」へ：① ③ ⑤
  → 特定製品への帰属が明確（直接費）→ 賦課

「製造間接費」を経由：② ④ ⑥
  → 複数製品に共通（間接費）→ 製造間接費に集計 → 配賦
```

### ドリル 3-3

```
① 仕掛品（製造途中 = 未完成）
② 製品（完成・未販売 = 在庫資産）
③ 売上原価（販売済み = 費用）
```

**よくある誤解**：「完成したら売上原価では？」
→ 「完成 ≠ 販売」。売れた時点で初めて費用（売上原価）になることを強調する。

## 補足資料

### 勘定連絡図の板書例

```
材料費 ─→ ┌──────────────────────┐
労務費 ─→ │ 製造間接費（間接費のみ）│ ─┐
経費   ─→ └──────────────────────┘  │（配賦）
                                       ↓
直接材料費 ──────────────→ ┌──────┐ 完成 ┌──────┐ 販売 ┌──────────┐
直接労務費 ──────────────→ │仕掛品│ ─→  │製品  │ ─→  │売上原価  │
直接経費   ──────────────→ └──────┘     └──────┘     └──────────┘
                                                              ↓
                              販売費及び一般管理費 ─────────→ 損益（利益）
                                                              ↑
                                                         売上高
```
