f-string とは？

f-stringは、文字列の先頭に f をつけることで、{} の中に変数や式を直接埋め込める仕組みです。
name = "田中" age = 16 print(f"名前：{name}") # 名前：田中 print(f"年齢：{age}歳") # 年齢：16歳
✅ 最も読みやすく、シンプル！

他の方法との比較（1/2）

❌ カンマ区切り
print("名前：", name)
余分なスペース
❌ + で連結
print("名前：" + str(name))
型変換が手間
❌ % フォーマット
print("名前：%s" % name)
書き方が複雑
❌ .format() メソッド
print("名前：{}".format(name))
対応が面倒

他の方法との比較（2/2）

✅ f-string（推奨）
print(f"名前：{name} 年齢：{age}歳")
最も読みやすく、シンプル！
💡 Python 3.6以降では f-string が標準。これを使おう！

{} の中には何が入る？

① 変数
f"点数：{score}"
② 式（計算）
f"合計：{x + y}"
③ メソッド呼び出し
f"大文字：{text.upper()}"
④ 条件式
f"成人：{age >= 18}"
💡 {} の中は「Pythonコード」として評価されます。計算も式も使えます！

フォーマット指定子 ― 細かく制御

: の後に指定子を書くと、表示方法を細かく制御できます。
値 = 100 print(f"{値:5}") # フォーマット指定子：「5」
指定子 意味 例 出力
> 右寄せ f"{name:>8}" 右に配置
< 左寄せ f"{name:<8}" 左に配置
^ 中央寄せ f"{name:^8}" 中央に配置

右寄せ・左寄せの実践例

左寄せ（商品名）
item1 = "りんご" item2 = "バナナ" print(f"{item1:<8} 150円") print(f"{item2:<8} 120円")
りんご 150円
バナナ 120円
右寄せ（金額）
name1 = "太郎" price1 = 1500 print(f"{name1:>8} {price1:>6}円")
太郎 1500円

ゼロ埋め（パディング）

数字の前にゼロを詰めたい場合（例：学籍番号、日時）に使います。
{7:03d}
007
{4:05d}
00004
{month:02d}
04
{day:02d}
21
year = 2026 month = 4 day = 21 print(f"日付：{year}-{month:02d}-{day:02d}") # 2026-04-21

桁区切り（カンマ区切り）

大きな数字を見やすくするために、千の位にカンマを自動挿入します。
1500000
1500000
{1500000:,}
1,500,000
125000000
125000000
{125000000:,}
125,000,000
price = 1500000 population = 125000000 print(f"価格：{price:,}円") # 価格：1,500,000円 print(f"人口：{population:,}人") # 人口：125,000,000人

小数点以下の桁数を指定

小数をどこまで表示するかを指定できます。
{3.14159:.2f}
3.14
{3.14159:.4f}
3.1416
{85.333333:.1f}
85.3
{85.333333:.0f}
85
pi = 3.141592653589793 print(f"π = {pi}") # π = 3.141592653589793 print(f"π = {pi:.2f}") # π = 3.14 print(f"π = {pi:.4f}") # π = 3.1416

パーセント表記

.1% を使うと、自動的に100倍されてパーセント記号がつきます。
{0.85:.1%}
85.0%
{0.8765:.2%}
87.65%
{0.333:.0%}
33%
{0.99:.1%}
99.0%
score = 85 max_score = 100 percentage = score / max_score print(f"得点率：{percentage:.1%}") # 得点率：85.0%

実践例① ― 店舗の売上レシート

item = "ノート" quantity = 3 unit_price = 150 subtotal = quantity * unit_price discount = subtotal * 0.1 total = subtotal - discount print(f"{item:<12} {quantity:>4}個 {unit_price:>6}円") print(f"{'小計':>24} {subtotal:>8}円") print(f"{'割引(10%)':>24} {discount:>8.0f}円") print(f"{'合計':>24} {total:>8.0f}円")
ノート 3個 150円 450円
小計 450円
割引(10%) 45円
合計 405円

実践例② ― 成績報告書

student_name = "田中 太郎" japanese = 78 math = 92 science = 85 english = 88 average = (japanese + math + science + english) / 4 print(f"{'学生氏名':<15}: {student_name}") print(f"{'国語':<15}: {japanese:>3}点") print(f"{'数学':<15}: {math:>3}点") print(f"{'理科':<15}: {science:>3}点") print(f"{'英語':<15}: {english:>3}点") print(f"{'平均':<15}: {average:>6.1f}点")
学生氏名 : 田中 太郎
国語 : 78点
数学 : 92点
理科 : 85点
英語 : 88点
平均 : 85.8点
