print() ― 画面に表示する

プログラムの基本中の基本。コンピュータに「何か言わせる」ことができます。
print("こんにちは！") print("プログラミングの世界へようこそ。")
実行結果：
こんにちは！
プログラミングの世界へようこそ。

print()の応用 ― カンマで複数表示

カンマ（,）で区切ると、複数の値をスペース付きで表示できます。
print("私の年齢は", 17, "歳です。") print("3 + 5 =", 3 + 5)
実行結果：
私の年齢は 17 歳です。
3 + 5 = 8

変数 ― コンピュータの「メモ帳」

変数は「名前付きの箱」。一度入れたデータは何度でも使えます。
山田 太郎
データ
→
name
変数名
name = "山田 太郎" print(name) # 山田 太郎

変数名のルール

✅ OK
score
player_name
my_score2
❌ NG
1score
player name
my-score
• 数字から始められない
• スペースは使えない
• Pythonの予約語（if, for など）は使えない
• ハイフン（-）は使えない

データの種類（型）

整数（int）
小数点なしの数
10, -3, 0
小数（float）
小数点ありの数
3.14, -0.5, 2.0
文字列（str）
""または''で囲った文字
"hello", "ABC"
真偽値（bool）
正（True）か偽（False）
True, False

型を確認する type()

どんな型か分からないときは type() で確認できます。
x = 42 y = 3.14 z = "こんにちは" print(type(x)) # <class 'int'> print(type(y)) # <class 'float'> print(type(z)) # <class 'str'>

文字列の「足し算」と「掛け算」

足し算（結合）
a = "Hello" b = " World" print(a + b)
→ Hello World
掛け算（繰り返し）
a = "Hello" print(a * 3)
→ HelloHelloHello

f-string ― 変数を文章に埋め込む

f をつけて {} で囲むと、変数を簡単に埋め込めます。
name = "田中" age = 16 score = 87.5 print(f"名前：{name}") print(f"年齢：{age}歳") print(f"点数：{score}点")
実行結果：
名前：田中
年齢：16歳
点数：87.5点

f-string ― 式も書ける

{} の中に計算式を書くこともできます。
price = 1500 tax_rate = 0.10 print(f"税込価格：{price * (1 + tax_rate)}円") print(f"消費税額：{price * tax_rate}円")
実行結果：
税込価格：1650円
消費税額：150.0円

演算子の優先順位

演算子 説明 例 結果
( ) 括弧 (2 + 3) * 4 20
** , * , / , // べき乗、掛け算、割り算 2 + 3 * 4 14
+ , - 足し算、引き算 10 + 5 - 3 12
% 割った余り 10 % 3 1
