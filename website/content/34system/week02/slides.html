input() ― ユーザーから入力を受け取る

プログラムが一時停止して、ユーザーがキーボードで入力するのを待ちます。
name = input("あなたの名前は？: ") print(f"こんにちは、{name}さん！")
実行結果：
あなたの名前は？: 田中
こんにちは、田中さん！

input() は「文字列」を返す ⚠

input() で受け取ったデータは、必ず文字列（str） になります。 数字として使いたいときは、変換が必要です。
data = input("数字を入力: ") print(type(data)) # <class 'str'> ← 文字列！ number = int(data) # int()で整数に変換 print(type(number)) # <class 'int'> ← 整数になった
💡 重要： 数値で計算する場合は必ず int() か float() で変換してください。

数値への変換 ― int() と float()

関数 変換先 使いどころ
int(x) 整数 個数・年齢・点数など小数が不要なとき
float(x) 小数 身長・体重・金額など小数が必要なとき
str(x) 文字列 数字を文字列として扱いたいとき
age = int(input("年齢: ")) # "17" → 17 height = float(input("身長(cm): ")) # "165.5" → 165.5 score = int(input("点数: ")) # "85" → 85

型変換のトラップ

✅ OK
x = int(float("3.14")) # floatにしてからint
❌ NG
x = int("3.14") # エラー！
⚠ 注意： int() は小数の文字列を直接変換できません。 先に float() で変換する必要があります。

同じ数値でも、型が違うと結果が変わる

税金計算のトラップ
price = 999 tax_rate = 0.10 # パターンA：int()で変換 tax_int = int(price * 0.10) # 99.9 → 99（切り捨て）⚠ total_int = price + tax_int # 1098円 # パターンB：小数で計算 tax_float = price * tax_rate # 99.9（そのまま） total_float = price + tax_float # 1098.9円
💡 学習ポイント： int() で変換すると小数点以下が切り捨てられます。 金額計算では注意が必要です！

複合代入演算子（省略形）

よく使う計算を短く書ける方法です。
書き方 意味 例
x += 20 x = x + 20 スコアを加算する
x -= 10 x = x - 10 体力を減らす
x *= 2 x = x * 2 金額を2倍にする
x //= 3 x = x // 3 3で割った結果を保存

実践例① ― 平均を計算する

score1 = int(input("1教科目の点数: ")) score2 = int(input("2教科目の点数: ")) score3 = int(input("3教科目の点数: ")) average = (score1 + score2 + score3) / 3 print(f"平均点：{average:.1f}点")
実行例：
1教科目の点数: 78
2教科目の点数: 85
3教科目の点数: 92
平均点：85.0点

実践例② ― 税込価格を計算する

price = int(input("商品の価格（円）: ")) tax_rate = 0.10 tax = int(price * tax_rate) # 端数は切り捨て total = price + tax print(f"本体価格：{price}円") print(f"消費税 ：{tax}円") print(f"税込合計：{total}円")
実行例：
商品の価格（円）: 999
本体価格：999円
消費税 ：99円
税込合計：1098円

データ型を選ぶチェックリスト

状況 選択 理由

金額（価格計算） float 小数が必要
身長・体重・温度 float 小数が自然に発生
