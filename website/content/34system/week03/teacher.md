## 📝 小テストの解答と解説（全5問）

**Q1.** パソコンが「正しい」と判断した時に出力される英単語はどれ？

- **正解:** **A. True**
- **解説:** False（間違い）とセットで覚えさせます。大文字始まりであることも確認してください。

**Q2.** 「自分のHP（hp）が 0 と等しいか？」を調べるための正しい書き方はどれ？

- **正解:** **C. `hp == 0`**
- **解説:** `=` 1つは箱に 0 を入れる（代入）ことになってしまいます。この区別は繰り返し強調してください。

**Q3.** 「所持金（money）が 100円 以上」を調べるための正しい書き方はどれ？

- **正解:** **B. `money >= 100`**
- **解説:** 日本語の「以上」は100を含みます。「より大きい（`>`）」は101からになることも確認してください。

**Q4.** Pythonで `if` の条件が合ったときだけ動かしたい命令文には、どういうルールが必要ですか？

- **正解:** **B. 命令文の前に半角スペース4つ分（インデント）字下げする**
- **解説:** Tabキー1回でも同じ効果です。

**Q5.** インデント（字下げ）を忘れた時にPythonが出すエラーの名前はどれですか？

- **正解:** **C. IndentationError**
- **解説:** エラー文の中の「Indent（インデント）」という単語が読めるようになれば、自力で解決できるようになります。
