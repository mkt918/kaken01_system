# 指導案: Week 05 「終わりなき繰り返し（while）」

## 🎯 今週のねらいと授業のゴール

プログラミングの醍醐味である「ループ（繰り返し）」を習得します。
最大のハイライトは「無限ループの恐ろしさを実際に体験させる」ことと、「強制終了（Ctrl+C）のやり方を体に刻み込ませる」ことです。

---

## ⏱️ タイムテーブル (180分)

### 1️⃣ 導入と while ループ (0分〜40分)

- **【解説】**: 「コンピュータが人間より優れている点は、文句を言わず何千回でも同じことを速く繰り返せることだ」と伝えます。
- **【演習】**: 「レッスン1」を実行させます。
- **【指導のポイント】**:
  - `count = count + 1` について、「数学の先生が見たら怒る式だけど、プログラミングでは『右で計算して、左の箱に上書き保存する』という意味なんだよ」と黒板で丁寧に図解してください。これを理解しないと今後のすべてで躓きます。
  - 「ドリル1-1」は完成コードを渡す設計にしているので、まず動かさせ、数値を変えて実験させてください。`time - 1` の行を消させて無限ループを体験させるのもここで行えます。

### 2️⃣ 恐怖の無限ループ体験イベント (40分〜80分)

- **【解説】**: ループには絶対に破ってはいけないルール（変化を書かないと終わらない）があることを語ります。
- **【演習】**: 「レッスン2」を、あえて全員一斉に実行させます。
- **【大パニックのコントロール】**:
  - 画面が猛烈に流れ続ける「無限ループ」に、生徒は「壊れた！」「止まらない！」とパニックになります。
  - そこで冷静に「**Ctrl と C を同時に押せ！**」と指示を出します。
  - この「Ctrl+Cで強制終了できる」という安心感を与えてから、原因を解説してください。

### 3️⃣ break と無限ループの正しい使い方 (80分〜120分) ※間で休憩10分

- **【解説】**: 一見悪者の無限ループも、「自らブレーキ（`break`）を踏めるなら」最強のテクニックになることを解説します。
- **【演習】**: 「レッスン3」のチャットボットと「ドリル3-1」のパスワード認証をやらせます。
- **【トラブルシューティング】**:
  - `break` を `if` の中（インデントを下げる）に入れないと、1回目で無条件にループが終わってしまいます。インデントの重要性を再確認させてください。
  - ドリル3-1は手順を5ステップで示しているので、1ステップずつ書かせるよう指示すると詰まりにくいです。

### 4️⃣ 総仕上げ：ミニプロジェクト（数当てゲーム） (120分〜170分)

- **【総合演習】**: ミッションを4ステップに分解しているので、1ステップずつ確認しながら進めさせてください。
  - ステップ1（変数作成）→ ステップ2（ループ枠を書く）→ ステップ3（入力処理）→ ステップ4（判定）の順で作らせると、どのパーツがどこにあるか整理しやすくなります。
  - 余裕がある生徒には「大きい・小さいのヒントを出す」チャレンジを提案すると熱中します。
- **【早く終わった人向け（RPGの戦闘シーン）】**:
  - `while slime_hp > 0:` の条件文を書けるかが勝負です。
  - 「逃げる（`break`）」を忘れると、HPが0になっても終わらないバグが発生します。

### 5️⃣ 小テストと振り返り (170分〜180分)

- 終わりの10分で小テストを実施し、答え合わせをして解散します。

## 📝 小テストの解答と解説（全5問）

**Q1.** プログラムで「〜の間、繰り返す」という命令はどれ？

- **正解:** **C. `while`**

**Q2.** プログラミングにおける `count = count + 1` の意味として正しいのは？

- **正解:** **B. 今の箱の中身に1を足して、また同じ箱に上書き保存する（カウントアップ）**
- **解説:** 数学と全く違う意味であることを再強調してください。

**Q3.** 終わらない繰り返し（無限ループ）を強制的に停止させるキーボード操作はどれ？

- **正解:** **A. `Ctrl + C`**
- **解説:** このショートカットは、ターミナルを使う世界中のエンジニアが毎日使う必須スキルです。

**Q4.** 繰り返し（ループ）を、途中で強制的に抜け出すための「ブレーキ」となる命令は？

- **正解:** **C. `break`**

**Q5.** わざと無限ループを作りたいときに使う書き方はどれ？

- **正解:** **B. `while True:`**
- **解説:** 条件が True（常に正しい）なので、永遠にループの中に閉じ込められます。`break` とセットで使うのが正しい使い方です。
