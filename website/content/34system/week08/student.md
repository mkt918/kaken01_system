## 🕒 レッスン 1: オリジナル魔法の作り方 `def`

新しく関数（魔法）を定義（define）するには、`def`（デフ）というキーワードを使います。

### 💻 ハンズオン（やってみよう）

以下のコードを入力して実行しましょう。

```python
# 1. ここで「say_hello」という魔法を定義（登録）しておきます
def say_hello():
    print("こんにちは！")
    print("今日も1日がんばりましょう！")

# 2. 定義しただけでは何も起きません。魔法を【使う（呼び出す）】必要があります。
say_hello()
say_hello()
```

「こんにちは！今日も1日がんばりましょう！」が2回連続で表示されましたね？

> **💡 ポイント**
> `def` のブロックも、`if` や `while` と同じように **インデント（スペース4個）** でお部屋を作ります。
> 「定義（def）」と「呼び出し（関数名()）」の2段階があることがポイントです。

### ✍️ ドリル 1-1：応援魔法を作ろう

「がんばれ！」「君ならできる！」と2行連続で励ましてくれる魔法 `cheer_up()` を作って、それを3回呼び出してください。

```python
# 魔法を定義する
def cheer_up():
    print("がんばれ！")
    print("君ならできる！")

# 魔法を3回呼び出す命令を3行書こう！
```

## 🕒 レッスン 2: 魔法の材料「引数（ひきすう）」

`print("こんにちは")` のように、魔法のカッコの中にデータを入れると、そのデータを使って魔法が発動します。このカッコの中に入れるデータを **「引数（ひきすう）」** と呼びます。

### 💻 ハンズオン（引数を渡す）

渡された名前に向かって挨拶する魔法を作ります。

```python
# name という「引数（材料を受け取る箱）」を用意する
def greet(name):
    print(f"やあ、{name} さん！調子はどう？")

# 使う時は、材料をカッコの中に入れて渡す！
greet("たろう")
greet("花子")
```

> **💡 ポイント**
> 引数 `name` は関数の中だけで使える「仮の箱」です。
> 呼び出すたびに中身（「たろう」「花子」）が変わるので、同じ関数を使い回せます！

### ✍️ ドリル 2-1：足し算の魔法陣

2つの数字を引数として受け取って、足し算の式と答えを表示する魔法 `tasizan(a, b)` を作ります。

```python
def tasizan(a, b):
    # a と b を足した結果を表示するコードを書こう！
    # ヒント：print(f"{a} + {b} = {a + b} です！")
    pass  # この行を削除して print 文に書き換えよう

tasizan(10, 5)
tasizan(100, 200)
```

> **⚠️ 注意**
> 呼び出すときに `tasizan(10)` と1つしか入れないと、引数が足りないというエラーが出ます。
> 定義した引数の数と、呼び出すときに渡す数を揃えることが大切です。

## 🕒 レッスン 3: 魔法の戻り値「return」

本物の魔法使い（プロのプログラマー）は、`print` で表示して終わりにするのではなく、**「処理した結果を返して（`return`）、あとは好きに使ってね」** というスタイルの関数をよく作ります。

### 💻 ハンズオン（returnで値を返す）

```python
def double_number(num):
    result = num * 2
    # 呼び出した人に結果を「返す（戻り値）」
    return result

# 魔法を使った結果が、箱（answer）に入る！
answer = double_number(10)
print(f"10の2倍は {answer} です！")
```

> **💡 print との違い**
> `print` は「画面に表示するだけ」ですが、`return` は「値をプログラムの中で次に使える状態で返す」ことができます。
> `answer = double_number(10)` のように、返ってきた値を変数に入れて次の処理に使えます。
