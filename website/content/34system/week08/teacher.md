## ⏱️ タイムテーブル (180分)

### 1️⃣ 導入と def (0分〜30分)

- **【解説】**: 「今まで使っていた `print` や `input` の正体は、誰かが長いコードを書いて作ってくれた『関数』だったんだよ。今日はみんながそれを作る側になる」と伝えます。
- **【演習】**: 「レッスン1」および「ドリル1-1」を実行させます。
- **【指導のポイント】**:
  - 初心者は「`def` で定義したのに画面に何も出ない！」と必ず叫びます。「魔法の呪文は、辞典に登録しただけでは発動しない。唱える（呼び出す）必要があるよ」と実演を交えて指導します。
  - 「定義（def）」と「呼び出し（関数名()）」の2段階があることを図で黒板に書くと伝わりやすいです。

### 2️⃣ 引数（ひきすう）と戻り値（return） (30分〜60分)

- **【解説】**: 引数は「魔法の材料」、`return` は「魔法の完成品のお届け」と例えます。
- **【演習】**: 「レッスン2」「ドリル2-1」と「レッスン3」をやらせます。
- **【トラブルシューティング】**:
  - `def tasizan(a, b):` なのに、呼び出すときに `tasizan(10)` と1つしか渡さずエラーを出す生徒が続出します。「大工さんに家を建ててと頼んだのに、木材だけ渡して釘を渡し忘れたら怒られるよね」と説明します。
  - `return` と `print` の違いが分からない生徒には、「`print` は結果を画面に捨てる、`return` は結果を次の人（呼び出した側）に渡す」と伝えると整理しやすいです。

### 3️⃣ ミニプロジェクト 1：多言語挨拶ロボ (60分〜100分) ※間で休憩10分

- **【総合演習 1】**: ミッションを4ステップに分解しているので、1ステップずつ確認しながら進めさせてください。
  - ユーザーからの `input` と自作関数の引数をドッキングさせる演習です。
  - `if lang == "1":` の `"1"` が文字列（クォートあり）である点に注意。`input()` の戻り値は常に文字列なので、`int()` で変換しない場合は文字列同士で比較する必要があります。

### 4️⃣ Step 1 総決算プロジェクト：RPGバトル (100分〜170分)

- **【総合演習 2】**: 最難関かつ最高に面白いプロジェクトです。完成コードは30行近くになります。
- **【指導ポイント】**:
  - 最初から全部書かせると絶対に混乱します。ミッションを6ステップに分解しているので、**必ずステップ1から順番に**作らせてください。
  - ステップ2で `while True:` の枠だけ作って `print("テスト")` を入れて動作確認 → ステップ3でコマンド入力を追加して確認 → のように、毎ステップ動作確認しながら積み上げるのがコツです。
  - 「完成した人から、魔法攻撃（コマンド3）や会心の一撃を追加していいよ！」と伝えると、生徒のモチベーションは爆発します。
  - RPGバトルは6ステップになっているので、詰まった生徒には「今どのステップで止まっている？」と声をかけて誘導してください。

## 📝 小テストの解答と解説（全5問 - 総まとめ号）

**Q1.** 新しい自作の魔法（関数）を作るための最初のキーワードはどれ？

- **正解:** **B. `def`**
- **解説:** Define（定義する）の略です。

**Q2.** 関数を呼び出すときに、カッコの中に入れて渡す「材料」データのことを何と呼ぶ？

- **正解:** **A. 引数（ひきすう）**
- **解説:** 「いんすう」と読み間違える人が多いので読み方も教えます。

**Q3.** 関数の処理が終わったあと、呼び出した側に「結果のデータ」を返すための命令はどれ？

- **正解:** **C. `return`**
- **解説:** リターン。球技のリターンと同じく、打ち返すイメージです。

**Q4.（復習）** Pythonが `IndentationError` を出すのは、主に何を忘れたときですか？

- **正解:** **B. 行の先頭の半角スペース（インデント）**
- **解説:** 8週間で最も生徒を苦しめ、そして成長させたエラーです。

**Q5.（復習）** これまで学んだ中で、**間違っている**組み合わせはどれですか？

- **正解:** **C. `input()` = 指定した回数だけループ（繰り返し）を行う**
- **解説:** 指定回数のループは `for` ループです。`input()` はユーザーから入力を受け取る命令です。
