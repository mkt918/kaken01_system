# Week 13: 大量データを一瞬で計算！「集計関数」

データベースの凄さは「探す・書き換える」だけではありません。何万件ものデータを一瞬で「数える」「合計する」「平均する」ことができる、データ分析の最強ツールでもあります。
今週は、計算に特化した **「集計関数」** と、データを切り分ける **`GROUP BY`（グループバイ）** を学びます。

---

## 🕒 レッスン 1: 「何個あるの？」を数える `COUNT`

「全部で何人いるの？」「ゲームに勝った人は何人？」などを数える魔法が **`COUNT`（カウント）** です。

### 💻 ハンズオン（やってみよう）
先生が用意した事前準備コードをRunして、生徒のテスト結果（`test_results`）テーブルを作成してください。

```sql
/* 生徒が全部で「何人」いるかを数える */
SELECT COUNT(*) FROM test_results;

/* 英語が80点以上の生徒が何人いるかを数える */
SELECT COUNT(*) FROM test_results
WHERE english >= 80;
```

### ✍️ ドリル 1-1：数学の赤点者数を調べろ！
数学（`math`）の点数が `30` 点 **未満（`<`）** の人が何人いるかを `COUNT(*)` と `WHERE` を使って調べてください。

```sql
SELECT ___________
FROM test_results
WHERE ___________;
```

---

## 🕒 レッスン 2: 合計を出す `SUM` と平均を出す `AVG`

テストの合計点や、お小遣いの合計額などを出す時は **`SUM`（サム）**、
クラスの平均点などを出す時は **`AVG`（アベレージ）** を使います。

### 💻 ハンズオン（やってみよう）
```sql
/* 全員の数学（math）の合計点数を出す */
SELECT SUM(math) FROM test_results;

/* 全員の英語（english）の平均点数を出す */
SELECT AVG(english) FROM test_results;
```

---

## 🕒 レッスン 3: 種類ごとに切り分けて計算する強魔術 `GROUP BY`

今までクラス「全員」の計算をしていました。
でも「クラス（A組・B組）**ごとに**平均点を出したい！」という時に使うのが、難易度MAXの超便利魔法 **`GROUP BY`（グループバイ）** です。

### 💻 ハンズオン（やってみよう）
```sql
/* クラス（class_name）ごとに、英語の平均点（AVG）を計算する */
SELECT class_name, AVG(english) 
FROM test_results
GROUP BY class_name;
```

> **💡 解説: `GROUP BY` の動き**
> 1. まず `GROUP BY class_name` によって、全員が「A組部屋」「B組部屋」というグループに分けられます。
> 2. 次に、その部屋ごとに `AVG(english)` （英語の平均）が計算されて、出力されます。
