## 🕒 レッスン 1: 黒い画面からの卒業！ウインドウを作る

Pythonには、アプリの画面を作るための **`tkinter`（ティーキンター）** という魔法の道具が最初から入っています。

### 💻 ハンズオン（やってみよう）
以下のコードをコピーして実行してみてください！

```python
import tkinter as tk

# 1. 土台となるウインドウ（アプリの画面）を作る
root = tk.Tk()

# 2. ウインドウのタイトルと大きさを決める
root.title("初めてのアプリ")
root.geometry("400x300")

# 3. ウインドウを表示し続ける（魔法のループ）
root.mainloop()
```

**【結果の確認】**
「初めてのアプリ」という本物のウインドウが現れました！×ボタンで閉じるとプログラムが終わります。

## 🕒 レッスン 2: ラベル、入力欄、ボタンを配置する

何もないウインドウに、パーツ（部品）を置いてみましょう。
パーツは **作って（定義）から、置く（pack）** という2ステップで画面に出ます！

### 💻 ハンズオン（やってみよう）
```python
import tkinter as tk

root = tk.Tk()
root.geometry("300x200")

# ① ラベル（文字）を作る → 置く
label = tk.Label(root, text="名前を入力してね")
label.pack()

# ② 入力ボックスを作る → 置く
entry = tk.Entry(root)
entry.pack()

# ③ ボタンを作る → 置く
button = tk.Button(root, text="押す！")
button.pack()

root.mainloop()
```

> **💡 解説: `pack()`（パック）の魔法**
> パーツを作っただけでは透明のままです。`pack()` と唱えると、上から順番にドスッ、ドスッと自動で綺麗に並べてくれます。

## 🕒 レッスン 3: ボタンを押した時に「機能」を動かす

ボタンを押したのに何も起きないのは寂しいですね。
Pythonの「関数（`def`）」を使って、ボタンを押した時にプログラムが動くようにします。

### 💻 ハンズオン（やってみよう）
```python
import tkinter as tk

# ボタンが押された時に実行される関数を「先に」作っておく
def hello_action():
    # 入力ボックス（entry）に入っている文字を取ってくる（get）
    name = entry.get()
    
    # ラベル（label）の文字を書き換える
    label.config(text=f"こんにちは、{name}さん！")

root = tk.Tk()
root.geometry("300x200")

label = tk.Label(root, text="あなたの名前は？")
label.pack()

entry = tk.Entry(root)
entry.pack()

# ボタンが押されたら(command)、上の hello_action 関数を動かす！
button = tk.Button(root, text="挨拶する", command=hello_action)
button.pack()

root.mainloop()
```

**【結果の確認】**
名前を入れてボタンを押すと、上の文字が変わりましたか？立派なアプリの完成です！

## 🚀 今日の総仕上げ2：ミニプロジェクト「GUI足し算アプリ」

`input()` の代わりにアプリの入力欄から2つの数字を受け取り、ボタンを押すと足し算の答えが表示されるアプリを作ります。

### 📝 ミッション
1. 入力ボックス（`Entry`）を **2つ** 用意します（`entry1`, `entry2`）。
2. 足し算ボタンと、答えを表示するための結果ラベル（最初は空っぽか「計算結果はここ」）を用意します。
3. 足し算関数（`def calc()` など）を作ります。
   - `num1 = int(entry1.get())`（※文字として受け取るので `int` で整数にします）
   - `num2 = int(entry2.get())`
   - 足した答えを、結果ラベルに `config` を使って表示します。
4. ボタンの `command=` に、作った関数をセットしてウインドウを出しましょう！
