# W1 教員用ガイド: 簿記とは何か

## 授業の目標

- 簿記の定義と目的を説明できる
- 複式簿記と単式簿記の違いを理解する
- 「取引」の概念を正確に把握する
- 日商簿記3級の位置付けを知る

## 指導ポイント

### 導入の発問例

「昨日の晩ごはん、何を食べましたか？その材料費っていくらか分かりますか？
　→ 家計でも記録しないと分からない。会社のお金はもっと複雑。だから簿記が必要。」

### 「取引」の誤解を防ぐ

- **よくある誤り**：注文・契約＝取引だと思いがち
- **正しい理解**：お金・モノ・権利が実際に動いたときが取引
- 「約束をした段階では取引にならない」と繰り返し強調する

### 複式簿記のイメージ

「お金が入ってきた＝なぜ入ってきたか、という2つの面を必ずセットで記録する」
→ W5以降の「仕訳」の伏線として意識させる

## ドリル1-1 解答

```
❌ ① 注文しただけ（取引なし）
✅ ② 現金が減り、商品が増えた
✅ ③ 建物（資産）が減少した
✅ ④ 現金が増え、借入金（負債）も増えた
❌ ⑤ 契約書サイン（まだ取引なし）
```
