# W2: 財務諸表を読もう ― 簿記の「最終成果物」

簿記の目的は、最終的に **財務諸表（ざいむしょひょう）** という報告書を作ることです。
今週は代表的な2つの財務諸表「貸借対照表」と「損益計算書」の基本を学びます。

---

## レッスン 1: 財務諸表とは？

財務諸表とは、**企業の財務状態と経営成績を外部に報告するための書類**です。

主な財務諸表は3つありますが、まず2つを押さえましょう。

| 財務諸表 | 英語名 | 何を示す？ |
|---------|--------|-----------|
| **貸借対照表** | Balance Sheet (B/S) | ある時点の「財産と借金の状態」 |
| **損益計算書** | Income Statement (P/L) | ある期間の「儲けの計算結果」 |

---

## レッスン 2: 貸借対照表（B/S）

### 貸借対照表とは

> **ある特定の日（決算日）における企業の財政状態を示した表**

「今この会社は何を持っていて（資産）、何を借りていて（負債）、自分のものはいくらか（純資産）」を表します。

### 貸借対照表の構造

```
┌─────────────────────────────────────────────┐
│              貸借対照表（B/S）                │
│                ○年○月○日現在                 │
├──────────────────┬──────────────────────────┤
│     資産の部      │        負債の部            │
│                  │                            │
│ 現金    100,000  │  借入金      200,000       │
│ 売掛金   50,000  │  買掛金       30,000       │
│ 建物    500,000  ├──────────────────────────┤
│                  │       純資産の部            │
│                  │  資本金      400,000       │
│                  │  繰越利益     20,000       │
├──────────────────┼──────────────────────────┤
│ 合計    650,000  │  合計        650,000       │
└──────────────────┴──────────────────────────┘
```

**ポイント：左（資産）と右（負債＋純資産）は必ず一致する！**

これを **「貸借平均の原理」** といいます。

### 基本等式

```
資産 ＝ 負債 ＋ 純資産
```

---

## レッスン 3: 損益計算書（P/L）

### 損益計算書とは

> **一定期間（通常1年間）の収益・費用・利益を示した表**

「この1年間で、いくら稼いで（収益）、いくら使って（費用）、いくら儲かったか（利益）」を表します。

### 損益計算書の構造

```
┌─────────────────────────────────────────────┐
│              損益計算書（P/L）                │
│          ○年○月○日〜○年○月○日              │
├──────────────────┬──────────────────────────┤
│     費用の部      │       収益の部             │
│                  │                            │
│ 仕入  300,000    │  売上高     500,000        │
│ 給料  100,000    │                            │
│ 家賃   50,000    │                            │
│ 当期純利益 50,000│                            │
├──────────────────┼──────────────────────────┤
│ 合計  500,000    │  合計       500,000        │
└──────────────────┴──────────────────────────┘
```

**ポイント：収益－費用＝当期純利益（もうけ）**

```
収益 ＝ 費用 ＋ 当期純利益
```

---

## レッスン 4: B/SとP/Lの関係

2つの財務諸表はつながっています。

```
P/L で計算した「当期純利益」
        ↓
B/S の「純資産（繰越利益）」に加算される
```

つまり、**損益計算書の結果が貸借対照表に反映される**のです。

---

## ドリル 2-1: B/Sのどちら側か答えよう

次の項目はB/Sの「資産」「負債」「純資産」のどれか答えましょう。

```
① 現金    → （        ）
② 借入金  → （        ）
③ 土地    → （        ）
④ 買掛金  → （        ）
⑤ 資本金  → （        ）
```

<details>
<summary>答えを見る</summary>

**答え：① 資産　② 負債　③ 資産　④ 負債　⑤ 純資産**

</details>

## ドリル 2-2: 当期純利益を計算しよう

下の損益計算書を完成させましょう。

| 費用 | 金額 | 収益 | 金額 |
|------|------|------|------|
| 仕入 | 200,000円 | 売上高 | 350,000円 |
| 給料 | 80,000円 | | |
| 広告費 | 20,000円 | | |
| **当期純利益** | （　　　）円 | | |

```
当期純利益 ＝ 収益合計 − 費用合計
　　　　　 ＝ 350,000 − (200,000 + 80,000 + 20,000)
　　　　　 ＝ 350,000 − 300,000
　　　　　 ＝ 50,000円
```

## まとめ

| キーワード | 意味 |
|-----------|------|
| 財務諸表 | 企業の財務状態・経営成績を報告する書類 |
| 貸借対照表（B/S） | ある時点の資産・負債・純資産の状態 |
| 損益計算書（P/L） | ある期間の収益・費用・利益 |
| 貸借平均の原理 | 資産 ＝ 負債 ＋ 純資産（常に成立） |

次回（W3）は、B/Sを構成する **「5つの要素」のうち、資産・負債・純資産**を深く学びます。
