# W3: 簿記の5要素① ― 資産・負債・純資産

簿記では、すべての取引を **5つの要素** に分類します。
今週はそのうちの3つ「**資産・負債・純資産**」を詳しく学びます。
これらは貸借対照表（B/S）を構成する要素です。

---

## レッスン 1: 資産（しさん）

### 資産とは

> **企業が所有する経済的価値のあるもの（財産）**

お金・モノ・権利など、プラスの価値を持つものすべてが資産です。

### 資産の分類

| 種類 | 意味 | 具体例 |
|------|------|--------|
| **流動資産** | 1年以内に現金化できるもの | 現金・普通預金・売掛金・商品 |
| **固定資産** | 長期間使用するもの | 建物・土地・車両・備品 |

### 主な資産勘定

| 勘定科目 | 説明 |
|---------|------|
| 現金 | 紙幣・硬貨・小切手 |
| 普通預金 | 銀行の普通預金口座 |
| 売掛金（うりかけきん） | 商品を売ったがまだもらっていない代金 |
| 商品 | 販売目的で持っている在庫 |
| 備品 | 机・パソコン・コピー機など |
| 建物 | 店舗・倉庫・事務所など |
| 土地 | 土地 |

---

## レッスン 2: 負債（ふさい）

### 負債とは

> **企業が将来返済・支払いをしなければならない義務（借金・未払い）**

マイナスの状態。いずれ資産が減る原因となるものです。

### 主な負債勘定

| 勘定科目 | 説明 |
|---------|------|
| 借入金（かりいれきん） | 銀行などからの借金 |
| 買掛金（かいかけきん） | 商品を買ったがまだ払っていない代金 |
| 未払金（みはらいきん） | 備品などを買ったがまだ払っていないお金 |
| 前受金（まえうけきん） | まだ商品を渡していないのに受け取ったお金 |

### 資産との対比

| | 資産 | 負債 |
|-|------|------|
| 意味 | もらえるもの・持っているもの | 払わなければならないもの |
| 例 | 売掛金（もらう権利） | 買掛金（払う義務） |
| B/S の位置 | 左側（借方） | 右側（貸方） |

---

## レッスン 3: 純資産（じゅんしさん）

### 純資産とは

> **資産から負債を引いた残り。会社の「本当の自分のもの」**

```
純資産 ＝ 資産 − 負債
```

会社を立ち上げたときに出資したお金（**資本金**）と、
これまでの利益の積み重ね（**繰越利益剰余金**）が主な内容です。

### 主な純資産勘定

| 勘定科目 | 説明 |
|---------|------|
| 資本金（しほんきん） | 会社設立時に出資した元手 |
| 繰越利益剰余金 | 過去の利益の積み重ね |

---

## レッスン 4: 増減のルール（超重要！）

資産・負債・純資産には、**増えるとき・減るときの記帳位置（借方・貸方）** のルールがあります。

```
        借方（左）    貸方（右）
資産  ：  増加   ←→   減少
負債  ：  減少   ←→   増加
純資産：  減少   ←→   増加
```

**覚え方のコツ**：
- 資産はB/Sの左にある → 増えるときも左（借方）に書く
- 負債・純資産はB/Sの右にある → 増えるときも右（貸方）に書く

## ドリル 3-2: B/Sを完成させよう

下の空欄を埋めてB/Sを完成させましょう。

| 資産の部 | 金額 | 負債・純資産の部 | 金額 |
|---------|------|----------------|------|
| 現金 | 150,000 | 借入金 | 200,000 |
| 売掛金 | 80,000 | 買掛金 | 50,000 |
| 建物 | 300,000 | 資本金 | 250,000 |
| | | 繰越利益剰余金 | （　　　） |
| **合計** | **530,000** | **合計** | **530,000** |

```
繰越利益剰余金 ＝ 530,000 − (200,000 + 50,000 + 250,000)
              ＝ 530,000 − 500,000
              ＝ 30,000円
```

## まとめ

| 要素 | 定義 | B/S上の位置 | 増加側 |
|------|------|------------|--------|
| 資産 | 企業の財産 | 左（借方） | 借方 |
| 負債 | 返済義務 | 右（貸方） | 貸方 |
| 純資産 | 資産−負債 | 右（貸方） | 貸方 |
