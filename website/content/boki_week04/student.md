## レッスン 1: 収益（しゅうえき）

### 収益とは

> **企業の活動によって発生した、資産の増加・負債の減少の原因となるもの**

「お金が入ってくる原因」＝ **稼ぎ** のことです。

### 主な収益勘定

| 勘定科目 | 説明 |
|---------|------|
| **売上（うりあげ）** | 商品・サービスを販売した収入（最重要！） |
| 受取利息（うけとりりそく） | 預金・貸付けの利息収入 |
| 受取手数料 | サービス提供で受け取った手数料 |
| 固定資産売却益 | 建物・土地を帳簿価額より高く売った利益 |

### 収益の増減ルール

```
        借方（左）    貸方（右）
収益  ：  減少   ←→   増加
```

収益はP/Lの右側（貸方）にある → **増えるときは右（貸方）に書く**

---

## レッスン 2: 費用（ひよう）

### 費用とは

> **収益を得るために消費した資産の減少・負債の増加の原因となるもの**

「お金が出ていく原因」＝ **出費・コスト** のことです。

### 主な費用勘定

| 勘定科目 | 説明 |
|---------|------|
| **仕入（しいれ）** | 販売目的の商品を買うためのお金（最重要！） |
| 給料（きゅうりょう） | 従業員への給与 |
| 家賃（やちん） | 店舗・事務所の賃貸料 |
| 水道光熱費 | 電気・水道・ガス代 |
| 通信費 | 電話・インターネット代 |
| 広告宣伝費 | 広告・チラシ代 |
| 支払利息 | 借入金の利息支払い |

### 費用の増減ルール

```
        借方（左）    貸方（右）
費用  ：  増加   ←→   減少
```

費用はP/Lの左側（借方）にある → **増えるときは左（借方）に書く**

---

## レッスン 3: 5要素の増減ルール ― 完全版

```
┌────────────┬────────────┬────────────┐
│            │  借方（左）  │  貸方（右）  │
├────────────┼────────────┼────────────┤
│ 資  産     │   増 加    │   減 少    │
│ 費  用     │   増 加    │   減 少    │
├────────────┼────────────┼────────────┤
│ 負  債     │   減 少    │   増 加    │
│ 純資産     │   減 少    │   増 加    │
│ 収  益     │   減 少    │   増 加    │
└────────────┴────────────┴────────────┘
```

**覚え方**：
- 左（借方）側：**資産・費用** が増える
- 右（貸方）側：**負債・純資産・収益** が増える

## レッスン 4: 仕訳（しわけ）入門

### 仕訳とは

> 取引を「借方と貸方」に分けて記録すること

すべての取引は、必ず **左（借方）＝ 右（貸方）** になるように記録します。

### 仕訳の形式

```
（借方）勘定科目　　金額　　／　（貸方）勘定科目　　金額
```

### 例題1: 商品を5万円で売り、現金を受け取った

```
取引の分析：
　現金（資産）が増加 → 借方
　売上（収益）が増加 → 貸方

仕訳：
（借方）現金　50,000　／　（貸方）売上　50,000
```

### 例題2: 家賃3万円を現金で払った

```
取引の分析：
　家賃（費用）が増加 → 借方
　現金（資産）が減少 → 貸方

仕訳：
（借方）家賃　30,000　／　（貸方）現金　30,000
```

### 例題3: 銀行から10万円を借り入れ、普通預金に入金した

```
取引の分析：
　普通預金（資産）が増加 → 借方
　借入金（負債）が増加   → 貸方

仕訳：
（借方）普通預金　100,000　／　（貸方）借入金　100,000
```

## ドリル 4-1: 5要素に分類しよう

```
① 売上        → （         ）
② 仕入        → （         ）
③ 給料        → （         ）
④ 受取利息    → （         ）
⑤ 支払利息    → （         ）
⑥ 繰越利益    → （         ）
```

**答え**：① 収益　② 費用　③ 費用　④ 収益　⑤ 費用　⑥ 純資産

## ドリル 4-2: 仕訳をやってみよう

次の取引の仕訳を書きましょう。

**① 商品を8,000円で仕入れ、現金で支払った**

```

```

**② 給料15,000円を現金で支払った**

```
（借方）（　　　）　15,000　／　（貸方）（　　　）　15,000
```

**③ 商品を12,000円で売り、現金を受け取った**

```
（借方）（　　　）　12,000　／　（貸方）（　　　）　12,000
```

**解答**：

```

③ （借方）現金　12,000　／　（貸方）売上　12,000
```
