## 指導ポイント

### 「仕訳」でつまずく生徒への対処

仕訳の手順を必ず以下の順で行わせる：

```
ステップ1：何が増えた・減った？
ステップ2：それは5要素のどれ？
ステップ3：増えた→借方？貸方？（増減ルール表を見る）
ステップ4：金額は同じか確認
```

「何も見ずに解く」より「ルール表を使いながら解く」ことを最初は推奨。
反復練習で自然に覚えさせる。

### 「売上 vs 仕入」の混乱防止

- 売上 → 売ったときに使う → 収益（P/L右）
- 仕入 → 買ったときに使う → 費用（P/L左）

商業の文脈で「売上原価」との混同が出てくるのはまだ先なので、
現時点では「仕入＝費用」として押さえる。

### 4週の学習のまとめ方

ホワイトボードに5要素の全体像を書いて、
「これ全部が簿記の基本！あとはこの組み合わせ」と伝えると
生徒の達成感・見通しが持てる。

## ドリル4-1 解答

① 収益　② 費用　③ 費用　④ 収益　⑤ 費用　⑥ 純資産

## ドリル4-2 解答

```
① （借方）仕入　8,000　  ／　（貸方）現金　8,000
② （借方）給料　15,000　 ／　（貸方）現金　15,000
③ （借方）現金　12,000　 ／　（貸方）売上　12,000
```

## 発展問題（W5以降の予習として）

「掛け（かけ）」取引への橋渡し：

「商品を10,000円で売ったが、代金は来週もらう予定」
→ 現金じゃなくて「売掛金（資産）」が増える！

```
（借方）売掛金　10,000　／　（貸方）売上　10,000
```
