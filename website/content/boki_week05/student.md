# W5: 現金の取引 ― 「現金」の正体と記帳

簿記でいう「現金」は、日常生活の「お財布の中身」とは少し違います。
今週は **現金勘定の範囲・記帳方法・現金過不足の処理** を3級レベルでしっかり習得します。

---

## レッスン 1: 簿記上の「現金」とは何か

### 現金勘定に含まれるもの

簿記3級では、以下のものをすべて「**現金**」として処理します。

| 種類 | 具体例 | ポイント |
|------|--------|---------|
| 通貨 | 紙幣・硬貨 | 言わずもがな |
| **通貨代用証券** | 他人振出の小切手 | 受け取ったらすぐ現金扱い |
| 通貨代用証券 | 郵便為替証書 | 現金と同様に流通 |
| 通貨代用証券 | 期限到来済みの公社債利札 | 現金化できる状態のもの |

> **重要**: 「**他人振出の小切手**」を受け取ったとき → **現金**で記帳する  
> ※ 自分が振り出した小切手（= 当座預金の引き出し）は現金ではない

### 現金として扱わないもの

| もの | 理由 | 正しい勘定 |
|------|------|-----------|
| 自分が振り出した小切手 | 当座預金口座から引き出す指示書 | 当座預金 |
| 約束手形・為替手形 | 受取手形・支払手形で処理 | 受取手形 / 支払手形 |
| 収入印紙 | 使用済みになって初めて費用 | 租税公課（または貯蔵品） |
| 郵便切手 | 未使用は資産 | 通信費（または貯蔵品） |

## レッスン 2: 現金の仕訳

### 基本パターン（増加・減少）

```
現金が増えるとき：（借方）現金 ×× ／（貸方）相手科目 ××
現金が減るとき：（借方）相手科目 ×× ／（貸方）現金 ××
```

### 例題1: 商品を売り上げ、現金を受け取った

> 商品を30,000円で売り渡し、現金を受け取った。

```
（借方）現金　30,000　／　（貸方）売上　30,000
```

### 例題2: 他人振出の小切手を受け取った

> 得意先から売掛金の回収として、同社振出の小切手50,000円を受け取った。

```
（借方）現金　50,000　／　（貸方）売掛金　50,000
```
→ 小切手は「現金」！

### 例題3: 現金で費用を支払った

> 電話代（通信費）8,000円を現金で支払った。

```
（借方）通信費　8,000　／　（貸方）現金　8,000
```

### 例題4: 商品を仕入れ、代金は現金で支払った

> 商品を20,000円で仕入れ、代金は現金で支払った。

```
（借方）仕入　20,000　／　（貸方）現金　20,000
```

## レッスン 3: 現金出納帳

### 現金出納帳とは

> 現金の入出金を日付順に記録する**補助簿（ほじょぼ）**

総勘定元帳の「現金」勘定を補助する帳簿で、残高を常に管理します。

### 現金出納帳の形式

| 日付 | 摘要 | 収入 | 支出 | 残高 |
|------|------|------|------|------|
| 4/1 | 前月繰越 | | | 100,000 |
| 4/3 | 売上代金受取 | 30,000 | | 130,000 |
| 4/5 | 仕入代金支払 | | 20,000 | 110,000 |
| 4/10 | 通信費支払 | | 8,000 | 102,000 |

**記入ルール**：
- 現金が入ってきた → **収入欄**に記入
- 現金が出ていった → **支出欄**に記入
- 残高 ＝ 前残高 ＋ 収入 − 支出

## レッスン 4: 現金過不足（超重要！）

### 現金過不足とは

> 帳簿上の現金残高と、実際に数えた現金（実際有高）が **一致しないとき**に使う勘定

実務でも頻繁に起こる事象です。

### パターン1: 実際有高が帳簿より「少ない」（不足）

```
帳簿残高 50,000円　実際有高 48,000円　→ 2,000円不足

仕訳：
（借方）現金過不足　2,000　／　（貸方）現金　2,000
```

→ 帳簿の現金を減らして、「現金過不足」で一時的に受ける

### パターン2: 実際有高が帳簿より「多い」（過剰）

```
帳簿残高 50,000円　実際有高 53,000円　→ 3,000円過剰

仕訳：
（借方）現金　3,000　／　（貸方）現金過不足　3,000
```

### パターン3: 原因が判明したとき（現金過不足の解消）

過不足を計上した後、原因が分かったら正しい勘定科目に振り替えます。

**例**：不足2,000円の原因が「通信費の記帳漏れ」と判明した

```
（借方）通信費　2,000　／　（貸方）現金過不足　2,000
```

### パターン4: 決算時まで原因不明のとき

> 決算日になっても原因不明の場合 → **雑損（ざっそん）** または **雑益（ざつえき）** に振り替える

| 状況 | 振替先 |
|------|--------|
| 不足が残っている | **雑損**（費用） |
| 過剰が残っている | **雑益**（収益） |

```
例）不足1,000円が期末まで原因不明
（借方）雑損　1,000　／　（貸方）現金過不足　1,000
```

## まとめ

| 論点 | ポイント |
|------|---------|
| 現金の範囲 | 通貨＋通貨代用証券（他人振出小切手・郵便為替・利札） |
| 現金出納帳 | 収入・支出・残高を日付順に記録する補助簿 |
| 現金過不足（不足） | 借方：現金過不足 ／ 貸方：現金 |
| 現金過不足（過剰） | 借方：現金 ／ 貸方：現金過不足 |
| 原因判明 | 現金過不足 → 正しい勘定へ振り替え |
| 決算時・原因不明 | 不足→雑損（費用）／ 過剰→雑益（収益） |
