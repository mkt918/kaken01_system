# W6: 当座預金の取引 ― 小切手と当座借越

企業は大きな金額の支払いに「小切手」を使います。
今週は **当座預金口座・小切手の振出と受取・当座借越** を3級レベルで学びます。

---

## レッスン 1: 当座預金とは

### 当座預金口座の特徴

| 項目 | 内容 |
|------|------|
| 目的 | 企業間の大口決済に使う口座 |
| 利息 | **利息はつかない** |
| 引き出し方法 | **小切手**を振り出すことで支払いを行う |
| 残高以上の引き出し | 原則不可（当座借越契約があれば可） |

> 普通預金との違い：当座預金は利息なし、小切手で決済する「法人向け口座」

### 当座預金口座の開設

銀行と**当座取引契約**を結び、お金を預けて口座を開設します。

```
取引例：銀行と当座取引契約を結び、現金500,000円を預け入れた。

（借方）当座預金　500,000　／　（貸方）現金　500,000
```

---

## レッスン 2: 小切手の振出（支払い）

### 小切手とは

> 当座預金口座から一定金額を引き出す「支払指示書」

**自分が小切手を振り出す** ＝ 当座預金の **減少**

### 小切手振出の仕訳

```
取引例：仕入先に対し商品代金100,000円を小切手を振り出して支払った。

（借方）仕入　100,000　／　（貸方）当座預金　100,000
```

```
取引例：備品30,000円を購入し、小切手を振り出して支払った。

（借方）備品　30,000　／　（貸方）当座預金　30,000
```

### 重要：自分が振り出した小切手を受け取った場合

自社が以前振り出した小切手が、何らかの事情で戻ってきたとき：

```
→ 当座預金の増加（振出の取消）

（借方）当座預金　××　／　（貸方）当座預金　×× … ではなく

実際は仕訳の相手科目によって異なる。
例）売掛金の代わりに受け取った場合：
（借方）当座預金　××　／　（貸方）売掛金　××
```

---

## レッン 3: 小切手の受取（入金）

### 他社振出の小切手を受け取った場合

前回（W5）で学んだ通り：

> **他人（他社）が振り出した小切手を受け取った → 現金で処理**

```
取引例：得意先から売掛金80,000円の回収として、
        同社振出の小切手を受け取った。

（借方）現金　80,000　／　（貸方）売掛金　80,000
```

### ただちに当座預金に預け入れた場合

受け取った小切手をすぐ銀行に持ち込んで入金したときは：

```
方法①：2段階で仕訳
　 受取時：（借方）現金　80,000　／　（貸方）売掛金　80,000
　 入金時：（借方）当座預金　80,000　／　（貸方）現金　80,000

方法②：1回でまとめて仕訳（同日の場合）
　 （借方）当座預金　80,000　／　（貸方）売掛金　80,000
```

**試験では「ただちに当座預金に預け入れた」という記述があれば方法②で解答する。**

---

## レッスン 4: 当座預金出納帳

### 当座預金出納帳とは

> 当座預金の入出金を記録する**補助簿**

| 日付 | 摘要 | 預入 | 引出 | 借または貸 | 残高 |
|------|------|------|------|-----------|------|
| 4/1 | 前月繰越 | | | 借 | 500,000 |
| 4/5 | 仕入代金支払（小切手） | | 100,000 | 借 | 400,000 |
| 4/10 | 売掛金回収（小切手入金） | 80,000 | | 借 | 480,000 |
| 4/15 | 備品購入（小切手） | | 30,000 | 借 | 450,000 |

**「借」＝残高がプラス（預金残高あり）**  
**「貸」＝残高がマイナス（当座借越）**

---

## レッスン 5: 当座借越（とうざかりこし）

### 当座借越とは

> 銀行と**当座借越契約**を結ぶことで、  
> **当座預金残高が0円を下回っても、上限額まで引き出せる仕組み**

企業にとっての「口座の一時的な赤字」です。

### 当座借越が発生する仕訳

```
取引例：当座預金残高100,000円のとき、
        小切手150,000円を振り出して支払った（当座借越契約あり）。

（借方）買掛金　150,000　／　（貸方）当座預金　150,000
```

→ 当座預金残高は **−50,000円** になる（借越）

### 決算時の当座借越の処理

決算日に当座預金残高がマイナス（借越状態）のとき：

```
当座預金（資産）がマイナス → 「当座借越」という負債勘定へ振り替え

（借方）当座預金　50,000　／　（貸方）当座借越　50,000
```

翌期首に再振替（再振替仕訳）：

```
（借方）当座借越　50,000　／　（貸方）当座預金　50,000
```

```
（借方）（　　　　）　200,000　／　（貸方）（　　　　）　200,000
```

**② 仕入先へ商品代金70,000円を小切手を振り出して支払った。**

```
（借方）（　　　　）　70,000　／　（貸方）（　　　　）　70,000
```

**③ 得意先から売掛金120,000円を同社振出の小切手で回収した。**

```

```

**④ ③の小切手をただちに当座預金に預け入れた。（③と④を合わせて1つの仕訳で行う場合）**

```
（借方）（　　　　）　120,000　／　（貸方）（　　　　）　120,000
```

## ドリル 6-2: 当座借越の処理

当座預金残高が80,000円のとき、小切手120,000円を振り出した（当座借越契約あり）。

**① 振出時の仕訳**

```
（借方）（　　　　）　120,000　／　（貸方）（　　　　）　120,000
```

**② 決算時の当座借越振替仕訳**（借越額40,000円）

```
（借方）（　　　　）　40,000　／　（貸方）（　　　　）　40,000
```

---

### ドリル解答

**6-1**
```
① 当座預金 200,000 ／ 現金 200,000
② 仕入 70,000 ／ 当座預金 70,000
③ 現金 120,000 ／ 売掛金 120,000
④ 当座預金 120,000 ／ 売掛金 120,000
```

**6-2**
```
① 買掛金 120,000 ／ 当座預金 120,000
② 当座預金 40,000 ／ 当座借越 40,000
```

---

## まとめ

| 論点 | 仕訳のポイント |
|------|--------------|
| 当座預金口座開設 | 現金 → 当座預金（資産の振替） |
| 小切手の振出 | 貸方：当座預金（減少） |
| 他社小切手の受取 | 借方：現金（通貨代用証券） |
| ただちに預入 | 借方：当座預金（現金を経由しない） |
| 当座借越の発生 | 当座預金がマイナスになる |
| 決算時の当座借越 | 当座預金（借方）→ 当座借越（負債・貸方） |

次回（W7）は現金・当座預金の **総合演習** と、試験頻出の応用問題に挑戦します。
