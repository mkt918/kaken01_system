# W6: 普通預金・定期預金・当座預金

企業は現金以外にも複数の「銀行口座」を使い分けてお金を管理します。
今週は3種類の預金口座（普通預金・定期預金・当座預金）と、当座預金特有の **当座借越** を学びます。

---

## 3種類の預金口座 比較表

| 項目 | 普通預金 | 定期預金 | 当座預金 |
|------|---------|---------|---------|
| 利息 | あり（低め） | あり（高め） | **なし** |
| 入出金 | 自由 | 満期まで原則不可 | 小切手で引き出す |
| 主な用途 | 日常的な受払い | 余剰資金の運用 | 企業間の大口決済 |

---

## レッスン 1: 普通預金

### 普通預金とは

> 銀行に開設する、自由に入出金できる預金口座

日常的な代金の受け取りや支払い、給料の振込などに使います。

### 普通預金の主な仕訳

**① 口座開設・預け入れ**

```
取引例：銀行に普通預金口座を開設し、現金200,000円を預け入れた。

（借方）普通預金　200,000　／　（貸方）現金　200,000
```

**② 売掛金が振り込まれた（入金）**

```
取引例：得意先から売掛金150,000円が普通預金口座に振り込まれた。

（借方）普通預金　150,000　／　（貸方）売掛金　150,000
```

**③ 口座から引き落とされた（出金）**

```
取引例：買掛金80,000円が普通預金口座から自動引き落とされた。

（借方）買掛金　80,000　／　（貸方）普通預金　80,000
```

**④ 利息の受け取り**

```
取引例：普通預金に利息500円が入金された。

（借方）普通預金　500　／　（貸方）受取利息　500
```

---

## レッスン 2: 定期預金

### 定期預金とは

> **一定期間（満期まで）引き出しができない**代わりに、普通預金より高い利息がつく預金

余剰資金を一定期間運用するときに使います。

### 定期預金の主な仕訳

**① 定期預金への預け入れ**

```
取引例：普通預金口座から500,000円を定期預金に振り替えた。

（借方）定期預金　500,000　／　（貸方）普通預金　500,000
```

**② 満期解約・元本の受け取り**

```
取引例：満期となった定期預金500,000円が普通預金口座に入金された。

（借方）普通預金　500,000　／　（貸方）定期預金　500,000
```

**③ 満期解約・利息も含めて受け取り**

```
取引例：満期となった定期預金500,000円と利息3,000円が
        普通預金口座に入金された。

（借方）普通預金　503,000　／　（貸方）定期預金　500,000
                              （貸方）受取利息　　  3,000
```

> **ポイント**：定期預金は「資産」。預け入れたら定期預金（借方）、解約したら定期預金（貸方）で処理する。

---

## レッスン 3: 当座預金

### 当座預金とは

> 企業間の大口決済に使う口座。**利息はなく、小切手を振り出すことで支払いを行う**

### 当座預金の主な仕訳

**① 口座開設**

```
取引例：銀行と当座取引契約を結び、現金300,000円を預け入れた。

（借方）当座預金　300,000　／　（貸方）現金　300,000
```

**② 小切手を振り出して支払い（自社が支払う）**

```
取引例：商品100,000円を仕入れ、小切手を振り出して支払った。

（借方）仕入　100,000　／　（貸方）当座預金　100,000
```

→ **自社振出の小切手 ＝ 当座預金の減少**

**③ 他社振出の小切手を受け取った（W5の復習）**

```
取引例：得意先から売掛金60,000円を同社振出の小切手で回収した。

（借方）現金　60,000　／　（貸方）売掛金　60,000
```

→ **他社振出の小切手 ＝ 現金（通貨代用証券）**

**④ 他社小切手をただちに当座預金へ預け入れ**

```
取引例：得意先から売掛金60,000円を同社振出の小切手で回収し、
        ただちに当座預金へ預け入れた。

（借方）当座預金　60,000　／　（貸方）売掛金　60,000
```

→ 「**ただちに当座預金へ預入**」 → 現金を経由せず直接当座預金で処理

## レッスン 4: 当座預金出納帳

### 当座預金出納帳とは

> 当座預金の入出金を記録する補助簿。残高の「借」「貸」も記入する。

| 日付 | 摘要 | 預入 | 引出 | 借/貸 | 残高 |
|------|------|------|------|-------|------|
| 4/1 | 前月繰越 | | | 借 | 300,000 |
| 4/3 | 仕入代金支払（小切手） | | 100,000 | 借 | 200,000 |
| 4/8 | 売掛金回収・即時預入 | 60,000 | | 借 | 260,000 |
| 4/20 | 給料支払 | | 150,000 | 借 | 110,000 |

- **借**＝残高がプラス（通常の預金残高あり）
- **貸**＝残高がマイナス（当座借越の状態）

## レッスン 5: 当座借越

### 当座借越とは

> 銀行と**当座借越契約**を結ぶことで、当座預金残高が0円を超えても上限額まで引き出せる仕組み

### 当座借越の発生

```
取引例：当座預金残高50,000円のとき、小切手90,000円を振り出した
        （当座借越契約あり）。

（借方）仕入　90,000　／　（貸方）当座預金　90,000

→ 当座預金残高：50,000 − 90,000 ＝ −40,000円（借越状態）
```

### 決算時の当座借越処理

決算日に当座預金がマイナスの場合 → **当座借越（負債）勘定へ振り替え**

```
（借方）当座預金　40,000　／　（貸方）当座借越　40,000
```

### 翌期首の再振替仕訳

```
（借方）当座借越　40,000　／　（貸方）当座預金　40,000
```

## まとめ

| 預金種類 | 特徴 | 増加（借方） | 減少（貸方） |
|---------|------|------------|------------|
| 普通預金 | 利息あり・自由な入出金 | 入金・振替 | 出金・振替 |
| 定期預金 | 利息高め・満期まで拘束 | 預け入れ | 満期解約 |
| 当座預金 | 利息なし・小切手決済 | 預け入れ・受取 | 小切手振出 |
| 当座借越 | **負債**（借越状態） | 決算振替 | 翌期再振替 |
