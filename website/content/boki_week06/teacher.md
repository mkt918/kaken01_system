# W6 教員用ガイド: 当座預金の取引

## 授業の目標

- 当座預金の特徴（利息なし・小切手で決済）を理解する
- 小切手の振出・受取の仕訳を正確に行える
- 「他社振出小切手＝現金」の原則を再確認する
- 当座借越の発生と決算時の処理ができる

## 指導ポイント

### 「自社振出」vs「他社振出」の徹底区別

3級で最も混乱しやすいポイント。板書で対比する：

```
【他社（得意先）振出の小切手を受け取った】
  → 現金（通貨代用証券）

【自社が小切手を振り出した】
  → 当座預金の減少
```

「誰が振り出したか」が判断基準。

### 「ただちに当座預金に預け入れた」の処理

試験では必ずこの表現が出る。
- 「ただちに」がある → 当座預金で直接処理（現金を経由しない）
- 「ただちに」がない → 一旦現金で受け取り、別途入金処理

### 当座借越の決算処理

「当座預金がマイナスになること」を視覚的に説明：

```
[通常時] 当座預金口座: 残高あり（資産）
[借越時] 当座預金口座: マイナス → 実態は「借金」= 負債

∴ 決算時に「当座借越」（負債）に振り替える
```

### 当座借越契約について

「契約の有無」で処理が変わるわけではない（仕訳は同じ）。
ただし試験問題で「当座借越契約を結んでいる」という前提を与えて
マイナス残高になる取引を出題するパターンが多い。

## ドリル6-1 解答

```
① 当座預金 200,000 ／ 現金 200,000
② 仕入 70,000 ／ 当座預金 70,000
③ 現金 120,000 ／ 売掛金 120,000
④ 当座預金 120,000 ／ 売掛金 120,000
```

## ドリル6-2 解答

```
① 買掛金 120,000 ／ 当座預金 120,000
② 当座預金 40,000 ／ 当座借越 40,000
```

## 試験頻出パターン

1. 小切手受取 → 現金（他社振出）
2. ただちに預入 → 当座預金（現金経由なし）
3. 当座借越の決算振替
4. 当座預金出納帳の残高計算
