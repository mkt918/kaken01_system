# W6 教員用ガイド: 普通預金・定期預金・当座預金

## 授業の目標

- 3種類の預金口座の特徴と違いを理解する
- 普通預金・定期預金・当座預金の仕訳を正確に行える
- 「他社振出小切手＝現金」「ただちに預入＝当座預金」を確実に区別できる
- 当座借越の発生と決算時処理ができる

## 指導ポイント

### 3種類の預金を最初に比較する

板書で比較表を見せてから各論に入ると全体像が掴みやすい。
特に「当座預金だけ利息なし」「小切手で引き出す」という点を強調。

### 定期預金の「資産」としての位置づけ

「定期預金に預けた ＝ 資産を移動させた（現金→定期預金）」と整理する。
満期時は「定期預金（資産）が消えて普通預金（資産）に変わる」だけ。

### 当座預金の2大ルール（W5との接続）

```
他社振出小切手を受け取った  → 現金（W5で学習済み）
ただちに当座預金に預け入れた → 当座預金（現金を経由しない）
```

W5で学んだ「他社小切手＝現金」を土台に、「ただちに預入＝例外的に当座預金」と対比させる。

### 当座借越の視覚的説明

```
[通常] 当座預金: +80,000円（資産）
[振出後] 当座預金: −50,000円 → 実態は「借金」
[決算] 負債の「当座借越」に振り替えて表示を正す
```

## ドリル解答

### ドリル6-1
```
① 普通預金 100,000 ／ 現金 100,000
② 普通預金 230,000 ／ 売掛金 230,000
③ 定期預金 300,000 ／ 普通預金 300,000
④ 普通預金 302,000 ／ 定期預金 300,000
                     受取利息   2,000
```

### ドリル6-2
```
① 当座預金 400,000 ／ 現金 400,000
② 買掛金 120,000 ／ 当座預金 120,000
③ 現金 70,000 ／ 売掛金 70,000
④ 当座預金 90,000 ／ 売掛金 90,000
```

### ドリル6-3
```
① 仕入 130,000 ／ 当座預金 130,000
② 当座預金 50,000 ／ 当座借越 50,000
```

## 試験頻出パターン

1. 定期預金の満期解約（利息込み）
2. 他社小切手受取 → 現金 vs ただちに預入 → 当座預金
3. 当座借越の決算振替仕訳
4. 当座預金出納帳の残高計算
