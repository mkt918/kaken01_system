## 復習チェックシート

まず、以下の問いに答えられるか確認しましょう。

| チェック項目 | 確認 |
|-------------|------|
| 「他社振出の小切手」を受け取ったとき → **現金** | □ |
| 「自社が小切手を振り出した」とき → **当座預金の減少** | □ |
| 現金が帳簿より少ない → **借方：現金過不足** | □ |

| 小切手をただちに当座預金へ預入 → **当座預金で直接処理** | □ |
| 当座借越の決算時振替 → **当座借越（負債）** | □ |

## 総合演習 問題4: 現金過不足の総合問題

次の一連の流れについて、各時点の仕訳を行いなさい。

**状況**：
- 6月15日：現金残高を確認すると、帳簿残高220,000円に対し実際有高は215,000円だった。
- 6月20日：差額5,000円のうち、3,000円は交通費の記帳漏れと判明した。
- 12月31日（決算日）：残り2,000円の原因は不明のままだった。

**① 6月15日 現金不足発見時**

```
（借方）（　　　　）　（　　）　／　（貸方）（　　　　）　（　　）
```

**② 6月20日 原因判明時**

```
（借方）（　　　　）　（　　）　／　（貸方）（　　　　）　（　　）
```

**③ 12月31日 決算時（残額2,000円の処理）**

```
（借方）（　　　　）　（　　）　／　（貸方）（　　　　）　（　　）
```

### 解答

```
① 現金過不足 5,000 ／ 現金 5,000
② 旅費交通費 3,000 ／ 現金過不足 3,000
③ 雑損 2,000 ／ 現金過不足 2,000
```

**解説**：
- ①で借方に現金過不足5,000円が計上される
- ②で3,000円が旅費交通費へ振り替えられ、現金過不足の残高は2,000円（借方）
- ③決算で借方残2,000円→雑損（費用）へ振り替え、現金過不足勘定をゼロにする

## W5〜W7 総まとめ表

```
┌──────────────────────────────────────────────────────────────┐
│              現金・当座預金 処理フローチャート                  │
│                                                              │
│  お金を受け取った                                            │
│    ├─ 紙幣・硬貨                         → 現金             │
│    ├─ 他社振出の小切手                   → 現金             │
│    ├─ 他社小切手をただちに預入           → 当座預金          │
│    └─ 当座預金口座に振り込まれた         → 当座預金          │
│                                                              │
│  お金を支払った                                              │
│    ├─ 現金（紙幣・硬貨）で支払い         → 現金の減少        │
│    └─ 小切手を振り出して支払い           → 当座預金の減少    │
│                                                              │
│  帳簿と実際が違う（現金）                                    │
│    ├─ 不足 → 現金過不足（借方）                              │
│    ├─ 過剰 → 現金過不足（貸方）                              │
│    ├─ 原因判明 → 正しい勘定へ振替                            │
│    └─ 決算・不明 → 雑損（不足）/ 雑益（過剰）               │
│                                                              │
│  当座預金がマイナス（当座借越）                              │
│    └─ 決算 → 当座借越（負債）へ振替                          │
└──────────────────────────────────────────────────────────────┘
```

## 試験直前チェック

- [ ] 他社振出小切手 → 現金（通貨代用証券）
- [ ] 自社振出小切手 → 当座預金の減少
- [ ] ただちに当座預金へ預入 → 現金経由なし
- [ ] 現金過不足（不足）→ 借方：現金過不足
- [ ] 現金過不足（過剰）→ 貸方：現金過不足
- [ ] 決算・原因不明・不足 → 雑損（費用）
- [ ] 決算・原因不明・過剰 → 雑益（収益）
- [ ] 当座借越の決算振替 → 当座借越（負債）
