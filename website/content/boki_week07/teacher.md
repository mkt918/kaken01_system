# W7 教員用ガイド: 現金・当座預金 総合演習

## 授業の目標

- W5・W6の知識を定着させる
- 試験形式の仕訳問題（10問）を時間内に解ける
- 現金出納帳・当座預金出納帳を正確に記入できる
- 現金過不足の一連の流れ（発生〜決算）を理解する

## 時間配分（50分授業）

| 時間 | 内容 |
|------|------|
| 0〜5分 | 復習チェックシートで確認 |
| 5〜20分 | 問題1（仕訳10問）→ 個人解答→解説 |
| 20〜30分 | 問題2（現金出納帳）→ 解説 |
| 30〜40分 | 問題3（当座預金出納帳）→ 解説 |
| 40〜48分 | 問題4（現金過不足総合）→ 解説 |
| 48〜50分 | まとめ表の確認・次週予告 |

## 指導ポイント

### 演習の進め方

- 問題1は最初に5〜7分で個人解答させ、採点させてから解説
- 間違いが多い問題（3・4・5・6番あたり）を重点解説
- 帳簿記入（問題2・3）は実際に書かせることで定着する

### 問題1 でつまずく問題の傾向

| 問題番号 | よくある誤り | 正しい考え方 |
|---------|-------------|-------------|
| 3 | 当座預金で処理 | 「他社振出」→ 現金 |
| 4 | ③と別仕訳 | 「ただちに預入」→ 当座預金で一本化 |
| 5 | 貸方：現金過不足 | 実際>帳簿（過剰）→ 借方：現金 |
| 6 | 現金過不足のまま | 原因判明→ 正しい勘定へ振替 |
| 10 | 当座預金（負債？） | 資産勘定が負債になる → 当座借越 |

### 問題4（現金過不足総合）の解説ポイント

黒板にT字勘定を書いて「現金過不足の残高がどう動くか」を追跡する：

```
      現金過不足
  借方  |  貸方
① 5,000 |
        | ② 3,000　← 旅費交通費へ振替（解消）
残高2,000|
③ 2,000 は雑損へ（決算で解消）
→ 期末残高はゼロになる
```

### 帳簿記入の採点基準

- 金額が正しい（2点）
- 残高の計算が正しい（1点）
- 借/貸の表記が正しい（1点）（当座預金出納帳のみ）

4.  当座預金 60,000 ／ 売掛金 60,000

6.  現金過不足 3,000 ／ 受取手数料 3,000
7.  水道光熱費 12,000 ／ 現金 12,000
8.  給料 200,000 ／ 当座預金 200,000
9.  仕入 80,000 ／ 当座預金 80,000
10. 当座預金 30,000 ／ 当座借越 30,000
```

```
