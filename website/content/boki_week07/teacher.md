# W7 教員用ガイド: 小口現金

## 授業の目標

- 小口現金の目的と仕組み（インプレスト・システム）を理解する
- 前渡し・報告・補給・報告＋即時補給の4パターンを仕訳できる
- 小口現金出納帳を正確に記入できる
- 「報告＋即時補給では小口現金が登場しない」理由を説明できる

## 時間配分（50分授業）

| 時間 | 内容 |
|------|------|
| 0〜5分 | 前回復習：普通預金・定期預金・当座預金のクイズ |
| 5〜15分 | 小口現金の仕組み・インプレスト制の説明 |
| 15〜25分 | 前渡し・報告・補給の仕訳（3パターン） |
| 25〜35分 | 報告＋即時補給の仕訳（最重要パターン） |
| 35〜45分 | 小口現金出納帳の記入演習（ドリル7-2） |
| 45〜50分 | ドリル7-1・まとめ |

## 指導ポイント

### 「報告＋即時補給」の仕訳が試験の山

最も重要なポイント。黒板で2段階に分けて説明する：

```
【分解して考える】
報告時：旅費交通費 800 ／ 小口現金 800   ← ①
補給時：小口現金  800 ／ 当座預金  800   ← ②

①と②を合算すると「小口現金」が相殺されて消える：
→ 旅費交通費 800 ／ 当座預金 800
```

「なぜ小口現金が出てこないか」を分解で説明すると腑に落ちやすい。

### 小口現金出納帳の「内訳欄」に注目

費用の種類ごとに欄を分けて記入する点は、現金出納帳にはない特徴。
試験では「費用の種類を正確に記入できるか」も問われる。

### 勘定科目の注意

- 「交通費」ではなく **「旅費交通費」** が3級の正式科目名
- 「切手代・電話代」→ **「通信費」**
- 「コピー用紙・事務用品」→ **「消耗品費」**
- 「お茶代・その他少額」→ **「雑費」**

## 試験頻出パターン

1. 前渡しの仕訳（当座預金 or 現金）
2. 報告のみの仕訳（小口現金が貸方）
3. **報告＋即時補給の仕訳**（小口現金登場なし）← 最頻出
4. 小口現金出納帳の残高計算
