# W1: 原価と原価計算 ― 「製造業のお金の記録」入門

工場では毎日、材料を仕入れ、機械で加工し、製品を作り出しています。
では、その製品を1つ作るのに**いくらかかったのか**、どうやって計算するのでしょうか？
今週は、工業簿記・原価計算の世界への入口を学びましょう。

---

## レッスン 1: 商業簿記と工業簿記

### 商業簿記とは？

> **商業簿記**とは、仕入れ先から商品を仕入れ、その商品をそのまま得意先へ販売する**商業（小売業・卸売業）**で使われる簿記です。

```
【商業簿記の流れ】

仕入れ先 → 商品を仕入れる → 得意先へ販売
```

商品に加工は行いません。仕入れた商品をそのまま売ります。

### 工業簿記とは？

> **工業簿記**とは、仕入れ先から**材料**を仕入れ、機械などを使って加工し、**製品**を製造して得意先へ販売する**製造業**などで使われる簿記です。

```
【工業簿記の流れ】

仕入れ先 → 材料を仕入れる → 加工・製造 → 製品が完成 → 得意先へ販売
```

### 商業簿記と工業簿記の比較

| 項目 | 商業簿記 | 工業簿記 |
|------|---------|---------|
| **主な業種** | 小売業・卸売業 | 製造業 |
| **仕入れるもの** | 商品（そのまま売れる） | 材料（加工が必要） |
| **販売するもの** | 仕入れた商品 | 自社で製造した製品 |
| **特徴** | シンプルな流れ | 製造工程が複雑 |

### 考え方の方向性の違い

簿記の目的は同じ「記録」ですが、アプローチが異なります。

| | 商業簿記 | 工業簿記 |
|--|---------|---------|
| **方向性** | 過去 → 現在（結果） | 現在 → 未来（予算・計画） |
| **中心となる作業** | 起きた出来事を仕訳する | 製品コストを予測・管理する |
| **最終目標** | 決算（利益の確定） | 原価管理・利益計画 |

商業簿記は「起きた出来事に対して仕訳を行い、決算に向かう」のに対し、
工業簿記は「最終結果（利益目標）に向かって、予定や予算を立てながら管理する」のが特徴です。

## レッスン 2: 工業簿記と原価計算

### 原価計算が必要な理由

製造活動は、企業の**内部**で行われる活動です。商業とは違い、製品を作るのにどれだけお金がかかったかを、**自ら計算する**必要があります。

例えば、1本のボールペンを作るには…

| かかるコスト | 具体例 |
|------------|--------|
| 材料費 | プラスチック、インク、バネ |
| 労務費 | 工場で働く人の給料 |
| 経費 | 機械の電気代、工場の家賃 |

このように**製品1つを作るのにかかった金額の合計**が **「原価（げんか）」** です。

> **原価計算**とは、製品の製造にかかった原価を正確に計算・集計する手続きのことです。

### 原価計算はどんな業種で使える？

原価計算は製造業だけのものではありません！

| 業種 | 「製品」に相当するもの | 原価の例 |
|------|-------------------|---------|
| 製造業 | ボールペン、車など | 材料費・労務費・電気代 |
| ホテル業 | 1泊の宿泊サービス | 従業員給料・家賃・光熱費 |
| 病院 | 診察・治療サービス | 医師・看護師の給料・医薬品代 |
| 学校 | 1コマの授業 | 教師の給料・設備費 |

どんな業種でも「そのサービス・製品を提供するのにいくらかかったか」を計算し、**今後の利益計画を立てる**ことができます。

### 原価の種類：製造原価・販売費・一般管理費・総原価

製品に関わるコストには、大きく分けて3種類あります。

```
総原価 = 製造原価 + 販売費 + 一般管理費
```

| 費用の種類 | 内容 | 例 |
|----------|------|-----|
| **製造原価** | 製品を**製造する**ためにかかった費用 | 材料費・工場の人件費・電気代 |
| **販売費** | 製品を**販売する**ためにかかった費用 | 広告費・営業担当者の給料・配送費 |
| **一般管理費** | 会社**全体を管理する**ためにかかった費用 | 社長・事務員の給料・オフィス家賃 |

> **製造原価** が「工業簿記・原価計算」でメインに扱う原価です。

## ドリル 1-1: 商業簿記 vs 工業簿記

次の説明が「商業簿記」に当てはまるか「工業簿記」に当てはまるか答えましょう。

```
（  ）① コンビニが飲料メーカーから飲み物を仕入れて販売した

（  ）② 自動車メーカーが鉄鋼や部品を組み立てて車を製造し販売した

（  ）③ 衣料品店が衣料メーカーから服を仕入れてそのまま販売した

（  ）④ 食品メーカーが農家から野菜を仕入れ、加工して冷凍食品を製造した
```

**答え：①③ → 商業簿記　②④ → 工業簿記**

## ドリル 1-2: 原価の分類

次のコストを「製造原価」「販売費」「一般管理費」に分類しましょう。

```
（  ）① 工場で使う金属材料の購入費

（  ）② テレビCMの広告費

（  ）③ 工場の電気代

（  ）④ 経理担当者（本社勤務）の給料

（  ）⑤ 工場で働く従業員の給料

（  ）⑥ 製品を得意先へ届けるトラックの燃料代
```

- 製造原価：① ③ ⑤
- 販売費：② ⑥
- 一般管理費：④

</details>

## まとめ

| キーワード | 意味 |
|-----------|------|
| 商業簿記 | 仕入れた商品をそのまま販売する商業で使われる簿記 |
| 工業簿記 | 材料を加工して製品を製造・販売する製造業で使われる簿記 |
| 原価 | 製品を製造するためにかかった金額（製造原価） |
| 原価計算 | 製品1つあたりの製造原価を計算・集計する手続き |
| 製造原価 | 製品の製造にかかった費用（材料費・労務費・経費） |
| 販売費 | 製品の販売にかかった費用 |
| 一般管理費 | 会社全体の管理にかかった費用 |
| 総原価 | 製造原価 ＋ 販売費 ＋ 一般管理費 |
