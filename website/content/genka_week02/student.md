# W2: 原価計算のあらまし ― 「原価はどう分類するのか」

製品を作るためにかかったコスト（原価）は、どのような視点で整理するかによって分類方法が変わります。
今週は、原価の代表的な分類方法を2つ学びましょう。

---

## レッスン 1: 発生形態別分類（材料費・労務費・経費）

### 発生形態別分類とは？

> **発生形態別分類**とは、「何を使ったために発生した原価なのか」という基準で、原価を分類する方法です。

この方法では、製造原価を次の **3つ** に分類します。

| 分類 | 内容 | 具体例 |
|------|------|--------|
| **材料費** | 製品の製造に使用した**物品**の消費高 | 鉄板、木材、プラスチック |
| **労務費** | 製品の製造に使用した**労働力**の対価 | 工場作業員の賃金・給料 |
| **経費** | 材料費・労務費**以外**のすべての原価 | 電気代・設備の減価償却費 |

### 各費目の詳細

#### 材料費
製品を製造するために消費される物品（材料）のコストです。

```
例：
・鉄鋼メーカー → 鉄板・鉄鉱石の購入費
・食品メーカー → 小麦粉・砂糖などの原材料費
・家具メーカー → 木材・ネジの購入費
```

#### 労務費
製品の製造に携わる労働者に支払う賃金などです。
工場で働く人への支払いが中心になります。

```
例：
・工場の組立作業員の賃金
・品質検査担当者の給料
・製造ラインの監督者の手当
```

#### 経費
材料費でも労務費でもないコストをまとめて「経費」といいます。

```
例：
・工場の電気代・水道代（光熱費）
・機械・設備の減価償却費
・工場建物の賃借料
・外注加工費（外部に加工を依頼した費用）
```

## レッスン 2: 製品との関連による分類（直接費・間接費）

### 直接費と間接費とは？

製品ごとに「このコストはいくらかかったか」を**直接計算できるかどうか**で分類します。

| 分類 | 意味 | 例 |
|------|------|----|
| **製造直接費** | 特定の製品への消費額を**直接計算できる**原価 | 製品Aに使った鉄板の量が明確 |
| **製造間接費** | 複数の製品に共通して発生し、**直接計算できない**原価 | 工場全体の電気代（どの製品に何円かは分からない） |

### 3分類 × 2分類 = 6つの原価分類

材料費・労務費・経費のそれぞれに「直接・間接」を組み合わせると、**6つの分類**になります。

| | 直接費 | 間接費 |
|--|--------|--------|
| **材料費** | 直接材料費 | 間接材料費 |
| **労務費** | 直接労務費 | 間接労務費 |
| **経費** | 直接経費 | 間接経費 |

#### 各分類のイメージ

| 分類 | 具体例 |
|------|--------|
| 直接材料費 | 製品Aだけに使った鉄板（どの製品に何kg使ったか分かる） |
| 間接材料費 | 工場全体で使う機械油・補助材料（どの製品に使ったか分からない） |
| 直接労務費 | 製品Aの組立だけをした作業員の賃金（何時間かかったか分かる） |
| 間接労務費 | 工場全体を管理する監督者の給料（どの製品に何時間かは分からない） |
| 直接経費 | 製品Aの設計だけに依頼した外注加工費 |
| 間接経費 | 工場全体の電気代・減価償却費（どの製品に何円かは分からない） |

## レッスン 3: 操業度による分類（変動費・固定費）

原価は、生産量（**操業度**）の変化に応じてどう変わるかでも分類できます。

### 基本の2分類

| 分類 | 内容 | 例 |
|------|------|----|
| **変動費** | 操業度の増減に比例して増減する原価 | 材料費（作れば作るほど増える） |
| **固定費** | 操業度に関わらず一定額発生する原価 | 工場の家賃・設備の減価償却費 |

```
【イメージ図】

  原価
   ↑
   │        変動費（操業度に比例↗）
   │      ／
   │    ／
   │  ／
   │／─────────────── 固定費（一定）
   └──────────────→ 操業度（生産量）
```

### 応用の2分類（参考）

| 分類 | 内容 | 例 |
|------|------|----|
| **準変動費** | 操業度ゼロでも一定額発生し、さらに操業度に比例して増加するもの | 電気代（基本料金＋使用量分） |
| **準固定費** | ある操業度の範囲内では固定だが、一定量を超えると段階的に増加するもの | パート従業員の増員（一定量を超えたら追加雇用） |

> **試験対策ポイント**：変動費と固定費の違いをしっかり押さえましょう。準変動費・準固定費は参考程度で OK です。

## ドリル 2-1: 発生形態別に分類しよう

次のコストを「材料費」「労務費」「経費」に分類しましょう。

```
（  ）① 工場で使うアルミ板の購入費

（  ）② 工場の電気代

（  ）③ 製造ラインの作業員への賃金

（  ）④ 工場の機械の減価償却費

（  ）⑤ 製品に貼り付けるラベルシールの購入費

（  ）⑥ 工場長の給料
```

**答え：**
- 材料費：① ⑤
- 労務費：③ ⑥
- 経費：② ④

---

## ドリル 2-2: 直接費・間接費に分類しよう

次のコストを「直接費」「間接費」に分類し、さらに費目（材料費・労務費・経費）も答えましょう。

```
（         ）① 製品Aの組立だけをした作業員の賃金

（         ）② 工場全体の照明の電気代

（         ）③ 製品Bだけに使用した特殊鋼材の購入費

（         ）④ 複数の製品ラインを管理する工場長の給料
```

**答え：**
- ① 直接労務費（製品Aの組立のみ → 直接計算できる）
- ② 間接経費（工場全体 → どの製品に何円か分からない）
- ③ 直接材料費（製品Bのみに使用 → 直接計算できる）
- ④ 間接労務費（複数製品に関わる → 直接計算できない）

## まとめ

| キーワード | 意味 |
|-----------|------|
| 発生形態別分類 | 何を使ったかで原価を分類（材料費・労務費・経費） |
| 材料費 | 製造に使う物品の消費高 |
| 労務費 | 製造に使う労働力の対価（賃金など） |
| 経費 | 材料費・労務費以外のすべての原価 |
| 製造直接費 | 製品ごとに直接計算できる原価 |
| 製造間接費 | 製品ごとに直接計算できない原価（複数製品で共通） |
| 変動費 | 操業度に比例して増減する原価 |
| 固定費 | 操業度に関わらず一定額発生する原価 |
