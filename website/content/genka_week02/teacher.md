## 指導ポイント

### 導入の発問例

> 「工場でスマートフォンを1台作るとき、どんなコストがかかると思う？それをどうやって3つに分ける？」

W1で学んだ「材料・給料・電気代」を想起させ、今週の3分類（材料費・労務費・経費）に自然につなげる。

### 発生形態別分類で混乱しやすい点

**誤解①**「給料は全部労務費では？」
→ 本社の事務員や営業の給料は「一般管理費」や「販売費」。工場で製造に携わる人の給料が「労務費」。

**誤解②**「経費って何でも入るの？」
→「材料費でも労務費でもないもの」と整理。電気代・減価償却費・外注費などが代表例。

### 直接費・間接費の教え方

「どの製品に何円かかったか、ハッキリ分かるか？」という問いで判断させる。

**板書例**
```
製品A 専用の材料 → 材料費が明確 → 直接材料費 ✅
工場全体の電気代 → どの製品に何円か不明 → 間接経費 ✅
```

6分類の表は板書して視覚的に整理させる。

### 変動費・固定費のイメージ化

**身近な例で説明**
- 変動費：「材料費はモノを多く作るほど増える（コンビニのおにぎりの具材）」
- 固定費：「工場の家賃は製品を1個も作らなくてもかかる（使わなくても払う）」

## ドリル解答と解説

### ドリル 2-1: 発生形態別分類

```
① アルミ板の購入費 → 材料費
② 工場の電気代 → 経費
③ 製造ラインの作業員の賃金 → 労務費
④ 機械の減価償却費 → 経費
⑤ ラベルシールの購入費 → 材料費（補助材料）
⑥ 工場長の給料 → 労務費
```

**ポイント**：減価償却費は「モノ（設備）を使ったコスト」だが、現金の支出ではないため材料費ではなく経費に分類する。

### ドリル 2-2: 直接費・間接費

```
① 製品A専用の作業員の賃金 → 直接労務費
② 工場全体の電気代 → 間接経費
③ 製品B専用の特殊鋼材 → 直接材料費
④ 複数製品を管理する工場長の給料 → 間接労務費
```

**ポイント**：「特定の製品にだけ対応している → 直接費」「複数製品にまたがる → 間接費」。

## 補足資料

### 6分類の整理表（板書用）

| | 直接費 | 間接費 |
|--|--------|--------|
| 材料費 | 直接材料費 | 間接材料費 |
| 労務費 | 直接労務費 | 間接労務費 |
| 経費 | 直接経費 | 間接経費 |

### 変動費・固定費の試験対策

- 変動費の代表：直接材料費、直接労務費（出来高払いの場合）
- 固定費の代表：減価償却費、工場の家賃、固定給の工場長給料
- 準変動費・準固定費は参考程度（出題頻度低）

### 次週（W3）の予告

W3では「材料費の計算」を学ぶ。
材料の購入から消費までの流れ、消費高の計算方法（先入先出法・平均法）を扱う。
今週の「直接材料費・間接材料費」の区別が前提知識になる。
