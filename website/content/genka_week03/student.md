# W3: 勘定連絡図 ― 「原価はどう流れて利益になるのか」

材料・人件費・電気代などのコストは、製品が完成し売れるまでの間、どのように帳簿の中を流れていくのでしょうか？
今週は**勘定連絡図**を使って、原価の流れを一本の流れとして理解します。

## レッスン 1: 勘定連絡図とは

**勘定連絡図**とは、製造業における原価の流れを勘定科目ごとに図で示したものです。
「材料を仕入れる」→「製品を作る」→「完成する」→「売れる」→「利益が出る」という一連の流れを**帳簿（勘定）の動き**として表します。

```
【勘定連絡図の全体像】

 材料費 ─┐
 労務費 ─┼──→【製造間接費】──┐
 経　費 ─┘                    │
                               ↓（配賦）
 直接材料費 ─┐               ↓
 直接労務費 ─┼──────────→【仕 掛 品】──→【製 品】──→【売上原価】
 直接経　費 ─┘                                              ↓
                                              【販売費及び一般管理費】──→【損益（利益）】
                                                                      ↑
                                                                  【売 上 高】
```

---

## レッスン 2: 製造間接費への集計

W2で学んだ6分類のうち、**間接費**にあたるものをすべてまとめた勘定が**製造間接費**です。

### 製造間接費に集まるもの

| 費目 | 内容 |
|------|------|
| **間接材料費** | 工場全体で使う機械油・補助材料など |
| **間接労務費** | 工場長の給料・複数ラインを管理する監督者の賃金など |
| **間接経費** | 工場全体の電気代・減価償却費・賃借料など |

```
【製造間接費への集計】

 間接材料費 ─┐
 間接労務費 ─┼──→【製造間接費】
 間接経　費 ─┘
```

> **ポイント**：間接費は「どの製品に何円かかったか直接分からない」ため、いったん**製造間接費**に集めてから配賦します。

## レッスン 3: 仕掛品への集計（製造間接費の配賦）

集まった製造間接費は、一定の**配賦基準**（直接作業時間・機械稼働時間など）で各製品に按分され、**仕掛品**勘定へ移ります。

同時に、**直接費**（直接材料費・直接労務費・直接経費）もそのまま**仕掛品**へ移ります。

```
【仕掛品への流れ】

 直接材料費 ─┐
 直接労務費 ─┼──────────────────→【仕 掛 品】
 直接経　費 ─┘                              ↑
                                             │（配賦）
 間接材料費 ─┐                              │
 間接労務費 ─┼──→【製造間接費】────────┘
 間接経　費 ─┘
```

### 仕掛品とは？

> **仕掛品（しかかりひん）**とは、製造の途中にある、まだ完成していない製品のことです。

```
例：
・自動車工場でボディの溶接まで終わったが、
  内装や検査がまだの状態 → 仕掛品
・パン工場で生地をこねたが、まだ焼いていない状態 → 仕掛品
```

**仕掛品勘定の中身**

```
【仕掛品】
─────────────────────────────────────
 直接材料費（賦課）      ↑
 直接労務費（賦課）      │ ← 直接費がそのまま入る
 直接経費（賦課）        │
 製造間接費（配賦）      ↑ ← 間接費が配賦されて入る
─────────────────────────────────────
 ↓ 完成したら「製品」へ
```

---

## レッスン 4: 製品への振替（完成）

仕掛品のうち、製造が**完成した分だけ**が**製品**勘定へ振り替えられます。

```
【仕掛品 → 製品】

【仕 掛 品】
 ─────────────
  直接費        │─→ 完成分 ─→【製 品】
  製造間接費    │
 ─────────────
  未完成分は仕掛品のまま残る
```

> **製品**とは：完成した製品で、まだ販売していないもの。倉庫に在庫として保管されている状態。

| 状態 | 勘定科目 |
|------|---------|
| 製造中（未完成） | 仕掛品 |
| 完成・在庫あり（未販売） | 製品 |
| 販売済み | 売上原価 |

---

## レッスン 5: 売上原価への振替（販売）

製品が**販売された時点**で、はじめて**売上原価**勘定へ振り替えられます。
販売前の製品はあくまで「資産（在庫）」であり、費用にはなりません。

```
【製品 → 売上原価】

【製 品】（在庫・資産）
 ─────────────
  完成した製品   │─→ 売れた分だけ ─→【売 上 原 価】（費用）
 ─────────────
  売れ残りは製品のまま
```

---

## レッスン 6: 損益計算（利益の計算）

販売された製品の原価（売上原価）に**販売費及び一般管理費**を加えた合計が、その期間に実際にかかった**総費用**です。
この総費用と**売上高**を比べることで**利益**が計算されます。

```
売上高
  ─ 売上原価
  ─────────────────────
  = 売上総利益（粗利）

  ─ 販売費及び一般管理費
  ─────────────────────
  = 営業利益
```

### 各項目の意味

| 項目 | 内容 | 例 |
|------|------|-----|
| **売上高** | 製品を販売して得た収益 | 製品を1,000円で100個販売 → 100,000円 |
| **売上原価** | 販売した製品の製造原価 | 1個800円の製品を100個分 → 80,000円 |
| **売上総利益** | 売上高 − 売上原価 | 100,000 − 80,000 = 20,000円 |
| **販売費及び一般管理費** | 販売活動・会社管理にかかった費用 | 広告費・営業給料・本社家賃など |
| **営業利益** | 売上総利益 − 販売費及び一般管理費 | 20,000 − 8,000 = 12,000円 |

## レッスン 7: 勘定連絡図まとめ

原価が材料から利益になるまでの全体の流れをもう一度整理します。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 STEP 1: 費用の発生
  材料費・労務費・経費 が発生
   → 直接費 と 間接費 に分類

 STEP 2: 製造間接費へ集計
  間接材料費 + 間接労務費 + 間接経費 → 【製造間接費】

 STEP 3: 仕掛品へ集計
  直接費（賦課）+ 製造間接費（配賦）→ 【仕掛品】

 STEP 4: 製品へ振替（完成）
  仕掛品のうち完成した分 → 【製品】（在庫・資産）

 STEP 5: 売上原価へ振替（販売）
  製品のうち売れた分 → 【売上原価】（費用）

 STEP 6: 損益計算（利益の計算）
  売上高 − 売上原価 − 販売費及び一般管理費 ＝ 営業利益
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## ドリル 3-1: 勘定の流れを並べよう

次のア〜カを、勘定連絡図の流れの順番に並べましょう。

```
ア. 売上原価
イ. 製品（完成・在庫）
ウ. 仕掛品
エ. 製造間接費
オ. 材料費・労務費・経費（発生）
カ. 損益（営業利益）
```

<details>
<summary>答えを見る</summary>

**答え：オ → エ → ウ → イ → ア → カ**

発生 → 製造間接費 → 仕掛品 → 製品 → 売上原価 → 利益

</details>

## ドリル 3-2: 直接費・間接費の行き先を答えよう

次のコストは、まず「製造間接費」に集まりますか？それとも直接「仕掛品」へ移りますか？

```
（     ）① 製品A専用の鉄板代（直接材料費）
（     ）② 工場全体の電気代（間接経費）
（     ）③ 製品B組立専任の作業員の賃金（直接労務費）
（     ）④ 工場長の給料（間接労務費）
（     ）⑤ 製品C専用の外注加工費（直接経費）
（     ）⑥ 工場全体で使う機械油代（間接材料費）
```

<details>
<summary>答えを見る</summary>

- 直接「仕掛品」へ：① ③ ⑤（直接費 → 賦課）
- 「製造間接費」を経由：② ④ ⑥（間接費 → 製造間接費に集計 → 配賦 → 仕掛品）

</details>

## ドリル 3-3: 勘定科目を答えよう

次の状態にある製品は、どの勘定科目で処理されますか？

```
（     ）① 工場で製造中だが、まだ完成していない
（     ）② 完成して倉庫に保管されているが、まだ売っていない
（     ）③ 顧客に販売済み
```

<details>
<summary>答えを見る</summary>

- ① 仕掛品（製造途中 = 未完成）
- ② 製品（完成・未販売 = 在庫資産）
- ③ 売上原価（販売済み = 費用）

</details>

## まとめ

| キーワード | 意味 |
|-----------|------|
| 勘定連絡図 | 原価が材料から利益になるまでの流れを勘定で示した図 |
| 製造間接費 | 間接材料費・間接労務費・間接経費を集めた勘定 |
| 賦課 | 直接費を特定の仕掛品にそのまま割り当てること |
| 配賦 | 製造間接費を基準で按分して仕掛品に割り当てること |
| 仕掛品 | 製造途中でまだ完成していない製品 |
| 製品 | 完成したが、まだ販売していない在庫 |
| 売上原価 | 販売した製品の製造原価（販売時点で費用になる） |
| 売上総利益 | 売上高 − 売上原価 |
| 販売費及び一般管理費 | 販売活動・会社管理にかかった費用 |
| 営業利益 | 売上総利益 − 販売費及び一般管理費 |
