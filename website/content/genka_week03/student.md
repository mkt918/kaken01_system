# W3: 原価計算のあらまし② ― 「原価計算の流れと種類」

原価計算にはどのような手順があり、どんな種類があるのでしょうか？
今週は原価計算の全体の流れと、3つの対比で整理される計算方法の種類を学びます。

---

## レッスン 1: 原価計算の流れ（3段階）

製造原価の計算は、次の **3つの段階** を順番に行います。

```
① 費目別計算  →  ② 部門別計算  →  ③ 製品別計算
```

### ① 費目別計算

> 原価を「材料費・労務費・経費」に分類し、それぞれの**消費高**を計算する手続きです。

W2で学んだ3分類（発生形態別分類）がここで使われます。

### ② 部門別計算

> 費目別に計算した原価要素のうち、**製造間接費**を工場の**部門ごと**に分類・集計する手続きです。

「切削部門」「組立部門」「仕上部門」など、工場内の各部門に間接費を割り当てます。

### ③ 製品別計算

> 製品ごとに原価を集計し、**1単位あたりの製造原価**を計算する手続きです。

各部門に集まった原価を、最終的に製品1個あたりのコストへ落とし込みます。

---

### 原価計算の流れ（全体像）

```
【材料費】┐
【労務費】┼─→ ① 費目別計算 ─→ ② 部門別計算 ─→ ③ 製品別計算
【経　費】┘     (3分類へ整理)    (部門ごとに集計)  (製品1個の原価)
```

### 計算期間について

| 目的 | 計算期間 |
|------|---------|
| 財務諸表の作成（外部報告） | 1年間 |
| 内部管理・原価管理 | **1か月単位**（こちらが主流） |

商業簿記と同じく1年単位で財務諸表を作りますが、製造管理のために**毎月計算する**のが実務の標準です。

---

## レッスン 2: 個別原価計算と総合原価計算

製品別計算の具体的な方法は、生産形態によって2種類に分かれます。

### 個別原価計算

> **受注生産（オーダーメイド）**の企業で使われる方法です。

注文ごとに製品が異なるため、1つ1つの注文（製造指図書）ごとに原価を集計します。

```
例：
・建設業（ビルの建設）
・造船業（船の建造）
・特注機械メーカー
・広告制作会社
```

| 特徴 | 内容 |
|------|------|
| 生産形態 | 個別受注生産（オーダーメイド） |
| 原価集計 | 注文（製造指図書）ごとに集計 |
| 向いている業種 | 建設・造船・特注品製造 |

### 総合原価計算

> **大量見込み生産**の企業で使われる方法です。

同じ製品を大量に継続して作るため、1か月間の製造原価合計を完成個数で割って1個あたりの原価を求めます。

```
1個あたりの製造原価 = 1か月の製造原価合計 ÷ 1か月の完成個数
```

```
例：
・飲料メーカー（ペットボトル飲料）
・自動車メーカー（大量生産の乗用車）
・食品メーカー（袋入りスナック菓子）
```

| 特徴 | 内容 |
|------|------|
| 生産形態 | 大量見込み生産 |
| 原価集計 | 1か月分をまとめて集計し個数で割る |
| 向いている業種 | 飲料・食品・自動車など大規模工場 |

### 比較表

| | 個別原価計算 | 総合原価計算 |
|--|-------------|-------------|
| 生産形態 | 受注生産（オーダーメイド） | 大量見込み生産 |
| 原価の集計単位 | 注文（製造指図書）ごと | 1か月ごと |
| 1個の原価の求め方 | 各注文の集計額がそのまま製品原価 | 合計 ÷ 完成個数 |

## レッスン 3: 実際原価計算と標準原価計算

**実際にかかった原価**で計算するか、**目標原価**で計算するかの違いです。

| | 実際原価計算 | 標準原価計算 |
|--|-------------|-------------|
| 内容 | 実際にかかった原価をそのまま集計 | 目標となる原価（原価標準）を事前に設定し、それで計算 |
| 目的 | 実績の把握 | 目標 vs 実績の差異分析・コスト管理 |
| 特徴 | シンプル・確実 | 経営管理に活用しやすい |

> **原価標準**：製品1単位を製造するのに「かかるべきコスト」として事前に定めた目標原価のこと。

標準原価計算では「目標よりも多くかかったか・少なかったか」を分析し、無駄やコスト超過の原因を探ります。

---

## レッスン 4: 全部原価計算と部分原価計算

製品の製造原価に**すべてのコストを含める**か、**一部だけ**にするかの違いです。

| | 全部原価計算 | 部分原価計算 |
|--|-------------|-------------|
| 内容 | 製造にかかった**すべての原価**で製品原価を計算 | **一部の原価**（例：変動費のみ）で製品原価を計算 |
| 代表的な手法 | 全部原価計算（標準的） | 直接原価計算（変動費のみで計算） |
| 主な用途 | 財務諸表の作成 | 内部の意思決定・利益計画 |

> **直接原価計算**（部分原価計算の代表例）：変動費だけで製品原価を計算し、固定費は製品に含めない方法。利益計画や価格設定の意思決定に活用される。

---

## ドリル 3-1: 原価計算の流れを並べよう

次の説明を正しい順番に並べましょう。

```
ア. 製品1個あたりの製造原価を計算する
イ. 原価を材料費・労務費・経費に分類し、消費高を計算する
ウ. 製造間接費を工場の各部門ごとに集計する
```

**答え：イ → ウ → ア**（費目別 → 部門別 → 製品別）

---

## ドリル 3-2: 個別 vs 総合を判断しよう

次の企業はどちらの原価計算を使うか答えましょう。

```
（  ）① 毎日同じペットボトル飲料を100万本製造している飲料メーカー

（  ）② 顧客の注文に合わせて特注の橋を建設している建設会社

（  ）③ 同じ仕様の乗用車を年間50万台大量生産している自動車メーカー

（  ）④ クライアントごとに異なるWebサイトを制作している会社
```

**答え：①③ → 総合原価計算　②④ → 個別原価計算**

## まとめ

| キーワード | 意味 |
|-----------|------|
| 費目別計算 | 原価を3分類し消費高を計算（第1段階） |
| 部門別計算 | 製造間接費を部門ごとに集計（第2段階） |
| 製品別計算 | 製品1個あたりの原価を計算（第3段階） |
| 個別原価計算 | 受注生産向け・注文ごとに原価集計 |
| 総合原価計算 | 大量生産向け・月単位で集計し個数で割る |
| 実際原価計算 | 実際にかかった原価で計算 |
| 標準原価計算 | 目標原価を設定し差異を分析 |
| 全部原価計算 | すべての原価で製品原価を計算 |
| 部分原価計算 | 一部の原価（変動費など）のみで計算 |
