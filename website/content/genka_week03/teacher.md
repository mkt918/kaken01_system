## 指導ポイント

### 導入の発問例

> 「工場で原価を計算するとき、最初から製品1個の値段を計算できると思う？順番はどうなると思う？」

生徒に考えさせてから3段階の流れを提示すると定着しやすい。

### 原価計算の3段階の教え方

「分類する → 部門に振り分ける → 製品にまとめる」という**漏斗（じょうご）のイメージ**で説明するとわかりやすい。

```
【全体の原価】
     │
     ↓ ① 費目別（材料費・労務費・経費に仕分け）
     │
     ↓ ② 部門別（各工場部門に振り分け）
     │
     ↓ ③ 製品別（製品1個の原価に落とし込む）
【製品1個の原価】
```

### 個別 vs 総合：判別のポイント

「同じものを大量に作るか、注文ごとに違うものを作るか」で判断させる。

**身近な例でのイメージ**
- 個別：「ケーキ屋でデコレーションケーキを特注」「建物を建てる」
- 総合：「コンビニのおにぎりを毎日10万個製造」「缶コーヒーを工場で大量生産」

### 実際・標準・全部・部分は概要理解でOK

W3時点では概念の把握が目標。詳細な計算方法は後の週で扱う。
「実際 vs 目標（標準）」「すべて vs 一部（直接）」という対比を板書して視覚化する。

## ドリル解答と解説

### ドリル 3-1: 流れの順番

```
正しい順番：イ → ウ → ア
（費目別計算 → 部門別計算 → 製品別計算）
```

**誤答パターン**：「ア（製品別）を最初に置く」
→「製品の原価は最後に計算するもの。まず材料費などを整理してから」と説明。

### ドリル 3-2: 個別 vs 総合

```
① ペットボトル飲料100万本 → 総合原価計算（同じ製品の大量生産）
② 特注の橋の建設 → 個別原価計算（注文ごとに異なる）
③ 乗用車50万台の大量生産 → 総合原価計算
④ クライアントごとのWeb制作 → 個別原価計算（案件ごとに異なる）
```

**ポイント**：IT業界やサービス業でも個別原価計算は使われることを補足する。

## 補足資料

### 計算期間の補足

月次計算が主流な理由を補足すると理解が深まる：
- 毎月の原価を把握することで「今月は材料費が増えた → 原因を調べよう」という**迅速な管理**が可能
- 1年単位では問題発覚が遅くなる

### 対比整理表（板書用）

| 対比軸 | 方法A | 方法B |
|--------|-------|-------|
| 生産形態 | 個別原価計算（受注生産） | 総合原価計算（大量生産） |
| 計算基準 | 実際原価計算（実績） | 標準原価計算（目標） |
| 範囲 | 全部原価計算（すべて） | 部分原価計算（一部のみ） |

- 材料の購入・消費の記録方法
