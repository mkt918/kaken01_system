## 1. 変数を作ってみよう

変数とは、名前をつけることができる「箱」のことです。Pythonでは、 `=` を使って箱にデータ（文字や数字）を入れます。

```python
name = "たろう" # " " で囲むと「文字」になる
age = 14        # そのまま書くと「数字」になる

print(f"名前は {name}、年齢は {age} 歳だね。")
```

### 👉 やってみよう
VS Code等のエディタを開き、上のコードをそのまま書き写して実行してみましょう。

## 2. 失敗から学ぼう（エラーの体験）

文字の「10」と、数字の「10」は、コンピュータにとって全くの別物です。
これを**型（種類）**と言います。

以下のコードを実行すると…エラー（赤文字）が出るはずです！

```python
num_text = "10"
num_real = 10

# エラーになります！ TypeError: can only concatenate str ...
print(num_text + num_real) 
```

### 💡 なぜエラーになるの？
文字(`str`)の箱にあるものと、数字(`int`)の箱にあるものは足し算できないからです。
計算したいときは、文字を「数字」に変身させる魔法 `int()` を使います。

```python
# これなら正解！
print(int(num_text) + num_real) 
```
