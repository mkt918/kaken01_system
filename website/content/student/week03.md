# Week 03: 条件分岐とインデント

今週は、プログラムが「自分で考えて道を選ぶ」ための命令、**if文（条件分岐）**を学びます。

## 1. True or False?

パソコンは「Yes（正しい）」か「No（間違い）」の2つでしか判断できません。

```python
my_score = 85

print("点数は80点より高い？:", my_score > 80) # True（正しい）
print("点数は100点と同じ？:", my_score == 100) # False（間違い）
```

## 2. if文のカタチと「インデント」

Pythonでは、if文のあとに「スペース4つ」を空けるルール（インデント）があります。
これを守らないとエラーになってしまいます。

```python
age = int(input("年齢は？: "))

if age >= 20:
    # ここにスペースが4つ入っているね！
    print("大人です！")
else:
    # else（それ以外）のあともスペース4つ！
    print("子供です！")
```
