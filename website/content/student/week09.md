# Week 09: データベースの「表」からデータを取り出そう

ここからのStep 2では、Pythonではなく**SQL（エスキューエル）**という、データベースを操作するための専用の言葉を学びます。

## 1. データベースって何？

エクセルやスプレッドシートのような「表（テーブル）」の集まりです。
プログラムを閉じてもデータが消えない魔法の箱です。

| id | name | level |
|:---|:---|:---|
| 1 | スライム | 3 |
| 2 | ドラゴン | 50 |

## 2. 全てのデータを引き出す呪文「SELECT *」

テーブルにある全部のデータを表示したい時は、このように書きます。

```sql
SELECT * FROM monsters;
```

- `SELECT` : 取り出して見せて！
- `*` : （アスタリスク）全部の列を！
- `FROM monsters` : monstersという名前の表から！
