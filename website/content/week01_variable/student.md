## 🕒 レッスン 1: パソコンにしゃべらせる魔法 `print()`

プログラミングの最初の仕事は、パソコンに「これを画面に表示して！」と命令することです。
Python（パイソン）という言葉では、**`print()`** （プリント）という命令を使います。

### 💻 ハンズオン（やってみよう）
1. **VS Code**（ブイエスコード）を開き、新しく `main.py` というファイルを作ります。
2. 以下のコードをコピーして、1行目に貼り付けてください。

```python
print("こんにちは！プログラミングの世界へ！")
```

3. 画面右上の **▷（再生ボタン）** をクリックして実行しましょう。
4. 下の黒い枠（ターミナル）に「こんにちは！プログラミングの世界へ！」と表示されましたか？おめでとうございます！これがあなたの最初のプログラムです。

### ⚠️ 絶対に守るルール
文字を表示させたいときは、必ず **`" "`（ダブルクォーテーション）** または **`' '`（シングルクォーテーション）** で囲みます。
さらに、**必ず「半角（英語モード）」を使って**入力します。

### ✍️ ドリル 1-1：いろいろ表示してみよう
以下のコードを追加して、実行してみましょう。

```python
print("私の名前は〇〇です。")
print("好きな食べ物は〇〇です。")
```
※〇〇の部分を自分の名前に変えてください。

### 💣 エラーを体験しよう！
プログラミングにエラー（失敗）はつきものです。わざとエラーを出してみましょう。
以下のコードを書いて実行してください。

```python
# わざと "（ダブルクォーテーション）を消してみる
print(こんにちは！)
```

赤い文字で `SyntaxError`（シンタックスエラー：文法が間違っているよ）と出ましたね？
**エラーは敵ではありません。パソコンからの「ここが間違っているよ」という優しいヒントです。** 恐れずにどんどんエラーを出しましょう！

## 🕒 レッスン 2: データを覚えておく「箱」（変数）

長い手紙を毎回タイピングするのは大変ですよね。そこで**「変数（へんすう）」**という便利な機能を使います。
変数とは、好きな名前を付けられる **「データを入れる箱」** のことです。

### 💻 ハンズオン（やってみよう）
以下のコードを書いて実行してみましょう。前のコードは消してもOKです。

```python
# name という箱を作って、"たろう" を入れる
name = "たろう"

# age という箱を作って、14 を入れる
age = 14

# 箱の中身を表示する（" " で囲まないのがポイント！）
print(name)
print(age)
```

> **💡 ポイント**
> `name = "たろう"` の `=` は、数学の「等しい」という意味ではありません。
> プログラミングでは「**右にあるものを、左の箱に入れる（代入する）**」という意味になります！

### ✍️ ドリル 2-1：自分のステータス箱を作ろう
ゲームのキャラクターのように、自分のステータスを箱に入れてみましょう。

```python
my_hp = 100
my_mp = 50
my_job = "魔法使い"

# printを使って、それぞれの箱の中身を表示してみましょう！
print(my_hp)
print(my_mp)
print(my_job)
```

## 🕒 レッスン 3: フュージョン！文字と箱を合体させる魔法

「たろう」「14」だけ表示されても、ただの数字と文字の羅列で意味がわかりませんね。
**フォーマット文字列（f文字列）** という最強の技を使うと、普通の文章の途中に「箱（変数）の中身」を自然に埋め込むことができます。

### 💻 ハンズオン（やってみよう）
`print(f"...")` のように、最初に **小文字の f** を書くのが合図です。箱の中身を展開（オープン）したい場所は **`{ }`（波カッコ）** で囲みます。

```python
player_name = "ゆうしゃ"
lv = 5

print(f"おお {player_name} よ！ レベルが {lv} にあがった！")
```

### ✍️ ドリル 3-1：自己紹介ジェネレーター
先ほど作ったステータス箱を使って、プロフィールの文章を作ってみましょう。

```python
my_hp = 100
my_job = "魔法使い"

# f文字列を使って文章を作る
print(f"私の職業は {my_job} です。体力は {my_hp} あります。")
```
