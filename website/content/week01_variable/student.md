## 🕒 レッスン 1: パソコンにしゃべらせる魔法 `print()`

プログラミングの最初の仕事は、パソコンに「これを画面に表示して！」と命令することです。
Python（パイソン）という言葉では、**`print()`**（プリント）という命令を使います。

### 💻 ハンズオン（print を試す）

以下の手順で試してみましょう。

以下の手順で試してみましょう。

- **VS Code**（ブイエスコード）を開き、新しく `main.py` というファイルを作ります。
- 以下のコードを1行目に入力してください。

```python
print("こんにちは！プログラミングの世界へ！")
```

- 画面右上の **▷（再生ボタン）** をクリックして実行しましょう。
- 下の黒い枠（ターミナル）に「こんにちは！プログラミングの世界へ！」と表示されましたか？おめでとうございます！これがあなたの最初のプログラムです。

### ⚠️ 絶対に守るルール

文字を表示させたいときは、必ず **`" "`（ダブルクォーテーション）** または **`' '`（シングルクォーテーション）** で文字を囲みます。
また、**必ず「半角（英語モード）」で入力します**。全角のカッコや記号を使うとエラーになります。

### ✍️ ドリル 1-1：いろいろ表示してみよう

以下のコードを追加して実行してみましょう。〇〇の部分は自分の情報に書き換えてください。

```python
print("私の名前は〇〇です。")
print("好きな食べ物は〇〇です。")
print("好きなゲームは〇〇です。")
```

### 💣 エラーを体験しよう

プログラミングにエラー（失敗）はつきものです。わざとエラーを出してみましょう。

```python

```

赤い文字で `SyntaxError`（シンタックスエラー：文法が間違っているよ）と出ましたね？
**エラーは敵ではありません。パソコンからの「ここが間違っているよ」という優しいヒントです。** 恐れずにどんどんエラーを出しましょう！

## 🕒 レッスン 2: データを覚えておく「箱」（変数）

長い文章を毎回タイピングするのは大変ですよね。そこで**「変数（へんすう）」**という便利な機能を使います。
変数とは、好きな名前を付けられる **「データを入れる箱」** のことです。

### 💻 ハンズオン（変数に入れる）

以下のコードを書いて実行してみましょう。前のコードは消してもOKです。

```python
# name という箱を作って、"たろう" を入れる
name = "たろう"

# age という箱を作って、14 を入れる
age = 14

# 箱の中身を表示する（変数は " " で囲まないのがポイント！）
print(name)
print(age)
```

> **💡 ポイント**
> `name = "たろう"` の `=` は、数学の「等しい」という意味ではありません。
> プログラミングでは「**右にあるものを、左の箱に入れる（代入する）**」という意味になります！

### ✍️ ドリル 2-1：自分のステータス箱を作ろう

ゲームのキャラクターのように、自分のステータスを箱に入れて表示してみましょう。

```python
my_hp = 100
my_mp = 50
my_job = "魔法使い"

# printを使って、それぞれの箱の中身を1行ずつ表示してみましょう！
```

> **ヒント：** `print(変数名)` で箱の中身を表示できます。3行書いてみましょう。

## 🕒 レッスン 3: フュージョン！文字と箱を合体させる魔法（f文字列）

「たろう」「14」だけ表示されても、ただの単語の羅列で意味がわかりませんね。
**フォーマット文字列（f文字列）** という最強の技を使うと、普通の文章の途中に「箱（変数）の中身」を自然に埋め込むことができます。

### 💻 ハンズオン（f文字列を使う）

`print(f"...")` のように、クォーテーションの**すぐ前に小文字の f** を書くのが合図です。箱の中身を展開したい場所は **`{ }`（波カッコ）** で囲みます。

```python
player_name = "ゆうしゃ"
lv = 5

print(f"おお {player_name} よ！ レベルが {lv} にあがった！")
```

> **⚠️ 注意**
> `{ }` は「波カッコ（なみかっこ）」です。`[ ]` や `( )` と間違えないようにしましょう！

### ✍️ ドリル 3-1：自己紹介ジェネレーター

先ほど作ったステータス箱を使って、自己紹介の文章を f文字列で作ってみましょう。

```python
my_hp = 100
my_job = "魔法使い"

# f文字列を使って文章を作る（2行以上の文章を作ってみよう！）
print(f"私の職業は {my_job} です。体力は {my_hp} あります。")
```
