プログラミングの世界へようこそ！
今週は、パソコンに文字を表示させる「魔法の言葉」と、データを入れることができる「名前付きの箱」について学びます。

---

## ステップ1: パソコンにしゃべらせる魔法「print」

### 🎯 目標：文字を表示させよう
Python（パイソン）という言葉を使って、画面に文字を表示させましょう。使うのは **`print()`** （プリント）という命令です。

### 👉 やってみよう (1)
1. **VS Code**（ブイエスコード）を開きます。
2. 以下のコードをコピーして、エディタに貼り付けてください。

```python
print("こんにちは！プログラミングの世界へ！")
```

3. 右上の **▷（再生ボタン）** をクリックして実行しましょう。
4. 下の黒い画面（ターミナル）に「こんにちは！プログラミングの世界へ！」と表示されましたか？

> **💡 ポイント**
> 文字を表示させたいときは、必ず **`" "`（ダブルクォーテーション）** または **`' '`（シングルクォーテーション）** で囲むというルールがあります。これが無いとエラーになります。

---

## ステップ2: データを入れる「箱」を作ろう

毎回長い手紙を書くのは大変ですね。そこで**「変数（へんすう）」**という便利な機能を使います。
変数とは、好きな名前を付けられる **「データを入れる箱」** のことです。

### 👉 やってみよう (2)
以下のコードをコピーして実行してみましょう。

```python
# name という箱を作って、"たろう" を入れる
name = "たろう"

# age という箱を作って、14 を入れる
age = 14

print(name)
print(age)
```

> **💡 ポイント**
> `name = "たろう"` の `=` は、数学の「等しい」という意味ではありません。
> プログラミングでは「**右にあるものを、左の箱に入れる**」という意味になります！

---

## ステップ3: フュージョン！文字と箱を合体させる

「たろう」と「14」だけ表示されても、少し寂しいですよね。
**フォーマット文字列（f文字列）** という技を使うと、普通の文章の途中に「箱の中身」を埋め込むことができます。

### 👉 やってみよう (3)
`print(f"...")` のように、最初に **小文字の f** を書くのがコツです。箱の中身を展開したい場所には **`{ }`（波カッコ）** を使います。

```python
# 自分の名前に変えてみよう！
player_name = "ゆうしゃ"
lv = 5

print(f"おお {player_name} よ！ レベルが {lv} にあがった！")
```

さあ、`player_name` と `lv` を自分の好きな名前に変えて、もう一度実行してみましょう！
