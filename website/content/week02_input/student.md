## 🕒 レッスン 1: 画面から答えを聞き出す魔法 `input()`

先週は `name = "たろう"` のように、プログラムを書く人がデータを直接箱に入れていました。
今週からは、**パソコンの前に座っている人**に質問をして、その答えを箱に入れる方法を学びます。使うのは **`input()`**（インプット）という命令です。

### 💻 ハンズオン（やってみよう）

以下のコードをVS Codeに入力して、実行してみましょう。

```python
# アプリを使う人に質問をして、答えを user_name という箱に入れる
user_name = input("あなたの名前を教えてください: ")

# 聞き出した名前を使って挨拶する（先週の復習・f文字列）
print(f"ようこそ、{user_name} さん！ プログラミングの世界へ！")
```

実行すると、ターミナルで `あなたの名前を教えてください:` と止まって**文字を入力できる**ようになります。
自分の名前を入力して、**Enterキー** を押してみましょう！パソコンが返事をしてくれましたか？

> **⚠️ 注意**
> ターミナルが「止まった」と感じたら、まずターミナル内をクリックしてからキーボードを叩いてみましょう。入力待ちの状態です。

### ✍️ ドリル 1-1：いろいろ聞き出そう

## 🕒 レッスン 2: 計算をしてみよう！…あれ？（エラーの体験）

次に、年齢を入力してもらって、パソコンに計算をしてもらいましょう。

### 💻 ハンズオン（エラーを体験する）

以下のコードを実行して、年齢を聞かれたら「14」と入力してEnterを押してください。

```python
# 年齢を聞き出して age という箱に入れる
age = input("あなたは何歳ですか？: ")

# 「5年後」の年齢を計算して表示したい！
print("5年後のあなたの年齢は...")
print(age + 5)
```

**エラー発生！！**
画面に赤い文字で `TypeError: can only concatenate str (not "int") to str` と表示されましたか？

### 💡 なぜエラーになったの？

実は、**`input()` で聞き出した答えは、どんなに数字に見えてもすべて「文字（str）」として扱われる** という強力なルールがあるからです。
私たちの世界では「14」と聞けば数字ですが、パソコンにとって `input()` で入ってきた「14」は「イチ、ヨンという文字のシール」でしかありません。文字のシールには、足し算（＋5）はできないのです。

---

## 🕒 レッスン 3: 「文字のシール」を「計算機の数字」に変身させる魔法 `int()`

エラーを乗り越えましょう！
文字の "14" を、計算用の 14 に変身させる魔法が **`int()`**（インテジャー：整数の英語 Integer の略）です。

### 💻 ハンズオン（int() で修正する）

エラーになったコードを、以下のように修正してもう一度実行してみましょう！

```python
age = input("あなたは何歳ですか？: ")

# 数字に変身させた年齢を age_number という新しい箱に入れる
age_number = int(age)

# 今度は計算できるはず！
print("5年後のあなたの年齢は...")
print(age_number + 5)
```

今度は正しく計算されましたね！

### ➕➖✖️➗ Pythonの四則演算ルール

Pythonでは計算の記号が少し違います。

| 演算   | 記号 | 例      | 結果 |
| ------ | ---- | ------- | ---- |
| 足し算 | `+`  | `3 + 4` | 7    |
| 引き算 | `-`  | `10 - 3`| 7    |
| 掛け算 | `*`  | `3 * 4` | 12   |
| 割り算 | `/`  | `10 / 4`| 2.5  |

## 🚀 今日の総仕上げ：ミニプロジェクト「BMI計算アプリ」を作ろう

人の健康状態を表す「BMI（肥満度）」を計算する本格的なアプリを作ります！
BMIの計算式は **「体重(kg) ÷ (身長(m) × 身長(m))」** です。

### 📝 ミッション

ユーザーから「身長」と「体重」を `input()` で聞き出し、BMIを計算して表示させてください。

注意点が2つあります。

1. 身長は「センチ」ではなく「**メートル**」で計算します（例：160cmなら1.6m）。
2. 小数を含む数字に変換するときは `int()` ではなく **`float()`**（フロート）を使います。

#### 完成イメージ

```text
BMI計算アプリへようこそ！
身長(m)を入力してください（例: 1.6）: 1.7
体重(kg)を入力してください（例: 60）: 65
あなたのBMIは 22.49 です！
```

### 🏆 ヒントコード

```python

h_text = input("身長(m)を入力してください（例: 1.6）: ")
```
