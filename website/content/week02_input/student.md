## 🕒 レッスン 1: 画面から答えを聞き出す魔法 `input()`

先週は `name = "たろう"` のように、プログラムを書く人がデータを直接箱に入れていました。
今週からは、**パソコンの前に座っている人**に質問をして、その答えを箱に入れる（代入する）方法を学びます。使うのは **`input()`**（インプット）という命令です。

### 💻 ハンズオン（やってみよう）
以下のコードをVS Codeに貼り付けて、実行してみましょう。

```python
# アプリを使う人に質問をして、答えてもらった内容を user_name という箱に入れる
user_name = input("あなたの名前を教えてください: ")

# 聞き出した名前を使って挨拶する（先週の復習・f文字列）
print(f"ようこそ、{user_name} さん！ プログラミングの世界へ！")
```

実行すると、ターミナルで `あなたの名前を教えてください: ` と止まって**文字を入力できる**ようになります。
自分の名前を入力して、**Enterキー** を押してみましょう！パソコンが返事をしてくれましたか？


## 🕒 レッスン 2: 計算をしてみよう！…あれ？（エラーの体験）

次に、年齢などを入力してもらって、パソコンに足し算（計算）をしてもらいましょう。

### 💻 ハンズオン（やってみよう）
以下のコードを実行して、年齢を聞かれたら「14」と入力してEnterを押してください。

```python
# 年齢を聞き出して age という箱に入れる
age = input("あなたは何歳ですか？: ")

# 「5年後」の年齢を計算して表示したい！
print("5年後のあなたの年齢は...")
print(age + 5)
```

**【エラー発生！！】**
画面の下（ターミナル）に赤い文字で `TypeError: can only concatenate str (not "int") to str` と表示されましたか？

### 💡 なぜエラーになったの？
実は、**`input()` の魔法で聞き出した答えは、どんなに数字に見えてもすべて「文字（str）」として扱われる** という強力なルールがあるからです。
私たちの世界では「14」と聞けば数字ですが、パソコンにとって `input()` で入ってきた「14」は、「イチ、ヨンという文字の絵柄（シール）」でしかないのです。文字のシールには、足し算（＋5）はできません。

---

## 🕒 レッスン 3: 「文字のシール」を「計算機」の数字に変身させる魔法 `int()`

エラーを乗り越えましょう！
文字の "14" を、計算用の 14 に変身させる魔法が **`int()`**（整数＝Integerの略）です。

### 💻 ハンズオン（やってみよう）
エラーになったコードを、以下のように直してもう一度実行してみましょう！

```python
age = input("あなたは何歳ですか？: ")

# age_number という【新しい箱】に、数字に変身させた(int)年齢を入れる！
age_number = int(age)

# 今度は計算できるはず！
print("5年後のあなたの年齢は...")
print(age_number + 5)
```

今度は正しく「19」と計算されましたね！

### ➕➖✖️➗ Pythonの四則演算ルール
Pythonでは計算の記号が少し違います。
- 足し算: `+`
- 引き算: `-`
- 掛け算: `*` (アスタリスク) 例：`3 * 4` は 12
- 割り算: `/` (スラッシュ) 例：`10 / 2` は 5.0

### ✍️ ドリル 3-1：生まれ年を計算しよう
現在の年（2024など）から、入力してもらった年齢を引き算して、何年生まれかを簡易的に計算するアプリを作ってみましょう。

```python

age_num = int(age)
# "あなたはだいたい 〇〇 年生まれですね！" と表示するコードを書いてみよう（2024 - age_num）
```
