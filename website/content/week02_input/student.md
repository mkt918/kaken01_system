# Week 02: パソコンと会話する「入力」と「計算」

今週は、パソコンを使う人（ユーザー）から情報を聞き出す魔法と、パソコンの得意技である「計算」について学びます。

---

## ステップ1: 画面から答えを聞き出す魔法「input」

先週は自分たちで `name = "たろう"` のようにデータを直接箱に入れました。
今週は、**パソコンの前に座っている人**に質問をして、その答えを箱に入れる方法を学びます。使うのは **`input()`**（インプット）という命令です。

### 👉 やってみよう (1)
以下のコードをVS Codeに貼り付けて、実行してみましょう。

```python
# 質問をして、答えてもらった内容を user_name という箱に入れる
user_name = input("あなたの名前を教えてください: ")

# 聞き出した名前を使って挨拶する
print(f"ようこそ、{user_name} さん！")
```

実行すると、ターミナルで `あなたの名前を教えてください: ` と止まって**文字を入力できる**ようになります。
自分の名前を打ち込んで、**Enterキー** を押してみましょう！

---

## ステップ2: 足し算をしてみよう！…あれ？

次に、年齢などを入力してもらって、パソコンに足し算（計算）をしてもらいましょう。

### 👉 やってみよう (2)
以下のコードを実行して、年齢を聞かれたら「14」と入力してEnterを押してください。

```python
# 年齢を聞き出して age という箱に入れる
age = input("あなたは何歳ですか？: ")

# 「5年後」の年齢を計算して表示する
print("5年後のあなたの年齢は...")
print(age + 5)
```

**【エラー発生！！】**
画面の下（ターミナル）に赤い文字で `TypeError: can only concatenate str (not "int") to str` と表示されましたか？

---

## ステップ3: 「文字のシール」と「数字の計算機」

なぜエラーになったのでしょう？ 実は、**`input()` の魔法で聞き出した答えは、すべて「文字（str）」として扱われる**という強力なルールがあるからです。

- `"14"`（文字のシール）：足し算できない！
- `14`（計算機用の数字）：足し算できる！

文字の "14" を、計算用の 14 に変身させる魔法が **`int()`**（整数＝Integerの略）です。

### 👉 やってみよう (3)
エラーになったコードを、以下のように直してもう一度実行してみましょう！

```python
age = input("あなたは何歳ですか？: ")

# age_number という新しい箱に、数に変身させた(int)年齢を入れる！
age_number = int(age)

print("5年後のあなたの年齢は...")
print(age_number + 5)
```

今度は正しく「19」と計算されましたね！
