# Week 03: プログラムに知能を与える「もしも」の分かれ道

今週は、プログラムが「自分で考えて、状況によって別の行動をする」ための命令、**if文（条件分岐）**を学びます。ゲームで「HPが0になったらゲームオーバーになる」「鍵を持っていたら扉が開く」のも、すべてこの仕組みを使っています。

---

## 🕒 レッスン 1: パソコンの「正しい・間違い」の判断

パソコンは「Yes（正しい）」か「No（間違い）」の2つでしか物事を判断できません。まずはそれを確認してみましょう。

### 💻 ハンズオン（やってみよう）

以下のコードを実行してみてください。

```python
score = 85

# 80より大きいかな？（大なり記号）
print(score > 80)

# まさか100点ピッタリ？（等しいかどうかは == を使う！）
print(score == 100)
```

結果に **True**（トゥルー：正しい）や **False**（フォルス：間違い）が表示されましたね？
パソコンの脳内は常にこの「True」と「False」で情報を処理しています。

### ✍️ ドリル 1-1：結果を予想しよう

以下のプログラムを実行する**前に**、結果が「True」になるか「False」になるかを予想してから実行してみてください。

```python
my_money = 500

print(my_money >= 500)  # >= は「以上」の意味
print(my_money < 300)   # <  は「未満（より小さい）」の意味
print(my_money != 100)  # != は「等しくない（ノットイコール）」の意味
```

予想は当たっていましたか？

## 🕒 レッスン 2: もし〜なら（if のカタチ）

この「True/False」の判断を使って、「もし合格点（80点以上）なら『おめでとう』と表示する」プログラムを作りましょう。

### 💻 ハンズオン（ifで分岐させる）

以下のコードを実行し、テストの点数を入力してください。

```python
# 点数を入力して数字に変換する（Week2の復習）
score = int(input("テストは何点だった？: "))

# もしも（if）、score が 80 以上のとき
if score >= 80:
    print("合格！おめでとう！")
    print("よくがんばりました。")

print("--- 判定おわり ---")
```

「85」を入れた場合と「50」を入れた場合で結果が変わることを確認しましょう！プログラムが自分で判断してメッセージを変えましたね！

## 🕒 レッスン 3: 絶対に守るルール「インデント」とエラーの戦い

上記のコードの `print("合格！...")` の前には、**半角スペース4つ**のスキマが空いています。これを **インデント（字下げ）** と呼びます。

Pythonでは「if文がTrueのときに実行するグループ（お部屋）」を、このインデントで表します。スペースが無いと絶対にエラーになります。

### 💻 ハンズオン（インデントエラーを体験）

`print` の前のスペースをバックスペースキーで全部消して、左端にくっつけて実行してみてください。

```python
score = 85

if score >= 80:
# スペースが無いとどうなる？
print("合格！")
```

`IndentationError`（インデントエラー）が出ましたね。このエラーを見たら「スペース忘れてる！」と気づけるようになりましょう。

> **💡 スペースを楽に入れるコツ**
> 半角スペース4つを手打ちする代わりに、キーボード左端の **Tabキー** を1回押すだけでインデントが入ります！

### ✍️ ドリル 3-1：エラー修正パズル

以下のコードにはインデントの間違いがあります。正しい形に直して実行してください。

```python
hp = 0

if hp == 0:
print("勇者は倒れた！")
print("ゲームオーバー")
```

## 🚀 今日の総仕上げ：ミニゲーム「数当てゲーム（1回勝負）」を作ろう

パソコンが思い浮かべた数字を、プレイヤーが当てるゲームを作りましょう。

### 📝 ミッション

- 正解の数字を `7` とします（変数 `answer` に入れておきましょう）。
- プレイヤーに `input()` で 1〜10 の数字を予想して入力させます。
- プレイヤーの入力（`int()` で数字に変換することを忘れずに！）と正解が **等しければ（`==`）**、「大正解！すごい！」と表示してください。

#### 完成イメージ（正解のとき）

```text
=== 数当てゲーム ===
1から10までの数字を予想してね: 7
大正解！すごい！
```

### 🏆 チャレンジ

「7」以外を入力したときに「はずれ！正解は 7 でした！」と表示するには、`if` の「それ以外」を表す **`else:`**（エルス）を使います。
`if` と同じインデントの高さで `else:` と書いて、中の命令はインデントしてください。来週のメインテーマですが、ぜひ試してみましょう！

## 🚀 早く終わった人へ：ミニプロジェクト「ラッキーナンバーおみくじ」

数当てゲームの仕組みを使って、今日の運勢を占う「おみくじアプリ」を作りましょう！

### 📝 ミッション（おみくじ）

- プレイヤーに 1〜3 の好きな数字を入力させます。
- 入力された数字によって結果を変えましょう。

> **ヒント:** 3種類の分岐は、`if` を3つ独立して書いても作れます。早く終わった人は、来週学ぶ `elif` を使った書き方にも挑戦してみましょう！

#### 完成イメージ（おみくじ）

```text

中吉！いつも通りの平和な日です。
```
