# Week 03: 「もしも」の分かれ道 (1)

今週は、プログラムが「自分で考えて道を選ぶ」ための命令、**if文（条件分岐）**を学びます。ゲームで「HPが0になったらゲームオーバーになる」のも、この仕組みを使っています。

---

## ステップ1: パソコンの「正しい・間違い」判定

パソコンは「Yes（正しい）」か「No（間違い）」の2つでしか物事を判断できません。まずはそれを確認してみましょう。

### 👉 やってみよう (1)
以下のコードを実行してみてください。

```python
my_score = 85

# 80より大きいかな？（> の向きに注意！）
print("点数は80点より高い？:", my_score > 80)

# まさか100点ピッタリ？（等しいかは == とイコールを2つ書く！）
print("点数は100点と同じ？:", my_score == 100)
```

結果に **True**（トゥルー：正しい）や **False**（フォルス：間違い）が表示されましたね？ パソコンの脳内はこの2つの言葉で判断しています。

---

## ステップ2: もし〜なら（if のカタチ）

この「True/False」を使って、「もし合格点(80点以上)なら『おめでとう』と表示する」プログラムを作りましょう。

### 👉 やってみよう (2)
以下のコードをコピーして実行してください。

```python
score = int(input("テストは何点だった？: "))

# もしも（if）、score が 80 以上のとき
if score >= 80:
    print("合格！おめでとう！")
    print("よくがんばりました。")

print("--- 判定おわり ---")
```

実行して、「85」と入れた場合と、「50」と入れた場合で、結果が変わることを確認しましょう！

---

## ステップ3: 絶対に守るルール「インデント（お部屋）」

上記のコードの `print("合格！...")` の前には、**半角スペース4つ**のスキマが空いています。これを**インデント**と呼びます。

Pythonでは「if文がTrueのときに実行するグループ（お部屋）」を、このインデント（スペース4つ）で表します。スペースを忘れるとエラーになります。

### 👉 やってみよう (3)
わざとエラーを出してみよう！ `print` の前のスペースを消して実行してみてください。

```python
score = 85

if score >= 80:
# ここにスペースが無いとどうなる？
print("合格！") 
```

`IndentationError`（インデントエラー）が出ましたね。このエラーを見たら「あっ、スペース忘れてる！」と気づけるようになりましょう。
