# Week 03: プログラムに知能を与える「もしも」の分かれ道

今週は、プログラムが「自分で考えて、状況によって別の行動をする」ための命令、**if文（条件分岐）**を学びます。ゲームで「HPが0になったらゲームオーバーになる」「鍵を持っていたら扉が開く」のも、すべてこの仕組みを使っています。

---

## 🕒 レッスン 1: パソコンの「正しい・間違い」の判断

パソコンは「Yes（正しい）」か「No（間違い）」の2つでしか物事を判断できません。まずはそれを確認してみましょう。

### 💻 ハンズオン（やってみよう）
以下のコードをコピーして実行してみてください。（英語の半角文字で書かれていますか？）

```python
score = 85

# 80より大きいかな？（大なり）
print(score > 80)

# まさか100点ピッタリ？（等しいかは == を使う！）
print(score == 100)
```

結果に **True**（トゥルー：正しい）や **False**（フォルス：間違い）が表示されましたね？ 
パソコンの脳内は常にこの「True」と「False」で情報を処理しています。

### ✍️ ドリル 1-1：結果を予想しよう！
以下のプログラミングを実行する前に、結果が「True」になるか「False」になるかを予想してから実行してみてください。
```python
my_money = 500

print(my_money >= 500)  # >= は「以上」の意味
print(my_money < 300)   # < は「未満」の意味
print(my_money != 100)  # != は「等しくない（ノットイコール）」の意味
```

## 🕒 レッスン 2: もし〜なら（if のカタチ）

この「True/False」の判断を使って、「もし合格点(80点以上)なら『おめでとう』と表示する」プログラムを作りましょう。

### 💻 ハンズオン（やってみよう）
以下のコードをコピーして実行し、テストの点数を入力してください。

```python
# 点数を入力して数字に変身させる
score_text = input("テストは何点だった？: ")
score = int(score_text)

# もしも（if）、score が 80 以上のとき
if score >= 80:
    print("合格！おめでとう！")
    print("よくがんばりました。")

print("--- 判定おわり ---")
```

実行して、「85」と入れた場合と、「50」と入れた場合で結果が変わることを確認しましょう！プログラムが自分で判断してメッセージを変えましたね！

## 🕒 レッスン 3: 絶対に守るルール「インデント」とエラーの戦い

上記のコードの `print("合格！...")` の前には、**半角スペース4つ**のスキマが空いています。これを **インデント（字下げ）** と呼びます。

Pythonでは「if文がTrueのときに実行するグループ（お部屋）」を、このインデントで表します。スペースを忘れると絶対にエラーになります。

### 💻 ハンズオン（わざとエラーを出す）
わざとエラーを出してみよう！ `print` の前のスペースを【バックスペースキー】で全部消して、左端にくっつけて実行してみてください。

```python
score = 85

if score >= 80:
# ここにスペースが無いとどうなる？
print("合格！") 
```

`IndentationError`（インデントエラー）が出ましたね。このエラーを見たら「あっ、スペース忘れてる！」と気づけるようになりましょう。

### ✍️ ドリル 3-1：エラー修正パズル
以下のコードにはインデントの間違いがあります。正しい形に直して実行してください。
（ヒント：ifの後に実行したい命令だけを、半角スペース4つ分字下げします）

```python
hp = 0

if hp == 0:
print("勇者は倒れた！")
print("ゲームオーバー")
```

## 🚀 今日の総仕上げ：ミニゲーム「数当てゲーム（1回勝負）」を作ろう

パソコンが思い浮かべた数字（今回は固定で「7」にします）を、プレイヤーが入力して当てるゲームを作りましょう。

### 📝 ミッション
- 正解の数字を `7` とします。（変数 `answer` などに入れておきましょう）
- プレイヤーに `input()` で 1〜10 の数字を予想して入力させます。
- プレイヤーの入力（数字に変身させるのを忘れずに！）と正解が **等しければ（`==`）**、「大正解！すごい！」と表示してください。
- （ヒント）等しかった場合のみ実行されるように `if` と インデント を使います。

**【完成イメージ（正解のとき）】**
```text
=== 数当てゲーム ===
1から10までの数字を予想してね: 7
大正解！すごい！
```

### 🏆 チャレンジ
もし「7」以外（例えば 3 など）を入力したときは、「はずれ！正解は 7 でした！」と表示するようにしたいです。
`if` の「それ以外」のときは、同じインデントの高さで **`else:`** （エルス）という言葉を使います。先生に聞きながら、または自分で調べて `else` を追加してみてください！

## 🚀 今日の総仕上げ2：ミニプロジェクト「ラッキーナンバーおみくじ」

数当てゲームの仕組みを使って、今日の運勢を占う「おみくじアプリ」を作りましょう！

### 📝 ミッション
- プレイヤーに 1〜3 の好きな数字を入力させます。
- `if` や `elif`、`else` （チャレンジした人）を使って、入力された数字によって結果を変えましょう。
  - `1` のとき：「大吉！今日は最高の1日になるよ！」
  - `2` のとき：「中吉！いつも通りの平和な日です。」
  - `3` のとき：「凶… 忘れ物に注意してね。」

**【完成イメージ】**
```text
=== 🔮 魔法のおみくじ 🔮 ===
1〜3 の好きな数字を選んでね: 2
中吉！いつも通りの平和な日です。
```
※余裕がある人は、占い結果を5種類くらいに増やしてみましょう！
