# Week 05: パソコンの最強特技「終わりなき繰り返し」

今週から、ついにパソコンが人間よりも圧倒的に優れている**「文句を言わずに、何万回でも同じ作業を繰り返す」**魔法を学びます。ゲームの「メイン画面がずっと表示され続ける」のも繰り返しの魔法のおかげです！

---

## 🕒 レッスン 1: 繰り返しのおまじない `while`

「もし〜なら」は `if` でしたが、「〜の間はずっと繰り返す」は **`while`（ホワイル）** と言います。

### 💻 ハンズオン（やってみよう）
以下のコードをコピーして実行してみましょう。

```python
count = 1

# count が 3 以下(<=)である「間はずっと繰り返す」
while count <= 3:
    print(f"ただいま {count} 回目です！")
    # 箱の中身を 1 増やす（超重要！）
    count = count + 1

print("繰り返しが終了しました！")
```

結果に「ただいま1回目、2回目、3回目」と出ましたか？
`count = count + 1` という書き方は変に見えるかもしれませんが、プログラミングでは「今の箱の中身（右）に1を足して、同じ箱に戻す（左）」という**カウントアップの超基本テクニック**です。

### ✍️ ドリル 1-1：カウントダウン！
ロケットの発射カウントダウンを作りましょう。5からスタートして、1までカウントし、最後に「発射！」と表示させます。

```python
time = 5

while time ________:    # time が 0 より大きい間、など条件を考えよう
    print(f"発射まであと {time} ...")
    time = time - 1     # 箱の中身を 1 減らす

print("🚀 発射！")
```

## 🕒 レッスン 2: 恐怖の「無限ループ」を体験せよ

繰り返しの魔法には、1つだけ**絶対にやってはいけない禁忌（タブー）**があります。それが「無限ループ」（終わらない繰り返し）です。

### 💻 恐怖のハンズオン（わざとバグを起こす）
以下のコードは、ある1行を書き忘れています。実行するとどうなるでしょうか？

```python
hp = 10

while hp > 0:
    print(f"勇者のHPは {hp} だ！まだまだ戦える！")
    # わざとダメージ（引き算）を書かない
```

ターミナルが猛烈な勢いで文字を吐き出し始めましたね？！一生終わりません！
**【緊急停止・助かる方法】**
ターミナルの黒い画面を一度クリックし、キーボードの **`Ctrl` キーを押しながら `C` キー** を押してください。（Macの場合は `Control + C` です）。
赤い文字で `KeyboardInterrupt` と出て、プログラムが強制終了します。

> **💡 無限ループの原因**
> `hp` を減らすコードを書き忘れたため、`hp > 0` の条件が**永遠にTrueのまま**になってしまったのが原因です！

## 🕒 レッスン 3: 魔法のブレーキ `break`

回数や条件で終わるだけでなく、「ある特別なことが起きたら、強制的に繰り返しをやめる（ループを抜ける）」という便利なブレーキが **`break`（ブレイク）** です。

### 💻 ハンズオン（やってみよう）
ユーザーが「やめる」と入力するまで永遠におうむ返しをする、ちょっと面倒なチャットボットです。

```python
while True:  # True と書くと、強制的に永遠に繰り返す（無限ループ）になります
    word = input("何か話しかけてね（'やめる' で終了）: ")
    
    if word == "やめる":
        print("じゃあね！バイバイ！")
        break  # ここで繰り返しをブチッと切って外に出る！
    
    print(f"あなたは「{word}」と言いましたね！")
```

`while True:` でわざと無限ループを作り、「やめる」という言葉（`if`）を待って `break`（脱出）する、これはプロのエンジニアも毎日使う最強の組み合わせです。

### ✍️ ドリル 3-1：合い言葉で開く扉
無限ループ（`while True:`）を使って、「開けゴマ」と入力されるまで永遠に「合い言葉が違います！」と言い続けるプログラムを作ってください。正解の時だけ `break` します。

Week 3で作った「数当てゲーム」は1回間違えたら即ゲームオーバーのクソゲーでした。
今回は `while` を使って、**当たるまで何度でも挑戦できる**ゲームに進化させましょう。

### 📝 ミッション
- 正解の数字（例: 7）を変数に入れます。
- `while True:` を使って無限ループを開始します。
- プレイヤーに `input` で数字を予想させます。
- 正解（`==`）なら「大正解！」と表示して `break` でループを抜けます。
- はずれなら「違います！もう一度！」と表示して、ループを繰り返させます。

いよいよ本格的なゲームの戦闘画面です。勇者とスライムがお互いにダメージを与えあうループを作りましょう！

### 📝 ミッション

- プレイヤーに「攻撃しますか？ (1=攻撃 / 0=逃げる)」と `input` で聞きます。
