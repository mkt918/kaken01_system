# Week 05: パソコンの最強特技「終わりなき繰り返し」

今週から、ついにパソコンが人間よりも圧倒的に優れている**「文句を言わずに、何万回でも同じ作業を繰り返す」**魔法を学びます。ゲームの「メイン画面がずっと表示され続ける」「HPが0になるまでバトルが続く」のも、すべて繰り返しの魔法のおかげです！

---

## 🕒 レッスン 1: 繰り返しのおまじない `while`

「もし〜なら」は `if` でしたが、「〜の間はずっと繰り返す」は **`while`（ホワイル）** と言います。

### 💻 ハンズオン（やってみよう）

以下のコードを入力して実行してみましょう。

```python
count = 1

# count が 3 以下(<=)である「間はずっと繰り返す」
while count <= 3:
    print(f"ただいま {count} 回目です！")
    # 箱の中身を 1 増やす（超重要！）
    count = count + 1

print("繰り返しが終了しました！")
```

「ただいま1回目、2回目、3回目」と出ましたか？

> **💡 `count = count + 1` の意味**
> 数学では「countはcount+1と等しい」は成立しませんが、プログラミングでは違います。
> 「**右で計算して（count + 1）、同じ箱に上書き保存する（count =）**」という意味です。
> これは「カウントアップ」の超基本テクニックです！

### ✍️ ドリル 1-1：カウントダウンを作ろう

ロケットの発射カウントダウンを作りましょう。**5からスタートして、1ずつ減らし、0になったら「発射！」**と表示させます。

以下のコードはすでに動く状態です。まず実行して動きを確認し、その後 `5` を `10` に変えて動作が変わることを確認しましょう。

```python
time = 5

while time > 0:
    print(f"発射まであと {time} ...")
    time = time - 1     # 箱の中身を 1 減らす

print("🚀 発射！")
```

> **確認ポイント：** `time = time - 1` を消して実行するとどうなりますか？（無限ループになります！Ctrl+Cで止めましょう）

## 🕒 レッスン 2: 恐怖の「無限ループ」を体験せよ

繰り返しの魔法には、1つだけ**絶対にやってはいけない禁忌（タブー）**があります。それが「無限ループ」（終わらない繰り返し）です。

### 💻 ハンズオン（バグを体験する）

以下のコードは、ある1行を書き忘れています。実行するとどうなるでしょうか？

```python
hp = 10

while hp > 0:
    print(f"勇者のHPは {hp} だ！まだまだ戦える！")
    # わざとダメージを書かない ← ここが原因！
```

ターミナルが猛烈な勢いで文字を吐き出し始めましたね！一生終わりません！

**緊急停止方法：** ターミナルをクリックし、**`Ctrl + C`** を押してください。
赤い文字で `KeyboardInterrupt` と出て、プログラムが強制終了します。

> **💡 無限ループの原因と対策**
> `hp` を減らすコードを書き忘れたため、`hp > 0` の条件が**永遠にTrueのまま**になったのが原因です。
> `while` を使うときは必ず「ループの中で条件を変える処理」を書いてください。

## 🕒 レッスン 3: 魔法のブレーキ `break`

「ある特別なことが起きたら、強制的に繰り返しをやめる（ループを抜ける）」という便利なブレーキが **`break`（ブレイク）** です。

### 💻 ハンズオン（やってみよう）

ユーザーが「やめる」と入力するまでおうむ返しをするチャットボットです。

```python
while True:  # True と書くと、意図的に永遠に繰り返す（意図的な無限ループ）
    word = input("何か話しかけてね（'やめる' で終了）: ")

    if word == "やめる":
        print("じゃあね！バイバイ！")
        break  # ここで繰り返しをブチッと切って外に出る！

    print(f"あなたは「{word}」と言いましたね！")
```

`while True:` でわざと無限ループを作り、`if` で特定の言葉を待って `break` で脱出する。これはプロのエンジニアも毎日使う最強の組み合わせです。

### ✍️ ドリル 3-1：合い言葉で開く扉

「開けゴマ」と入力されるまで「合い言葉が違います！」と言い続け、正解の時だけ「扉が開いた！」と表示して `break` するプログラムを作ってください。

以下の手順で作ってみましょう。

1. `while True:` で無限ループを開始する。
2. ループの中で `input()` を使って合い言葉を受け取る。
3. `if` で入力が「開けゴマ」と等しいか確認する（`==` を使う）。
4. 正解なら「扉が開いた！」と表示して `break`。
5. 不正解なら「合い言葉が違います！」と表示する。

大正解！
