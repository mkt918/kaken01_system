## 🕒 レッスン 1: 回数を指定する魔法 `for` と `range()`

「5回だけ繰り返す」というとき、Pythonでは `for` と `range()`（レンジ＝範囲）をセットで使います。

### 💻 ハンズオン（やってみよう）
以下のコードをコピーして実行してみましょう。

```python
# range(5) は「5回繰り返す」という魔法の呪文です
for i in range(5):
    print("プログラミングは楽しい！")

print("繰り返し終了！")
```

「プログラミングは楽しい！」がピッタリ5行表示されましたね？
先週の `while` のように `count = count + 1` と書かなくても、`for` が自動的に数を数えて、指定した回数で勝手にストップしてくれます！

## 🕒 レッスン 2: 謎の箱 `i` の正体を暴け！

先ほどのコードで `for i in range(5):` と書きました。この **`i`**（アイ）とは一体何者なのでしょうか？

### 💻 ハンズオン（i の中身を覗き見しよう）
以下のコードを実行して、ターミナルにどんな数字が表示されるかよく見てください。

```python
for i in range(5):
    print(f"いま、i の中身は {i} です！")
```

**【結果の確認】**
`0`, `1`, `2`, `3`, `4` と表示されましたね？
> **💡 絶対に忘れてはいけないルール**
> パソコンは数を数えるとき、**「1」からではなく「0」から数え始めます。**
> だから `range(5)` と書くと最初の `i` は 0 になり、5回目である `4` まで進んだら終了するのです！

### ✍️ ドリル 2-1：階段を作ろう
文字のかけ算を使うと、文字を増やすことができます（例：`"★" * 3` ＝ ★★★）。
`i` を使って、星が1個ずつ増えていく「5段の階段」を作ってみましょう。

※ヒント：`i` は 0 から始まるので、`i + 1` 個の星を表示すると美しい階段になります。
```python

    star_count = i + 1
    # ↓ このカッコの中を埋めてみよう！
```

## 🕒 レッスン 3: 日々の積み重ね（合計の計算）

`for` ループは、ゲームのスコアのように「どんどん足し合わせていく」という計算が大得意です。

### 💻 ハンズオン（やってみよう）
「1日目：100円、2日目：200円…」というように、貯金箱にお金を入れていく様子をプログラミングします。

```python
# 最初、貯金箱の中身は 0円（カラッポ）
total_money = 0

for i in range(3):
    in_money = int(input("いくら貯金する？: "))
    
    # 今までの合計に、新しく入力されたお金を足して、合計箱に戻す！
    total_money = total_money + in_money
    print(f"チャリン。今の貯金額は {total_money} 円です。")

print("3日間の貯金が終わりました！")
```

### ✍️ ドリル 3-1：1から10まで全部足すと？
コンピュータに、1 + 2 + 3 + ... + 10 の計算をやらせてみましょう。
`range(1, 11)` と書くと、1からスタートして **10まで**（11の直前まで）の数字が順番に `i` に入ります。

```python
total = 0

# i は 1 から 10 まで変化する
for i in range(1, 11):
    # いまの total に i を足して、箱に戻そう
    total = total + _____
    
print(f"1から10まで全部足すと、{total} です！")
```
