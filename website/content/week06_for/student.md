# Week 06: 正確に回数を数える魔法「forループ」

先週の `while` は「ある条件まで（または無限に）繰り返す」魔法でした。
今度は「ちょうど10回繰り返したい！」「3回だけ実行したい！」というように、**回数がピッタリ決まっているとき**に使える超便利な魔法 **`for`** について学びます。

---

## 🕒 レッスン 1: 回数を指定する魔法 `for` と `range()`

「5回だけ繰り返す」というとき、Pythonでは `for` と `range()`（レンジ＝範囲）をセットで使います。

### 💻 ハンズオン（やってみよう）

以下のコードを入力して実行してみましょう。

```python
# range(5) は「5回繰り返す」という魔法の呪文です
for i in range(5):
    print("プログラミングは楽しい！")

print("繰り返し終了！")
```

「プログラミングは楽しい！」がピッタリ5行表示されましたね？
先週の `while` のように `count = count + 1` と書かなくても、`for` が自動的に数を数えて、指定した回数で勝手にストップしてくれます！

> **💡 while との使い分け**
>
> - **繰り返す回数が決まっている** → `for` が便利
> - **ある条件が満たされるまで繰り返す** → `while` が便利

## 🕒 レッスン 2: 謎の箱 `i` の正体を暴け！

先ほどのコードで `for i in range(5):` と書きました。この **`i`**（アイ）とは一体何者なのでしょうか？

### 💻 ハンズオン（`i` の中身を覗き見しよう）

以下のコードを実行して、ターミナルにどんな数字が表示されるかよく見てください。

```python
for i in range(5):
    print(f"いま、i の中身は {i} です！")
```

`0`, `1`, `2`, `3`, `4` と表示されましたね？

> **💡 絶対に忘れてはいけないルール**
> パソコンは数を数えるとき、**「1」からではなく「0」から数え始めます。**
> だから `range(5)` と書くと `i` は 0, 1, 2, 3, 4 と変化し、5回繰り返したところで終了します。

### ✍️ ドリル 2-1：星の階段を作ろう

文字のかけ算を使うと、文字を増やすことができます（例：`"★" * 3` ＝ `★★★`）。
`i` を使って、星が1個ずつ増えていく「5段の階段」を作ってみましょう。

**ポイント：** `i` は 0 から始まるので、`i + 1` 個の星を表示すると美しい階段になります。

以下のコードを実行して動きを確認しましょう。

```python
for i in range(5):
    star_count = i + 1
    print("★" * star_count)
```

> **確認：** ★ / ★★ / ★★★ / ★★★★ / ★★★★★ の5段になりましたか？

## 🕒 レッスン 3: 日々の積み重ね（合計の計算）

`for` ループは、「どんどん足し合わせていく」計算が大得意です。

### 💻 ハンズオン（やってみよう）

「1日目：100円、2日目：200円…」というように、貯金箱にお金を入れていく様子をプログラミングします。

```python
# 最初、貯金箱の中身は 0円（カラッポ）
total_money = 0

for i in range(3):
    in_money = int(input("いくら貯金する？: "))

    # 今までの合計に、新しく入力されたお金を足して、合計箱に戻す！
    total_money = total_money + in_money
    print(f"チャリン。今の貯金額は {total_money} 円です。")

print("3日間の貯金が終わりました！")
```

> **💡 `total_money = total_money + in_money` の意味**
> 先週の `count = count + 1` と同じ仕組みです。「今の合計に新しい金額を足して、合計の箱に上書き保存する」という意味です。

### ✍️ ドリル 3-1：1から10まで全部足すと？

コンピュータに、1 + 2 + 3 + ... + 10 の計算をやらせてみましょう。

`range(1, 11)` と書くと、1からスタートして **10まで**（11の直前まで）の数字が順番に `i` に入ります。

```python
total = 0

for i in range(1, 11):
    total = total + i   # いまの total に i を足して箱に戻す

print(f"1から10まで全部足すと、{total} です！")
```

> **確認：** 答えは 55 になりましたか？
