# 指導案: Week 06 「正確に回数を数える魔法（for）」

## 🎯 今週のねらいと授業のゴール

初心者にとって非常に難解な `for i in range()` の構文と、プログラム独特の「0から始まるカウント」の常識を体に覚えさせます。
また、合計値を蓄積していくアキュムレータ・パターン（`total = total + x`）の理解にも多くの時間を割きます。

## ⏱️ タイムテーブル (180分)

### 1️⃣ 導入と for in range() (0分〜40分)

- **【解説】**: 「whileは手動ブレーキの車、forは目的地を入れたら勝手に止まる自動運転車」と例えると理解しやすいです。
- **【演習】**: 「レッスン1」と「ドリル1-1」を実行させます。
- **【指導のポイント】**:
  - なぜ `range(5)` の後に `:` がいるのかを思い出させます。「これも `if` や `while` と同じ、お部屋の入り口だよ」と指導し、インデント（スペース4個）が必要なことも確認させます。

### 2️⃣ 0から始まる魔力と i の正体 (40分〜80分)

- **【解説】**: IT業界における「数は必ず0ベースから数える」という掟（ゼロオリジン）を教えます。エレベーターの「グランドフロア（0階）」などを例に出すと良いでしょう。
- **【演習】**: 「レッスン2」と「ドリル2-1」をやらせます。ドリル2-1は完成コードを渡す設計なので、動かしてから「`i + 1` を `i` だけに変えたらどうなる？」と試させてください。
- **【トラブルシューティング】**:
  - `i + 1` の意味が分からず躓く生徒がいます。「最初の `i` は 0 だから、星が 0 個（つまり何も表示されない）になっちゃうよね？」と実演して見せると腹落ちします。

### 3️⃣ 日々の積み重ね算（アキュムレータ） (80分〜120分) ※間で休憩10分

- **【解説】**: 変数は値の「上書き」ができることを復習します（`total = total + i`）。
- **【演習】**: 「レッスン3」と「ドリル3-1」を実施させます。ドリル3-1はコードを全部渡しているので、まず動かし、正解55が出たことを確認させます。
- **【指導のポイント】**:
  - `range(1, 11)` が「10まで」で「11は含まない」ことが直感に反してイライラする生徒が多いポイントです。「Pythonのルールで終了番号そのものは含まれない、そういう決まりだから仕方ない！」とここでは割り切らせて構いません。

### 4️⃣ 総仕上げ：ミニプロジェクト（貯金シミュレーター） (120分〜170分)

- **【総合演習】**: ミッションを5ステップに分解しているので、1ステップずつ確認しながら進めさせてください。
  - ステップ1（入力取得）→ ステップ2（合計変数）→ ステップ3（ループ枠）→ ステップ4（加算）→ ステップ5（表示）の順で積み上げると、どのパーツがどこにあるか整理しやすくなります。
  - `print(f"{i + 1} ヶ月目： {total} 円")` と書けるかがカギです（`i` のままだと「0ヶ月目」と表示されてしまうため）。
- **【早く終わった人向け（階段ビルダー）】**:
  - 自由な段数を入力させた上で、文字が増えていく様子は達成感があります。「■」だけでなく、好きな顔文字などで階段を作らせて、オリジナリティを発揮させましょう。

## 📝 小テストの解答と解説（全5問）

**Q1.** 繰り返す「回数」がハッキリと決まっている時に便利なループの命令はどれ？

- **正解:** **C. `for`**

**Q2.** `for i in range(3):` のように書いたとき、変数 `i` は最初どんな数字からスタートしますか？

- **正解:** **B. `0` から**
- **解説:** この法則は生涯にわたって生徒を助け、そしてエラーの温床にもなるプログラミング最大の掟です、と強調します。

**Q3.** `for i in range(3):` のループは、合計何回繰り返されますか？

- **正解:** **B. 3回**
- **解説:** 中身は `0, 1, 2` ですが、カウントしてみると「1回、2回、3回」なので、書いてある数字の回数だけ実行されることを確認します。

**Q4.** 文字のかけ算を使って `"★" * 4` と書いたとき、どのように表示されますか？

- **正解:** **C. `★★★★`**

**Q5.** `range(1, 6)` と書いたとき、作られる数字の範囲として正しいのはどれ？

- **正解:** **A. 1, 2, 3, 4, 5**
- **解説:** 右側に書かれた数字の直前で終わるトラップです。この仕様は生徒が混乱しやすいので繰り返し確認してください。
