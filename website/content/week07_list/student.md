# Week 07: データを一網打尽！「リスト」の魔法

今までは、1つの箱には「1つのデータ」しか入れられませんでした（`name = "たろう"` など）。
でも、クラス全員の点数や、勇者の持ち物すべてを1つずつ別の箱に入れていたら、100個も1000個も箱を作らなくてはいけません。
今週は、**「複数の箱を一列に繋げてまとめる」**魔法、**リスト（配列）**を学びます！

---

## 🕒 レッスン 1: 複数データをまとめる `[ ]`（角カッコ）

リストを使うと、複数のデータを `[ ]` と `,`（カンマ）を使って1つの名前で管理できます。

### 💻 ハンズオン（やってみよう）
好きな果物を3つ入れたリストを作ってみましょう。

```python
# fruitsという「大きなロッカー」に3つの果物を入れる
fruits = ["リンゴ", "バナナ", "ブドウ"]

# そのままprintすると、カッコごと全部表示されます
print(fruits)
```

### 🕒 レッスン 1-2: 特定のひとつだけを取り出す「インデックス番号」
上のロッカーから「バナナだけ」を取り出したいときはどうすればいいでしょうか？
ロッカーの番号を指定しますが、ここで**先週学んだ「プログラミング最大の掟」**が牙を剥きます。

**【掟】数は常に「0」から数える！**
- "リンゴ" ＝ 0番目
- "バナナ" ＝ 1番目
- "ブドウ" ＝ 2番目

```python
fruits = ["リンゴ", "バナナ", "ブドウ"]

# 0番目（つまり一番最初）を取り出す
print(fruits[0])

# バナナ（1番目）を取り出す
print(fruits[1])
```

## 🕒 レッスン 2: 中身を書き換える！

リストの中のデータは、あとから自由に変更することができます。
勇者が「ひのきのぼう」を捨てて「はがねの剣」を手に入れたシーンを作りましょう。

### 💻 ハンズオン（やってみよう）
上の `items` リストを使って中身を書き換えます。

```python
items = ["ひのきのぼう", "やくそう", "布の服"]

print("最初の持ち物：")
print(items)

# 「0番目」の中身を上書きする
items[0] = "はがねの剣"

print("武器を拾った！今の持ち物：")
print(items)
```

## 🕒 レッスン 3: ドンドン追加する魔法 `append()` と、長さを測る `len()`

最初から中身が決まっていなくても、後からリストの末尾に新しいデータをズドンと追加していく方法が **`append()` （アペンド）** です。
また、リストの中に「いま何個のデータが入っているか」を数えてくれるのが **`len()` （レングス）** です。

### 💻 ハンズオン（やってみよう）
```python
members = []  # 最初は誰もいない、カラッポのリスト

# メンバーをドンドン追加（append）する
members.append("たろう")
members.append("じろう")
members.append("さぶろう")

print(members)

# いま何人いるか（長さ）を測る
number = len(members)
print(f"現在、メンバーは {number} 人です！")
```

## 🚀 今日の総仕上げ1：ミニプロジェクト「クラスの名言ランダム表示」

みんなから集めた「名言」の中から、実行するたびにランダムで1つを表示する日めくりカレンダーのようなアプリを作ります。
※ランダムなことをさせるために、プログラムの一番上に `import random` という秘密の呪文を書きます。

### 📝 ミッション
- `import random` を1行目に書きます。
- `meigen = ["あきらめたらそこで試合終了だよ", "努力は必ず報われる", ...]` と、好きな名言を3つ以上入れたリストを作ります。
- `random.choice(meigen)` という魔法を使うと、リストの中からランダムで1つを引いてきてくれます。
- それを `print` でカッコよく表示してください！

**【ヒントコード】**
```python
import random

meigen = ["名言1", "名言2", "名言3"]
today_meigen = random.choice(meigen)

# うまく表示してね！
```

## 🚀 今日の総仕上げ2：ミニプロジェクト「オリジナル単語帳アプリ」

自分で英単語（や好きなマンガのキャラ名など）をひたすら入力してリストに保存し、最後に一覧として画面に出力するアプリを作ります。

### 📝 ミッション
- 最初にカラッポのリスト `tango = []` を用意します。
- `while True:` を使って無限ループを作ります。
- その中で `input()` を使って文字を入力させます。
- もし入力された文字が `終了` だったら、`break` でループを抜けます。
- それ以外の文字なら、リストに `append()` します。
- ループを抜けた後、`len()` を使って「合計 〇〇 個の単語を覚えました！」と表示し、その後にリストの中身をキレイに出力してください。

**【完成イメージ】**

```text
単語を入れてね（「終了」でストップ）: orange
単語を入れてね（「終了」でストップ）: banana
単語を入れてね（「終了」でストップ）: 終了

お疲れ様！合計 3 個の単語を覚えました！
['apple', 'orange', 'banana']
```
