# Week 07: データを一網打尽！「リスト」の魔法

今までは、1つの箱には「1つのデータ」しか入れられませんでした（`name = "たろう"` など）。
でも、クラス全員の点数や、勇者の持ち物すべてを1つずつ別の箱に入れていたら、100個も1000個も箱を作らなくてはいけません。
今週は、**「複数のデータを一列に並べてまとめる」**魔法、**リスト（配列）**を学びます！

---

## 🕒 レッスン 1: 複数データをまとめる `[ ]`（角カッコ）

リストを使うと、複数のデータを `[ ]` と `,`（カンマ）を使って1つの名前で管理できます。

### 💻 ハンズオン（やってみよう）

好きな果物を3つ入れたリストを作ってみましょう。

```python
# fruits という「大きなロッカー」に3つの果物を入れる
fruits = ["リンゴ", "バナナ", "ブドウ"]

# そのまま print すると、カッコごと全部表示されます
print(fruits)
```

### 🕒 レッスン 1-2: 特定の1つだけを取り出す「インデックス番号」

上のロッカーから「バナナだけ」を取り出したいときはどうすればいいでしょうか？
ロッカーの番号（**インデックス**）を指定しますが、ここで**先週学んだ「0から数える」ルール**が登場します。

- `fruits[0]` → "リンゴ"（0番目）
- `fruits[1]` → "バナナ"（1番目）
- `fruits[2]` → "ブドウ"（2番目）

```python
fruits = ["リンゴ", "バナナ", "ブドウ"]

# 0番目（一番最初）を取り出す
print(fruits[0])

# 1番目（バナナ）を取り出す
print(fruits[1])
```

```python
items = ["ひのきのぼう", "やくそう", "布の服"]
```

## 🕒 レッスン 2: 中身を書き換える！

リストの中のデータは、あとから自由に変更することができます。
勇者が「ひのきのぼう」を捨てて「はがねの剣」を手に入れたシーンを作りましょう。

### 💻 ハンズオン（やってみよう）

```python
items = ["ひのきのぼう", "やくそう", "布の服"]

print("最初の持ち物：")
print(items)

# 「0番目」の中身を上書きする
items[0] = "はがねの剣"

print("武器を拾った！今の持ち物：")
print(items)
```

> **💡 ポイント**
> `items[0] = "はがねの剣"` は「0番目の箱に"はがねの剣"を代入する」という意味です。
> 変数の代入（`=`）とインデックス（`[ ]`）を組み合わせた書き方です。

### ✍️ ドリル 2-1：テストの点数修正パズル

先生がテストの点数を間違えて入力してしまいました。Cくんの点数は本当は 90点 でした。
リストの点数を直してあげてください。

```python
#          A君  B君  C君  D君
scores = [ 85,  70,  50,  95 ]

# Cくんは何番目か数えて、点数を 90 に書き換える命令を書こう！
# （ヒント：Aくんが0番目です）

print("修正後の点数：")
print(scores)
```

## 🕒 レッスン 3: ドンドン追加する魔法 `append()` と、長さを測る `len()`

最初から中身が決まっていなくても、後からリストの末尾に新しいデータを追加していく方法が **`append()`（アペンド）** です。
また、リストの中に「いま何個のデータが入っているか」を数えてくれるのが **`len()`（レングス）** です。

### 💻 ハンズオン（やってみよう）

```python
members = []  # 最初は誰もいない、カラッポのリスト

# メンバーをドンドン追加（append）する
members.append("たろう")
members.append("じろう")
members.append("さぶろう")

print(members)

# いま何人いるか（長さ）を測る
number = len(members)
print(f"現在、メンバーは {number} 人です！")
```

> **⚠️ よくあるミス**
> `members = members.append("たろう")` のように `=` を使って書いてしまうとリストが壊れます。
> `append` は「リストに追加する命令」なので、`=` なしで `members.append("たろう")` と書くだけでOKです。

### ✍️ ドリル 3-1：入力されたものをリストに溜めよう

`for` ループを使って、3回入力させてリストに溜め込んでみましょう。

以下のステップで作ってみましょう。

1. `words = []` でカラッポのリストを作る。
2. `for i in range(3):` でループを始める。
3. ループの中で `input()` を使って文字を受け取り変数に入れる。
4. `words.append(変数)` でリストに追加する。
5. ループ後に `print(words)` でリスト全体を表示する。

```python
words = []

for i in range(3):
    text = input("好きな言葉を入れてね: ")
    words.append(text)   # text を words リストに追加する

print("あなたが入れた言葉一覧:")
print(words)
```

## 🚀 今日の総仕上げ：ミニプロジェクト「クラスの名言ランダム表示」

みんなから集めた「名言」の中から、実行するたびにランダムで1つを表示する日めくりカレンダーのようなアプリを作ります。

### 📝 ミッション

以下のステップで順番に作ってみましょう。

**ステップ1：** プログラムの1行目に `import random` と書く（これでランダム機能が使えるようになる）。

**ステップ2：** 名言を3つ以上入れたリスト `meigen = ["名言1", "名言2", "名言3"]` を作る。

**ステップ3：** `random.choice(リスト名)` を使うと、リストの中からランダムで1つを引いてくれる。

**ステップ4：** 引いた名言を変数に入れて、`print()` でカッコよく表示する。

```python
import random

meigen = ["あきらめたらそこで試合終了だよ", "努力は必ず報われる", "自分を信じろ"]
today_meigen = random.choice(meigen)

# today_meigen をカッコよく表示しよう！
print(f"今日の名言：「{today_meigen}」")
```

> **チャレンジ：** 自分の好きな名言やセリフを10個以上入れてみましょう！

## 🚀 早く終わった人へ：ミニプロジェクト「オリジナル単語帳アプリ」

自分で単語をひたすら入力してリストに保存し、最後に一覧として出力するアプリを作ります。

### 📝 ミッション（単語帳）

以下のステップで順番に作ってみましょう。

**ステップ1：** カラッポのリスト `tango = []` を用意する。

**ステップ2：** `while True:` で無限ループを開始する。

**ステップ3：** ループの中で `input()` を使って単語を入力させる。

**ステップ4：** 入力された文字が「終了」だった場合は `break` でループを抜ける。

**ステップ5：** 「終了」以外の文字なら `tango.append()` でリストに追加する。

**ステップ6：** ループを抜けた後、`len()` で「合計〇〇個」と表示し、リストの中身を出力する。
