# 指導案: Week 07 「リストと配列」

## 🎯 今週のねらいと授業のゴール
データ構造の第一歩である「リスト（配列）」の概念を学びます。
何よりも「インデックスは0から始まる」というITの絶対法則の定着と、これまでの集大成として「ループと組み合わせたリスト操作」を体験させることが最大の目標です。

## ⏱️ タイムテーブル (180分)

### 1️⃣ 導入と [ ] (0分〜40分)
- **【解説】**: データが何百個もあるとき、箱（変数）を何百個も作るのは面倒だ！という課題を提示し、リストの便利さを教えます。
- **【演習】**: 「レッスン1」を実行させ、「[ ]」で囲むこと、「,」で区切ることを叩き込みます。
- **【トラブルシューティング（超重要）】**: 
  - `["リンゴ", バナナ", "ブドウ"]` のように、ダブルクォーテーションを片方書き忘れると、カンマが文字の一部として認識されるなどして大混乱が起きます。
  - 「色が変わっているところ（VS Codeのハイライト色）を見てごらん。バナナの色がおかしいよね？」と、エディタの色によるバグ発見の手法を教えてください。

### 2️⃣ 0から始まる魔力パート2（インデックス） (40分〜80分)
- **【解説】**: Week6の `range` に続き、またしても「0から数える」ルールの登場です。
- **【演習】**: 「レッスン1-2」と「ドリル1-1」「ドリル2-1」をやらせます。
- **【指導のポイント】**: 
  - 生徒は「1番目」と言われると `[1]` を書いてしまいます。常に「指を折って数えるときは、最初は0番！」と口酸っぱく指導してください。

### 3️⃣ 動的にリストを育てる（append） (80分〜120分) ※間で休憩10分
- **【解説】**: ゲームのインベントリ（持ち物袋）のように、後からアイテムが追加される機能を `append` で実現できることを説明します。
- **【演習】**: 「レッスン3」と「ドリル3-1」を実施させます。
- **【指導のポイント】**:
  - `append` の前に文字を代入しようとして `members = members.append("たろう")` と書いてしまう生徒がいます（Pythonではエラーにはなりませんが、リストがNoneになって破壊されます）。
  - `append` は「変数に大砲を撃ち込むイメージ」と伝え、「 `=` はいらないよ」と指導します。

### 4️⃣ 総仕上げ：ダブル・ミニプロジェクト (120分〜170分)
- **【総合演習 1】**: 「クラスの名言ランダム表示」。
  - プログラミングに「ランダム」要素が加わると、生徒のテンションが一気に上がります。
  - `import random` はおまじないとして丸暗記させます。「なんで一番上（1行目）に書くの？」という質問には「魔法使いが魔法をとなえるための本（ライブラリ）を、最初に棚から持ってくる必要があるからだよ」と答えてください。
- **【総合演習 2】**: 「オリジナル単語帳アプリ」。
  - Week5で学んだ `while True:` と `break` の完全な復習です。これを自力で何も見ずに書けるようになっていれば、Step1（Python基礎）の理解度は相当高いと言えます。

## 📝 小テストの解答と解説（全5問）

**Q1.** 複数のデータ（文字や数字）を1つにまとめるためのカッコの記号はどれ？
- **正解:** **C. `[ ]`**
- **解説:** `{ }`（辞書・セット）や `( )`（タプル・関数）など、プログラミングではカッコの種類で意味が激変することを伝えます。

**Q2.** `weapons = ["剣", "槍", "弓"]` というリストがあります。「槍」を取り出したいとき、数字は何番を指定する？
- **正解:** **B. `weapons[1]`**
- **解説:** はい、出ました0番ルール。引っかかった生徒には笑いながら「プログラミング界では生きていけないぞー！」と愛のあるツッコミを入れてください。

**Q3.** リストの中にある特定の番号のデータを別のものに変える（上書きする）正しい書き方は？
- **正解:** **A. `weapons[0] = "杖"`**
- **解説:** 変数の代入 `=` とインデックス `[]` を組み合わせた応用です。

**Q4.** カラッポのリストの末尾に、新しくデータをドンドン追加していく命令（技）の名前は？
- **正解:** **C. `append()`**
- **解説:** 今後、WebアプリやAIのプログラミングでも一生使い続ける英単語です。

**Q5.** リストの中に「いま何個のデータが入っているか」を調べる（長さを測る）命令は？
- **正解:** **B. `len()`**
- **解説:** Lengthの頭文字3つであることを教えてあげてください。
