## 🕒 レッスン 1: オリジナル魔法の作り方 `def`

新しく関数（魔法）を定義（define）するには、`def`（デフ）というキーワードを使います。

### 💻 ハンズオン（やってみよう）
以下のコードをコピーして実行しましょう。

```python
# 1. ここで「say_hello」という魔法を作って（定義して）おきます
def say_hello():
    print("こんにちは！")
    print("今日も1日がんばりましょう！")

# 2. まだ実行しても何も起きません。作った魔法を【使う（呼び出す）】必要があります。
say_hello()
say_hello()
```

「こんにちは！今日も1日がんばりましょう！」が2回連続で表示されましたね？
`def` のブロックも、`if` や `while` と同じように **インデント（スペース4個）** でお部屋を作ります。

### ✍️ ドリル 1-1：応援魔法を作ろう
「がんばれ！」「君ならできる！」と2行連続で励ましてくれる魔法 `cheer_up()` を作って、それを3回呼び出してください。

```python

    print("がんばれ！")
    print("君ならできる！")
```

## 🕒 レッスン 2: 魔法の材料「引数（ひきすう）」

`print("こんにちは")` のように、魔法のカッコの中にデータを入れると、そのデータを使って魔法が発動します。このカッコの中に入れるデータを **「引数（ひきすう）」** と呼びます。

### 💻 ハンズオン（やってみよう）
渡された名前に向かって挨拶する魔法を作ります。

```python
# name という「引数（材料を受け取る箱）」を用意する
def greet(name):
    print(f"やあ、{name} さん！調子はどう？")

# 使う時は、材料をカッコの中に入れて渡す！
greet("たろう")
greet("花子")
```

```python
# a と b の2つの材料を受け取る
def tasizan(a, b):
    # a と b を足した結果を表示するコードを書こう！
    # ヒント：print(f"{a} ＋ {b} ＝ {a + b} です！")
```

## 🕒 レッスン 3: 魔法の戻り値「return」

本物の魔法使い（プロのプログラマー）は、`print` で表示して終わりにするのではなく、**「処理した結果だけを返して（`return`）、あとは好きに使ってね」** というスタイルの関数をよく作ります。

### 💻 ハンズオン（やってみよう）
```python
def double_number(num):
    result = num * 2
    # 呼び出した人に結果を「返す（戻り値）」
    return result

# 魔法を使った結果が、箱（answer）に入る！
answer = double_number(10)
print(f"10の2倍は {answer} です！")
```
