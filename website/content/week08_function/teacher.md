## ⏱️ タイムテーブル (180分)

### 1️⃣ 導入と def (0分〜30分)
- **【解説】**: 「今まで使っていたprintやinputの正体は、誰かが超絶長いコードを書いて作ってくれた『関数』だったんだよ」「今日はみんながそれを作る側になる」と伝えます。
- **【演習】**: 「レッスン1」および「ドリル1-1」を実行させます。
- **【指導のポイント】**: 
  - 初心者は「`def`で定義したのに画面に何も出ない！」と必ず叫びます。「魔法の呪文は、辞典に登録しただけじゃ発動しない。唱える（呼び出す `say_hello()` を書く）必要があるよ」と実演を交えて指導します。

### 2️⃣ 引数（ひきすう）と戻り値（return） (30分〜60分)
- **【解説】**: 引数は「魔法の材料」、returnは「魔法の完成品のお届け」と例えます。
- **【演習】**: 「レッスン2」「ドリル2-1」と「レッスン3」をやらせます。
- **【トラブルシューティング】**: 
  - `def tasizan(a, b):` なのに、呼び出すときに `tasizan(10)` と1つしか入れずにエラー（`TypeError: tasizan() missing 1 required positional argument: 'b'`）を出す生徒が続出します。
  - 「大工さんに『家を建てて！』って頼んだのに、木材（材料a）だけ渡して、釘（材料b）を渡し忘れたら怒られるよね。パソコンも同じだよ」と説明します。

### 3️⃣ ミニプロジェクト 1：多言語挨拶ロボ (60分〜100分) ※間で休憩10分
- **【総合演習 1】**: ユーザーからの `input` と、自作関数の `引数` をドッキングさせる演習です。
  - 実は関数を使わなくても書けるプログラムですが、あえて `def` の中に処理を押し込む（カプセル化する）ことで、「外はスッキリ、中はゴチャゴチャ」というプログラミングの美しい設計（デザインパターン）の初歩を無意識に体験させます。

### 4️⃣ Step 1 総決算プロジェクト：RPGバトル (100分〜170分)
- **【総合演習 2】**: 最難関かつ最高に面白いプロジェクトです。なんと講師用解答例は30行近くになります。
- **【指導ポイント】**:
  - 最初から全部書かせると絶対に混乱します。以下のように順番に作らせてください（講師が前のスクリーンで実況コーディングするのもアリです）。
    1. まず変数（勇者とボスのHP）を作る。
    2. `while True:` を書いて、まずは中身をパス（`pass`）にしておく。
    3. `input` でコマンドを選ばせる。
    4. `if` と `elif` で、自分の攻撃と回復を作る。
    5. 外側のインデントに戻って（ターンの最後に）、ボスのランダム攻撃を作る。
    6. HPが0以下になったかのゲームオーバー判定（`break`）を入れる。
  - 「完成した人から、魔法攻撃（コマンド3）や、会心の一撃を追加していいよ！」と伝えると、生徒のモチベーションは爆発します。

## 📝 小テストの解答と解説（全5問 - 総まとめ号）

**Q1.** 新しい自作の魔法（関数）を作るための、一番最初のキーワード（英単語）はどれ？
- **正解:** **B. `def`**
- **解説:** Define（定義する）の略です。関数＝Definitionです。

**Q2.** 関数を呼び出すときに、カッコの中に入れて渡す「材料」データのことを何と呼ぶ？
- **正解:** **A. 引数（ひきすう）**
- **解説:** いんすう、と読み間違える大人が多いので読み方も教えます。

**Q3.** 関数の処理が終わったあと、呼び出した側に「結果のデータ」を返すための命令はどれ？
- **正解:** **C. `return`**
- **解説:** リターン。球技のリターンと同じく、打ち返すイメージです。

**Q4.【復習】** Pythonが `IndentationError` を出すのは、主に何を忘れたときですか？
- **正解:** **B. 行の先頭の半角スペース（インデント）**
- **解説:** 8週間で最も生徒を苦しめ、そして成長させたエラーです。

**Q5.【復習】** これまで学んだ中で、**間違っている**組み合わせはどれですか？
- **正解:** **C. `input()` = 指定した回数だけループ（繰り返し）を行う**
- **解説:** もちろん、指定回数のループは `for` ループです。`input()` は入力を受け取る魔法ですね。
